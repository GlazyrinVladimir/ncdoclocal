# !Идентификация пиков

## Идентификация пиков

Идентификация — процесс отнесения каждого пика, обнаруженного на хроматограмме, к определённому компоненту, заданному в составе рабочего метода анализа. Этот этап основывается на сопоставлении рассчитанных параметров пиков с информацией таблицы компонентов.

### Основные способы идентификации компонентов

## **Идентификация по абсолютному времени удерживания**

Идентификация по абсолютному времени удерживания — это наиболее простой и широко применяемый метод сопоставления пиков на хроматограмме с известными компонентами.

Принцип метода

Идентификация осуществляется путем сравнения времени удерживания каждого пика, обнаруженного на хроматограмме, с временем удерживания компонента, хранящемся в таблице компонентов методики анализа при строго фиксированных условиях. При этом в таблице компонентов должна содержаться следующая информация для каждого вещества:

* Наименование компонента;
* Время удерживания (в единицах времени);
* Окно поиска по времени — область допуска изменения времени удерживания компонента, в пределах которой будет производиться поиск соответствующего пика.

### Окно поиска

Окно поиска задаёт границы допустимого диапазона времени удерживания вокруг библиотечного значения, как в положительную, так и в отрицательную сторону. При его подборе необходимо учитывать:

* Ширина окна должна быть достаточно большой, чтобы учесть неизбежные мелкие изменения времени удерживания, вызванные вариациями условий анализа, износа колонки и другими факторами.
* При этом окно не должно быть чрезмерно широким, чтобы избежать пересечения с соседними пиками и, как следствие, ошибочной идентификации.

### Выбор пика в случае нескольких кандидатов

Если в окно поиска попадает несколько пиков, алгоритм идентификации выбирает пик, обладающий максимальной вероятностью соответствия компоненту. Обычно критериям предпочтения служат:

* Наибольшая интенсивность (высота пика);
* Минимальное отклонение от библиотечного времени удерживания.

### Практические рекомендации

* Следует минимизировать ширину окна поиска, сбалансировав надёжность идентификации и риск ложного совпадения.
* В случае невозможности провести качественную идентификацию по абсолютному времени удерживания рекомендуется использовать методы идентификации по относительным параметрам или дополнительные детекторы.
* Периодически актуализировать времена удерживания в базе компонентов при изменениях условий анализа и состоянии колонок.

## **Идентификация по температурам кипения**

Температуры кипения используются как частный случай индексов удерживания, с формулами расчёта, аналогичными индексам. Для реперных компонентов задаются известные температуры кипения в библиотеке.

## **Идентификация по относительному времени удерживания**

При проведении газохроматографических анализов условия проведения часто изменяются: колеблется расход газа-носителя, температура колонки меняется, и со временем колонки подвергаются износу и старению. Эти факторы приводят к смещению времени удерживания компонентов на хроматограмме, из-за чего пики могут не попадать в заранее заданные окна поиска по абсолютному времени удерживания.

### Особенности задачи

* Редко возможно назначить достаточно широкое окно поиска по абсолютному времени, чтобы оно покрывало колебания времени удерживания без попадания соседних пиков;
  * При длительных анализах c программируемой температурой колонки сдвиги времени удерживания могут быть значительными и нелинейными.

### Методика решения

Для корректной идентификации в таких условиях используется метод относительного времени удерживания, базирующийся на "стандартах времени":

1. **Назначение стандартов времени**\
   Один или несколько пиков, которые стабильно присутствуют на хроматограмме и хорошо разделены, назначаются пиками-стандартами времени. Для этих пиков задаётся увеличенное окно поиска (обычно 2-5%), что повышает вероятность их надёжного обнаружения.
2. **Определение коэффициента коррекции времени**\
   После обнаружения на хроматограмме пика-стандарта вычисляется коэффициент отклонения: $$K = \frac{T_{ст.реал.}}{T_{ст.библ}} \\ T_{ст.реал.} - \text{фактическое время удерживания пика стандарта} \\ T_{ст.библ} - \text{библиотечное (ориентировочное) время удерживания этого пика}$$
3. **Коррекция ожидаемых времен удерживания остальных пиков**\
   Для остальных компонентов рассчитывается скорректированное время удерживания с учётом коэффициента: $$\\T_{коррект.}=T_{библ}*K$$
4. **Поиск пиков по скорректированному времени**\
   Идентификация остальных пиков осуществляется в окне поиска, смещённом и суженном относительно скорректированного значения времени удерживания.

### Преимущества метода

* Позволяет учитывать систематические смещения времени удерживания, связанные с изменениями условий и состоянием оборудования;
* Снижает необходимость задания расширенных и перекрывающихся окон поиска, что уменьшает ошибки идентификации;
* Способствует более стабильной и точной идентификации компонентов при изменяющихся аналитических условиях.

### Практические рекомендации

* Пики-стандарты должны быть устойчивыми, хорошо разделёнными и присутствовать в каждой серии анализов;
* Размер окна поиска для стандартов рекомендуется увеличить, чтобы гарантировать их надёжное обнаружение;
* Для повышения достоверности можно использовать несколько стандартов времени, распределённых по временному диапазону анализа;
* При изменении условий анализа коэффициенты корректировки необходимо периодически пересчитывать.

Идентификация по относительному времени удерживания применяется при изменениях условий анализа (расход газа-носителя, температура колонки) и старении колонок, когда пики могут не попадать в окна поиска по абсолютному времени удерживания. Обычно окно поиска не может быть расширено настолько, чтобы не захватывать соседние пики, особенно при длительных анализах с программируемой температурой.

Для решения этой проблемы один или несколько пиков выбираются в качестве "стандартов времени" — пиков с устойчивым присутствием и хорошо разделённых. Для них задаётся увеличенное окно поиска (2–5%), обеспечивающее их надёжное обнаружение на хроматограмме.

Далее проводится поиск пика стандарта времени по абсолютному времени удерживания. Для найденного пика рассчитывается коэффициент коррекции: $$K=\frac{T_{ст.реал}}{T_{ст.библ.}}\\T_{ст.реал}\text{-фактическое время удерживания стандарта,}\\{T_{ст.библ.}\text{-библиотечное время удерживания}}$$   Для остальных пиков рассчитывается ожидаемое скорректированное время удерживания: $$\\T_{ожид}=T_{библ}*K$$По этим скорректированным значениям выполняется поиск пиков в более узких окнах, повышая точность идентификации с учётом систематических сдвигов времени удерживания. Польза метода — в компенсации изменений условий анализа без увеличения ширины окон поиска и риска захвата соседних пиков.

Рекомендуется выбирать стабильные стандарты времени, задавать для них широкие окна поиска и периодически обновлять коэффициенты коррекции при изменениях условий анализа либо замене колонок.

## **Идентификация по относительному объёму удерживания**

\
Относительный объем удерживания является более точным параметром для идентификации компонентов по сравнению с относительным временем удерживания, поскольку учитывает мертвое время — время прохождения подвижной фазы (газа-носителя) от устройства ввода пробы до детектора. В литературе это время называют также временем удерживания несорбирующегося вещества.

### Определение

* Мертвое время — время, за которое газ-носитель проходит через колонку и систему без взаимодействия с неподвижной фазой, т.е. время удерживания активного компонента равного нулю.
* Время удерживания компонента — время прохождения компонента от момента ввода пробы до выхода пика.
* Время удерживания стандарта времени — время удерживания пика, выбранного в качестве стандарта (обычно один из реперных пиков с хорошо известным временем удерживания).

### Расчет относительного объема удерживания

Относительный объем удерживания вычисляется по формуле: $$R_i=\frac{T_{уд_i}-T_м} {T_{уд.ст.} - T_м}\\T_{уд_i}\text{- время удерживания анализируемого компонента}\\T_{уд.ст.}\text{ - время удерживания стандарта времени}\\T_м\text{ - мертвое время}$$/Особенности

* Относительный объем удерживания стандарта принимается за единицу.
* Использование относительного объема удерживания позволяет устранить влияние систематических смещений времени задержки газа и изменения условий анализа.
* Мертвое время определяется экспериментально (например, временем появления пика несорбирующегося вещества, такого как метан или воздух) или вычисляется на основании измерений соседних стандартных соединений.

### Практическое применение

Относительный объем удерживания обеспечивает более стабильную и точную идентификацию компонентов в различных условиях, особенно при сравнении результатов анализа на разных приборах или с разными рабочими методами.

## **Идентификация по времени удерживания и соотношениям интенсивностей пиков на нескольких детекторах**

В газохроматографии для повышения достоверности идентификации компонентов часто применяются два и более детектора, работающих либо параллельно, либо последовательно в течение одного анализа.

### Методика идентификации

1. **Первичная идентификация по времени удерживания**\
   На первом этапе пик идентифицируется на ведущем детекторе, который обычно используется для количественного анализа. В данном детекторе определяется абсолютное или скорректированное относительное время удерживания пика, что позволяет предварительно отнести пик к компоненту из библиотеки.
2. **Сопоставление интенсивностей пиков на других детекторах**\
   Для повышения точности идентификации вторая составляющая метода — анализ отношения интенсивностей пиков на остальных детекторах. Отношение интенсивностей (амплитуд или площадей пиков) рассчитывается между детекторами и сравнивается с установленными в библиотеке компонент значениями.
3. **Дополнительные критерии по наличию пика на вспомогательных детекторах**\
   Помимо количественного соотношения интенсивностей, для идентификации учитывается факт присутствия или отсутствия соответствующего пика на вспомогательных (неведущих) детекторах. Отсутствие сигнала там, где он ожидается, служит предупреждающим признаком возможной ошибки идентификации.

### Технические аспекты

* Соотношения интенсивностей пиков зависят от чувствительности каждого детектора, спектральной селективности и характера реагентов.
* В библиотеках программы NetChrom задаются стандартные отношения интенсивностей для каждого компонента, с которыми производится сверка наблюдаемых значений.
* Параллельно и последовательно работающие детекторы могут иметь различные принципы работы (пламенно-ионизационный, электронно-захватный, термоионный и др.), что влияет на характер сигнала и требует корректного учета.
* Использование нескольких детекторов значительно снижает вероятность ложной идентификации, особенно в сложных смесях, позволяя отличать компоненты с близкими временами удерживания.

### Преимущества

* Увеличение надёжности и точности идентификации.
* Возможность выявления компонентов, имеющих схожее время удерживания, но различающихся по химическим свойствам.
* Снижение ошибок из-за сдвигов времени удерживания.
* Более полное использование возможностей аналитической системы.

### Рекомендации

* Перед использованием метода требуется тщательная настройка и калибровка детекторных отношений в библиотеке компонента.
* В случаях возникновения противоречий между периодами удерживания и отношениями интенсивностей рекомендуется проводить дополнительный анализ или ручную проверку.
* Настройку рекомендуется выполнять регулярно, особенно при изменении условий анализа или оборудования.



## **Идентификация по индексам удерживания (индексы Ковача)**

Идентификация компонентов методами газовой хроматографии может основываться на относительных параметрах удерживания, менее чувствительных к изменениям условий анализа по сравнению с абсолютным временем удерживания. Одним из таких параметров является индекс удерживания, особенно широко применяемый — индекс Ковача.

### Определение индекса удерживания

Индекс удерживания — безразмерная величина, характеризующая положение пика компонента относительно пиков выбранных стандартов, как правило, ряда насыщенных углеводородов (алканов, парафинов). Индексы Ковача для н-алканов определяются как 100 × число атомов углерода (например, для н-гексана 600, для н-гептана 700).

### Типы индексов

* **Логарифмический индекс Ковача** характерен для случаев проведения анализа при постоянной (изотермической) температуре колонки. В таких условиях исправленное время удерживания (время удерживания минус мертвое время) увеличивается логарифмически с увеличением числа атомов углерода в гомологическом ряду.
* **Линейный индекс Ковача** более применим при программировании температуры колонки, когда зависимость времени удерживания становится более сложной и приближается к линейной в области изменения температуры.

### Смешанный тип индекса

В практике, при использовании программирования температуры с начальным изотермическим участком, применяется смешанный тип:

* На изотермическом участке вычисляется логарифмический индекс.
* На последующих участках с изменяющейся температурой применяется линейный индекс.

Начало программирования температуры задаётся в параметрах метода.

### Необходимые данные

В таблице компонентов указываются:

* Табличные значения индексов удерживания для каждого компонента;
* Ширина окна поиска по индексу удерживания;
* Тип компонента (обычный или реперный — «реперные» пики — пиковые стандарты с заведомо известными индексами, для которых задаётся увеличенное окно поиска 2-5%).

### Алгоритм идентификации по индексам удерживания

1. Идентифицируются реперные пики по времени удерживания (абсолютному или относительному).
2. Реперным пикам присваиваются соответствующие значения индексов из таблицы компонентов.
3. По известным индексам реперных пиков рассчитываются индексы удерживания для остальных пиков по формулам ниже и сравниваются с табличными значениями.

### Формулы расчёта индексов

* **Линейный индекс удерживания**: $$li = ln + (ln + 1-ln)*\frac{(ti-tn)}{(tn +1-tn)}$$
* **Логарифмический индекс удерживания**: $$li = ln + (ln+1-ln)*(log(t'i) - \frac{log(t'n))}{log(t'n+1)}-log(t'n))\\li\text{-время удерживания пика}\\ln,ln+1\text{-индексы предыдущего и последующих компонентов с известной величиной индекса}\\ti\text{-время удерживания интересующего пика}\\tn,tn+1\text{-время удерживания пиков,\,соответствующие\,\\предыдущему и последующему компонентам с известными индексами}\\t'.../text{-приведенное\,время\,удерживания}$$

**Приведённое время удерживания и мёртвое время**

Приведённое время удерживания вычисляется как разность между абсолютным временем удерживания и мёртвым временем — временем продвижения несорбирующего компонента (обычно метана) через колонку.

Мёртвое время определяется экспериментально или рассчитывается с помощью инструментальных средств, например, газового калькулятора.

**Выбор пика при нескольких кандидатах**

Если окно поиска по индексу содержит несколько пиков, выбор осуществляется на основе:

* Ближайшего к библиотечному значению индекса;
* Максимальной высоты пика;
* Максимальной площади пика.

### Ограничения

Если один пик попадает в окна поиска нескольких компонентов, необходимо уменьшить размер окон, чтобы избежать пересечений и ошибок.

### Выводы

Метод идентификации по индексам удерживания Ковача является универсальным и гибким, успешно применим для большинства рабочих методов. Он обеспечивает высокую стабильность и точность при сравнении данных, даже при изменяющихся условиях анализа.

Для корректной работы оператору достаточно грамотно выбрать реперные пики и регулярно корректировать их ожидаемое время удерживания.
