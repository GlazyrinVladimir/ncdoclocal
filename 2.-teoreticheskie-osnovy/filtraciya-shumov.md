# Фильтрация шумов

## Модуль первичной фильтрации и сглаживания сигнала

На хроматограмме в программе **«NetChrom 3.0»** всегда присутствуют шумы и выбросы, вызванные неидеальной работой аналитической и электронной систем хроматографа, включая элементы системы электронного управления потоками газов (ЭУПГ), терморегуляторы, детектор, аналого-цифровой преобразователь (АЦП) и соединения между ними.

Высокий уровень шумов снижает предел детектирования, а в сочетании с дрейфом нулевой линии существенно затрудняет последующую автоматическую обработку хроматограммы.

### Принцип работы цифровой фильтрации

Для снижения интенсивности шумов в программе применяется цифровая фильтрация. Основной принцип процесса заключается в математической обработке численного значения каждой точки хроматограммы с учётом значений её соседних точек. Количество таких соседних точек слева и справа от обрабатываемой определяет геометрическую ширину **окна фильтрации** (апертуру).

Чем больше точки в окне, тем сильнее сглаживаются шумы. Однако фильтрация искажает форму пиков, поэтому её рекомендуется применять только при высокой шумности сигнала. Оптимальная стратегия — начинать с минимального окна и постепенно увеличивать размер, пока качество сглаживания не станет удовлетворительным. При необходимости можно выполнить несколько проходов фильтрации.

В процессе вычислений программа пошагово перемещает расчетное окно вдоль всего массива данных хроматограммы от первого зарегистрированного отсчета АЦП до последнего. При этом соблюдаются следующие правила:

* **Зависимость сглаживания**: Чем шире установлено окно фильтрации (чем больше точек в него попадает), тем сильнее подавляются высокочастотные шумы и стабилизируется нулевая линия.
* **Искажение геометрии**: Любая фильтрация в той или иной степени деформирует реальную форму хроматографических пиков. Избыточная ширина окна может привести к потере метрологической точности.

**Рекомендации по настройке сглаживания:**

1. Применение фильтров целесообразно и рекомендуется только при высокой шумности исходного сигнала.
2. Подбор параметров рекомендуется проводить опытным путём: начинать с минимальной ширины окна и постепенно увеличивать её до достижения оптимального визуального результата.
3. При необходимости глубокой очистки сигнала от помех допускается выполнять фильтрацию в несколько последовательных проходов (итераций).

При каждом дискретном сдвиге окна фильтрации ПО «NetChrom 3.0» выполняет следующие операции:

* **Ограничение массива** — вертикальные границы окна отсекают локальный рабочий участок хроматограммы. В расчете текущего шага учитываются только те зарегистрированные отсчеты АЦП (на схеме — красные маркеры), которые физически находятся внутри этих границ.
* **Локализация расчетного центра** — программа определяет **точку середины окна** (выделена центральной вертикальной осью). Исходное зашумленное значение именно этой точки будет перезаписано программой по результатам вычислений.
* **Пересчет значения** — выбранный математический метод анализирует значения всех соседних точек внутри окна, вычисляет новое сглаженное значение и присваивает его центральной точке. После этого окно сдвигается вправо на один шаг (на один отсчет АЦП), и цикл повторяется.

## Методы фильтрации в программе

Выбор конкретного метода фильтрации и его параметров зависит от уровня шума и требований к сохранению формы пиков. В программе реализованы следующие инструменты:

### Фильтрация выбросов (Импульсный фильтр)

Предназначен для селективного удаления одиночных высокочастотных пиковых помех (электрических наводок и бросков напряжения).

* **Логика работы:** Выбросы автоматически распознаются алгоритмом по превышению значения сигнала заданного порога шума.
* **Результат:** Программа устраняет одиночные выбросы, заменяя дефектное значение точки средним арифметическим соседних валидных точек. Данный метод практически не искажает форму остальной хроматограммы.

### Фильтр среднего (Гауссовая фильтрация)

Инструмент линейного сглаживания, наиболее эффективный для подавления постоянного белого шума.

* **Логика работы:** Значения точек внутри окна последовательно усредняются с весами, распределёнными по нормальной гаунсовской функции.
* **Результат:** Обеспечивает качественное выпрямление нулевой линии, но уменьшает высоту и расширяет ширину хроматографических пиков.

{% hint style="info" %}
Несмотря на геометрическое изменение пропорций пика при «гауссовой» фильтрации (пики становятся ниже и шире), его интегральный параметр — площадь пика — остаётся строго неизменной, что полностью сохраняет корректность последующих количественных расчетов.
{% endhint %}

### Фильтр медианы

Нелинейный метод обработки, ориентированный на одновременное подавление шумов и протяженных по времени импульсных помех.

* **Логика работы:** Значения всех отсчетов внутри окна ранжируются (сортируются) по возрастанию, после чего исходное значение центральной точки заменяется медианой полученного ряда.
* **Результат:** Сглаживает шумы и выбросы с меньшим искажением формы пиков, что особенно эффективно проявляется на крутых восходящих и нисходящих склонах.

**Примечание:** Для корректной работы медианного фильтра ширина выбранного окна должна строго превышать геометрическую ширину самых больших одиночных выбросов на хроматограмме.

### Полиномиальный фильтр (Метод Савицкого-Голея)

Высокоточный фильтр, обеспечивающий оптимальный баланс между снижением уровня шума и сохранением исходных метрологических параметров пиков.

* **Логика работы:** По точкам окна аппроксимируется сглаживающая кривая (парабола) второго порядка, рассчитываемая методом наименьших квадратов (МНК). Исходное значение точки заменяется расчетным значением на полученной кривой.
* **Результат:** Качественно подавляет шумы высокой плотности, обеспечивая минимальное искажение формы, высоты и полуширины хроматографических пиков.

### Биномиальный фильтр

Наиболее эффективный и современный алгоритм дискретного сглаживания, реализованный в ПО «NetChrom 3.0».

* **Логика работы:** Значения точек хроматограммы рассчитываются как взвешенная сумма с учётом индивидуальных биномиальных коэффициентов (чисел треугольника Паскаля) для заданной ширины окна.
* **Результат:** Фильтр обеспечивает максимальную степень сглаживания случайных шумов и выбросов, практически не искажая исходную форму и контуры хроматографических пиков.

{% hint style="info" %}
Выбор метода и параметров фильтрации зависит от уровня шума и требований к сохранению формы пиков. Рекомендуется опытным путём подобрать оптимальные настройки.
{% endhint %}
