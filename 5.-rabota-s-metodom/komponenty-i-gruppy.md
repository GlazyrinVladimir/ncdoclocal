# !Компоненты и группы

## Панель оперативного управления библиотекой компонентов и групп

В правой части рабочего пространства модуля «Компоненты и группы» расположена интерактивная панель быстрого конфигурирования. Она предназначена для оперативного ведения справочника анализируемых веществ и создания метаструктур (групп) непосредственно в процессе графического анализа хроматограммы, минуя вызов модальных окон верхнего уровня.

Панель разделена на два независимых функциональных блока управления: **«Компоненты»** и **«Группы»**.

### Компоненты (Индивидуальные вещества)

Предназначен для ручной или автоматической регистрации целевых соединений, идентифицируемых по времени удерживания.

* **Элемент \[Создать] (значок `+`):** Инициализирует процедуру добавления новой строки в градуировочную таблицу метода. Оператор имеет возможность присвоить веществу имя и зафиксировать для него текущее положение курсора на графике в качестве опорного времени удерживания.
* **Элемент \[Удалить] (значок `–`):** Безвозвратно исключает выделенный компонент из текущего метода обсчета с автоматическим удалением связанных с ним калибровочных точек.
* **Интерактивное поле «Поиск»:** Поле контекстного фильтра. При вводе первых символов наименования вещества программа мгновенно скрывает несовпадающие строки в основной рабочей таблице метода, что критически важно при работе со сложными многокомпонентными смесями (более 50 веществ).

### Группы (Функциональные и фракционные соединения)

Используется для создания логических объединений индивидуальных компонентов (например, «Сумма ароматических углеводородов», «Фракция С6+») с целью расчета их суммарных площадей и концентраций.

* **Элемент \[Создать] (значок `+`):** Создает новую корневую группу в структуре метода. После создания оператор может перетаскивать (методом _Drag-and-Drop_) индивидуальные вещества из левого блока в созданную группу.
* **Элемент \[Удалить] (значок `–`):** Удаляет выбранную группу соединений. При этом входящие в неё индивидуальные компоненты сохраняются в базовом списке метода.
* **Интерактивное поле «Поиск»:** Осуществляет быструю навигацию по дереву созданных функциональных групп и фракций.

{% hint style="info" %}
При выделении любого вещества в блоке «Компоненты», программа автоматически подсвечивает на графическом поле хроматограммы соответствующий ему пик и его калибровочные ворота (окно идентификации). При выделении элемента в блоке «Группы» на графике подсвечиваются все пики, включенные в данную группу.
{% endhint %}

### Добавление компонента

Основные операции

**Как добавить компонент в метод:**

1. Нажмите кнопку **«Создать»** под заголовком «Компоненты».
2. В появившейся строке введите **Название** (например, _Метан_).
3. Укажите **Ожидаемое время** (в минутах), которое программа будет использовать для поиска пика на хроматограмме.
4. Настройте **Окно идентификации** — временной диапазон вокруг ожидаемого времени (например, ±5%), в пределах которого пик будет считаться именно этим веществом.

{% hint style="info" %}
Используйте поле **«Поиск»**, если в вашем методе более 20-30 компонентов. Это значительно ускоряет навигацию при корректировке времен удерживания.
{% endhint %}

## Параметры

Подраздел **«Параметры»** является основным справочником метода, где для каждого анализируемого вещества жестко задаются критерии его узнавания и принадлежность к аппаратным каналам.

Для каждого компонента заполняются следующие обязательные поля:

* **Имя компонента:** Уникальное текстовое наименование вещества, которое будет использоваться во всех расчетных таблицах и итоговом протоколе.
* **Тип компонента:** Определяет роль пика в математической модели метода.
  * Внутренний стандарт - компонент для расчета по методу внутреннего стандарта или компонент, относительно которого рассчитываются коэффициенты чувствительности по методу внутренней нормализации;&#x20;
  * Реперный - компонент или компоненты, относительно которых рассчитываются относительные времена, объемы удерживания или индексы;&#x20;
  * Основной - компонент, количество которого рассчитывается как разница между количеством пробы и суммарным количеством остальных компонентов или же используется как дополнительная метка компонента при некоторых видах дополнительных расчетов;&#x20;
  * Общий - условный компонент, к которому будут отнесены все неидентифицированные пики хроматограммы&#x20;
  * Свидетель - компонент, относительно которого корректируется объем введенной пробы, как правило это компонент с большим содержанием в пробе, например, этиловый спирт в водке;&#x20;
  * Запрет идентификации - данный пик не будет идентифицироваться на хроматограмме, чтобы устранить его из расчетов (вместо удаления пика или компонента);&#x20;
  * Специальный - компонент, концентрация которого выводится отдельной строкой в панели таблиц главного окна во вкладке Расчет,&#x20;
  * Подтверждающий компонент применяется для идентификации на двух каналах детекторов. Обычным (опорным) компонентом, принято выбирать на том канале, где пик компонента лучше отделяется от других.
* **Время удерживания:** Опорное значение времени в минутах, соответствующее максимуму концентрации компонента на выходе из колонки.
* **Окно времени (Временной допуск):** Интервал, определяющий границы поиска пика относительно заданного времени удерживания.
* **Рабочий детектор (каналы):** Указание конкретного детектора (ПИД, ДТП и др.), на котором будет производиться поиск и расчет данного компонента.
* **Группа (принадлежность):** Поле выбора, позволяющее на этапе создания компонента сразу включить его в состав определенной расчетной группы. Это обеспечивает автоматическое суммирование концентрации компонента в соответствующую строку отчета (например, отнесение «Бензола» к группе «Ароматика»).

## <mark style="color:$danger;">Свойства</mark>

Подраздел **«Свойства компонента»** включает в себя физико-химические константы, необходимые для сложных пересчетов и автоматического контроля качества результатов.

<mark style="color:$danger;">Задайте соответствующие свойства компонента, если они необходимы для выбранного количественного расчета, а также пределы концентраций компонента, при выходе за которые в таблице Компонентов значения концентраций вне указанных границ будут выделены красным цветом. - нетхром2</mark>

Данные параметры используются программным комплексом для автоматического пересчета концентраций (например, из объемных долей в массовые) и верификации полученных данных на соответствие нормам.

* **Молекулярная масса:** Задается для каждого компонента в а.е.м. Используется при расчетах, где требуется переход от молярных величин к массовым.
* **Массовый коэффициент:** Индивидуальный множитель, учитывающий относительную чувствительность детектора к данному веществу. Применяется для коррекции площадей пиков при использовании метода внутренней нормализации.
* **Давление насыщенных паров:** Вводится для компонентов, требующих учета летучести при специфических методах анализа (например, при расчете состава сжиженных газов или парофазном анализе).
* **Плотность:** Задается в выбранных единицах измерения (**гр/см³** или **кг/м³**). Параметр необходим для корректного пересчета результатов из объемных единиц в массовые и наоборот.
* **Пределы концентрации (от/до):** Задание диапазона допустимых значений для данного компонента.
