# Что такое метод

## Рабочий метод

**Метод** — это группа параметров, необходимых для анализа определенной пробы.&#x20;

Поскольку каждый тип пробы по разному ведет себя в газовом хроматографе (некоторым пробам требуется более высокая температура термостата, а другим необходимо пониженное давление газа или другой детектор) для каждого определенного типа анализа необходимо создать свой уникальный метод.

**Рабочий метод** позволяет осуществлять полное управление хроматографом и подключаемыми к нему периферийными устройствами.&#x20;

Рабочий метод <mark style="color:$danger;">хранится в папке "Методы" каждого хроматографа в</mark> виде файла с расширением \*.metx. С его помощью проводится анализ и снимается хроматограмма.&#x20;

{% hint style="info" %}
Для каждого подключенного прибора должен быть создан отдельный метод. Следует иметь в виду, что метод, созданный для основного прибора, не запустится на сателлите и наоборот.
{% endhint %}

Некоторые из параметров, сохраняемых в методе, определяют, как будет обработана проба при использовании метода.&#x20;

Параметры, содержащихся в методе:

* [паспорт метода](chto-takoe-metod.md#pasport): тип хроматографа, имя, комментарии;
* условия проведения анализа (режим работы хроматографа): [рабочие детекторы](chto-takoe-metod.md#detektory), [время анализа](chto-takoe-metod.md#detektory), [температуры](chto-takoe-metod.md#temperatury), [расходы газов](chto-takoe-metod.md#raskhod-gaza), [параметры продувки](chto-takoe-metod.md#produvka) и т.д.;
* [тип количественного расчета](chto-takoe-metod.md#raschyoty);
* анализируемые [компоненты](chto-takoe-metod.md#komponenty-i-gruppy) и их характеристики (наименование, время удержания, окно поиска,  канал детектирования, метод расчета площадей пиков, физико-химические свойства и т.д.);
* [группы](chto-takoe-metod.md#komponenty-i-gruppy) компонентов и их характеристики;
* [параметры обработки](chto-takoe-metod.md#obrabotka) хроматограмм: ширина пиков, порог обнаружения пиков, фильтрация шумов, настройка событий интегрирования;
* [параметры идентификации пиков](chto-takoe-metod.md#obrabotka);
* [внешние устройства](chto-takoe-metod.md#konfiguraciya-ustroistv) и параметры их работы.

Анализ данных и <mark style="color:$danger;">параметры отчета</mark> также сохраняются в методе при его создании. Эти параметры описывают, как интерпретировать хроматограмму, созданную для пробы, и <mark style="color:$danger;">какой тип отчета необходимо напечатать.</mark>

{% hint style="info" %}
При работе с методом условия проведения анализа из него передаются в хроматограф. **Метод записывается в хроматограммах**, снимаемых по текущему методу.
{% endhint %}

Хроматограмма сохраняется в виде файла с расширением \*.chrx. В тот же файл хроматограммы записывается и рабочий метод со всеми параметрами(температуры колонок, испарителей, детекторов, расходы газов, параметры обработки и др.). Этот метод носит название - **метод хроматограммы**.

{% hint style="info" %}
Просматривая хроматограмму, можно отредактировать параметры **метода хроматограммы**, а в последующем перенести изменения в **рабочий метод**, по которому данная хроматограмма была записана.
{% endhint %}
