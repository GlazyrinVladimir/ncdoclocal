# Паспорт

### Паспорт

Вкладка «Паспорт» предназначена для документирования метаданных хроматографического анализа, ведения сопутствующей учетной информации и обеспечения метрологической прослеживаемости результатов измерений. Вносимые в данный раздел сведения выполняют идентификационную функцию, не оказывают прямого влияния на физические параметры работы узлов хроматографа «Кристаллюкс-4000М», однако используются в качестве обязательных реквизитов при автоматическом формировании итогового протокола (отчета) анализа.

Интерфейс вкладки включает в себя следующие информационные поля и группы параметров:

* **«Хроматограф»** — нередактируемое информационное поле, в котором автоматически отображается наименование подключенной модели аналитического прибора (например, _Кристаллюкс-4000М_). Заполняется программным обеспечением «NetChrom» на этапе инициализации связи с аппаратной частью. Перед началом работы необходимо визуально верифицировать соответствие указанной модели фактическому типу оборудования.
* **«Модуль детекторов»** — информационное поле, отображающее текущую аппаратную конфигурацию измерительных каналов хроматографа (например, _ПИД-ПИД_). Данные считываются из конфигурации прибора автоматически.&#x20;

{% hint style="info" %}
Данное поле заполняется программным обеспечением «NetChrom» автоматически на этапе инициализации связи с прибором.&#x20;

Перед началом эксплуатации метода в обязательном порядке должна быть выполнена верификация соответствия типа программного модуля текущей физической конфигурации хроматографа «Кристаллюкс-4000М».
{% endhint %}

* **«Название метода»** — текстовое поле, отображающее имя текущего файла рабочего метода с расширением `*.metx` (например, _ПИД-ПИД.metx_). Служит для однозначной идентификации файла метода в базе данных и файловой структуре ПО.
* **«Комментарий»** — многострочное текстовое поле свободной формы. Предназначено для внесения оператором сопутствующих технологических заметок, наименования методики выполнения измерений (ГОСТ, МВИ), описания серии анализируемых пробоподготовок или других текстовых данных, необходимых для обеспечения прослеживаемости. Информация из данного поля автоматически выгружается в финальный отчет.
