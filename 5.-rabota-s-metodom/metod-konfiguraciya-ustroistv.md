# Метод — Конфигурация устройств

Раздел **«Конфигурация устройств»** предназначен для детальной настройки параметров внешних устройств, поставляемых с хроматографом..

Список доступных устройств в этом окне формируется автоматически на основании **общей конфигурации прибора**. Если какое-либо устройство отсутствует в списке, его необходимо предварительно активировать в конфигурации хроматографа.

{% hint style="warning" %}
Содержимое вкладки зависит от настроек в Конфигурации хроматографа.
{% endhint %}

## Дозатор автоматический газовый (ДАГ)

Вкладка **«ДАГ»** предназначена для настройки параметров работы автоматических дозаторов газовых проб. Настройки данного раздела определяют алгоритм забора пробы, объем и количество циклов анализа.

### **Параметры выбора устройства**

* **Тип устройства** (верхний выпадающий список): Выбор модификации установленного дозатора (**ДАГ-3, ДАГ-4, ДАГ-10/12**). От выбора типа зависят доступные алгоритмы управления.
* **Активные модули** (второй выпадающий список): Позволяет выбрать конкретный ДАГ из списка доступных в системе («Первый», «Второй», «Третий»), если в конфигурации прибора задействовано несколько дозаторов.

{% hint style="info" %}
При работе сателлитом и наличии двух ДАГ, если для первого хроматографа выбирается ДАГ номер 1, то на втором хроматографе (сателлите) - номер 2.
{% endhint %}

### **Режимы запуска и повтора**

* **Повторять непрерывно**: При включении данного режима (переключатель в положении «Вкл.») система будет автоматически инициировать новый цикл анализа сразу после завершения предыдущего. Используется для постоянного мониторинга потока.
* **Ручной старт серии анализов**: Активирует возможность ручного запуска последовательности анализов оператором.

Группа «**Параметры пробы**»

В данной группе задаются количественные характеристики цикла дозирования:

* **Объем пробы (мл.)**: Объем газовой пробы, который будет отобран дозатором для ввода в колонку.
  * Для точного определения этого значения воспользуйтесь функцией **«Калькулятор объема газовой пробы»** (вызывается нажатием кнопки рядом с полем ввода). Калькулятор позволяет рассчитать объем на основании физических параметров петли и условий отбора.
* **Число анализов**: Количество повторных циклов ввода пробы из одной серии.
  * Для непрерывных анализов включите переключатель Повторять непрерывно. Тогда число анализов будет неограниченно, а строка с числом анализов - недоступна.
* **Время отбора (сек.)**: Интервал времени, в течение которого происходит заполнение дозирующей петли или емкости пробой.

### **Время переключения** (сек.)

Здесь настраиваются временные задержки для исполнительных механизмов конкретных модулей (**ДАГ-1, ДАГ-2, ДАГ-3**).

* Параметр определяет время, необходимое для физического перемещения привода дозатора в рабочее положение и обратно.
* **Примечание:** Значения устанавливаются в соответствии с техническим паспортом на конкретную модель дозатора (обычно настраиваются один раз при пусконаладочных работах).

### Порядок настройки ДАГ

Для подготовки автоматического дозатора к работе выполните следующие действия:

1. **Выбор модели**: В верхнем раскрывающемся списке выберите тип установленного дозатора (например, **ДАГ-3**).
2. **Выбор активного модуля**: Во втором списке выберите порядковый номер дозатора в системе (**Первый**, **Второй** и т.д.).
3. **Выбор режима запуска**:&#x20;

* Активируйте **«Ручной старт серии анализов»**, если планируете запускать цикл по команде оператора.
* Активируйте **«Повторять непрерывно»**, если требуется автоматический циклический анализ потока.

4. **Расчет и ввод объема**:

* При необходимости нажмите на иконку **Калькулятора** рядом с полем «Объем пробы».
* Введите полученное значение в поле **Объем пробы (мл.)**.

5. **Задание цикла**: Укажите **Число анализов** и **Время отбора**.
6. **Настройка временных задержек**: В блоке **«Время переключения»** установите время срабатывания приводов (в соответствии с паспортом прибора).

## !!!!!!!!Испаритель/метанатор

Используется для включения режима программирования температуры испарителей.&#x20;

В данном окне введите:

·укажите начальную температуру испарителя для режима программирования;

·укажите время изотермического участка при данной температуре;

·введите значение скорости программирования температуры;

·задайте температуру, которую с данной скоростью надо достичь.

## Кран обратной продувки/переключающий кран

Вкладка **«Краны»** предназначена для управления пневматическими или электрическими кранами-переключателями и кранами обратной продувки, установленными на хроматографе. Настройки данной вкладки определяют состояние кранов в момент запуска метода и их временные характеристики.

В зависимости от аппаратной конфигурации, краны разделены на группы по их функциональному назначению.

### Краны обратной продувки

Используются для изменения направления потока газа-носителя в колонке (предколонке) для сброса тяжелых компонентов или защиты аналитической колонки.

* **Кран (№)**: Порядковый номер физического подключения крана к контроллеру.
* **Время переключения**: время переключения крана на обратную продувку.
  * _Примечание:_ Справа от поля ввода из выпадающего списка можно выбрать единицы измерения (**сек.** или **мин.**).
* **Состояние (Вкл./Откл.)**: Переключатель активации крана в текущем методе.
  * **Вкл**: Кран задействован в методе. Программное обеспечение будет подавать сигналы управления на данный кран согласно временной программе.
  * **Откл**: Кран деактивирован. Управление данным механизмом в рамках текущего метода не осуществляется.

### Переключающие краны

_Примечание: Данный тип устройств настраивается аналогично кранам обратной продувки, но имеет расширенные временные параметры._

Для переключающих кранов в интерфейсе доступны **два параметра времени переключения**:

* **Время переключения 1**: Время перехода из исходного состояния в рабочее.
* **Время переключения 2**: Время возврата крана в исходное состояние.

Эти параметры позволяют более точно настроить работу механизмов с разной скоростью срабатывания или специфическими требованиями к циклу дозирования.

**Порядок настройки**

1. Установите исходное состояние для каждого крана, задействованного в методе.
2. Введите время переключения в соответствии с техническими данными установленных приводов.
3. **Важно:** Убедитесь, что выбранные единицы измерения (минуты или секунды) соответствуют вашему алгоритму анализа.









## !!!!!!!Дозатор равновесного пара (ДРП)

В данной вкладке укажите параметры работы дозатора равновесного пара:

·укажите температуры вводимой пробы и крана;

·время наддува газа-носителя в пробу и отбора пробы;

·объем вводимой газовой пробы;

·число анализов из одного контейнера с пробой.

## !!!!!!!!!Автоматический дозатор сниженного  газа (ДАСГ)

В данной вкладке устанавливается режимы работы автоматического дозатора сжиженного газа:

·номер испарителя: первый или второй

·объем пробы;

·число анализов;

·время отбора пробы;

·время переключения в положение отбора;

·При выборе ручного старта серии анализов, анализ начинается не после выхода прибора на готовность, а при нажатии кнопки старт.

## !!!!!!!!!!Клапан Динса

·по необходимости напишите комментарий.
