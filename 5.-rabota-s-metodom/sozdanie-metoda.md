# Создание метода

## Создание метода (файловый менеджер)

В текущей версии ПО создание и хранение методов осуществляется через единый **Файловый менеджер**, который обеспечивает централизованный доступ к данным.

{% hint style="info" %}
Перед созданием метода необходимо убедиться, что выбран соответствующий активный прибор. Файловый менеджер позволяет переключаться между доступными хроматографами в процессе работы.
{% endhint %}

***

**Для создания рабочего метода выполните следующие действия:**

1. **Запуск менеджера:** На стартовой странице программного комплекса нажмите кнопку **Открыть**. Будет запущено окно файлового менеджера. <img src="../.gitbook/assets/image (25).png" alt="" data-size="line">

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (4).png" alt=""><figcaption><p>Рисунок 5.1 - стартовая страница</p></figcaption></figure></div>

{% hint style="info" %}
Файловый менеджер автоматически откроется для хроматографа, активного на стартовой странице.
{% endhint %}

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (28).png" alt=""><figcaption><p>Рисунок 5.2 - файловый менеджер</p></figcaption></figure></div>

2. **Выбор оборудования:** В верхней части окна в поле **Прибор** проверьте активный хроматограф. Если требуется создать метод для другого устройства, выберите его из выпадающего списка.

3. Нажмите кнопку **(`+`)** "Создать файл" и выберите пункт "Создать метод":

## **Альтернативный способ создания метода (управление хроматографом)**

Создание нового рабочего метода также возможно непосредственно из окна управления хроматографом (панель **«Запуск анализа»**). Этот способ позволяет оперативно перейти к настройке параметров, не закрывая текущую рабочую область.

**Порядок действий:**

1. В окне управления хроматографом перейдите в раздел **Запуск анализа**.
2. Найдите блок **Параметры запуска**, в котором расположено поле выбора метода.
3. Нажмите кнопку **Создать** (иконка с изображением чистого листа) - будет осуществлен автоматический переход к новосозданному методу.
4. После настройки всех параметров нажмите **Передать** (иконка бумажного самолетика), чтобы прибор принял новые установки и начал подготовку к анализу.

{% hint style="info" %}
Кнопка **Передать** становится активной только после того, как в методе заполнены все критические параметры и файл сохранен в базе данных.
{% endhint %}

## Структура метода

В появившемся окне метода отображается древовидная структура параметров, определяющая конфигурацию и логику выполнения всего цикла хроматографического анализа. Каждый элемент данной структуры представляет собой специализированный модуль данных, предназначенный для хранения и редактирования строго определенных групп параметров:

* **Паспорт метода** — является информационным заголовком файла метода. Раздел предназначен для ввода идентификационных данных аналитической задачи (наименование методики выполнения измерений, текстовое описание, фамилия оператора). В данном модуле также осуществляется автоматический контроль соответствия текущей аппаратной конфигурации хроматографа «Кристаллюкс-4000М» и выбранного программного шаблона.&#x20;
* **Параметры управления хроматографом** — инструментальный блок, регламентирующий физические условия проведения разделения. Раздел включает в себя настройки температурных режимов (термостата колонок, испарителей, детекторов), параметры регулирования расходов и давлений газов (газа-носителя, водорода, воздуха), временную программу управления исполнительными механизмами (переключение газовых или жидкостных кранов, управление обратной продувкой), а также специфические рабочие параметры установленных детекторов (ток питания ДТП, задержки поджига ПИД).
* **Параметры обработки хроматограммы** — математический блок, задающий алгоритм разметки зарегистрированного аналитического сигнала. Включает в себя временную таблицу параметров интегрирования (минимальная площадь и ширина пиков, критерии отделения пиков от шума, моменты фиксации и коррекции дрейфа базовой линии, интервалы запрета интеграции шумов и растворителя).
* **Компоненты и группы компонентов** — градуировочный (калибровочный) модуль, предназначенный для качественной идентификации целевых веществ. В данном разделе формируется список определяемых компонентов с указанием их абсолютных или относительных времен удерживания, а также окон идентификации. При реализации сложных групповых анализов (например, определение суммарных углеводородов) в этом же блоке создается иерархический список групп компонентов с индивидуальными критериями суммирования площадей.
* **Параметры расчёта** — метрологический блок, определяющий математический аппарат для количественного расчета концентраций. На основе введенных градуировочных стандартов здесь формируются калибровочные таблицы, задается тип аппроксимации калибровочных зависимостей (линейная, квадратичная, дробно-линейная), выбирается метод расчета результатов (метод абсолютной градуировки, внутренней нормализации, внутреннего или внешнего стандарта), а также настраиваются единицы измерения (мг/дм³, %, ppm) и формат представления итогового протокола анализа.
* **Параметры дополнительных устройств** — конфигурационный блок сопряжения с периферийным оборудованием. Модуль служит для задания алгоритмов работы и синхронизации с хроматографом внешних узлов, таких как автоматические дозаторы жидких проб (автосамплеры), дозаторы сжиженного газа, термостаты переключающих кранов, пиролитические приставки или системы пробоподготовки.
* **Параметры звуковых сигналов** — сервисный блок, предназначенный для настройки аудиооповещения оператора о ходе выполнения анализа. Раздел позволяет назначить специфические звуковые сигналы на ключевые технологические события (готовность прибора к анализу, старт записи хроматограммы, окончание времени анализа, выход параметров за границы установленных допусков или возникновение аварийных ситуаций).
