## Детекторы

Раздел **Детекторы** предназначен для выбора детекторов, участвующих в анализе, и настройки их ключевых рабочих параметров.

**Описание параметров**

* **Время анализа** — поле для задания общей длительности регистрации хроматографического сигнала (в минутах). Данное значение определяет период сбора данных со всех активных детекторов.
* **Рабочие детекторы** — список доступных детекторов хроматографа. Для включения детектора в текущую конфигурацию измерения необходимо установить флаг (чекбокс) напротив его наименования.
* **Ток, мА** — область настройки электрических параметров детекторов по теплопроводности. В поле ввода указывается значение тока питания моста (в миллиамперах) для выбранного детектора (например, **ДТП**).

{% hint style="info" %}
Используйте для активации только нужных детекторов в методе.

<mark style="color:$danger;">Если детектор не выбран в этом разделе, запись хроматограммы по данному каналу производиться не будет.</mark>
{% endhint %}

### Дополнительные параметры детектора ДТП&#x20;

При работе с ДТП необходимо  задать значение тока моста.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (166).png" alt=""><figcaption><p>Ток ДТП</p></figcaption></figure></div>

{% hint style="info" %}
Величина тока моста зависит от температуры детектора, расхода газа - носителя и типа газа-носителя. Чем больше значение тока моста, тем выше чувствительность детектора. Однако это значение не должно превышать:

* **75 мА**, если газ-носитель - **гелий или водород**,&#x20;
* **40 мА** для **аргона**
* **30мА для азота**.&#x20;
{% endhint %}

{% hint style="danger" %}
**Большее значение тока ДТП может привести к выходу детектора из строя.**&#x20;
{% endhint %}

При достижении значения тока моста больше, чем указано в разделе блокировка конфигурации хроматографа, на этапе подготовка или готовность в окне управления хроматографом во вкладке состояние будет выведено соответствующее предупреждение и хроматограф перейдет на нулевой этап работы.&#x20;

<mark style="color:$danger;">Для более жесткого контроля задания тока моста в программе по умолчанию установлено ограничение значения тока ДТП в зависимости от выбранного в конфигурации хроматографа типа газа - носителя, а именно какое бы значение тока моста не указывалось, программа не передаст хроматографу больше максимально допустимого значения для данного газа-носителя.</mark>

{% hint style="warning" %}
При использовании азота наличие даже следов кислорода (воздуха) при высоком токе быстро разрушает вольфрамовую нить.

Никогда не включайте ток моста, если через детектор не идет **поток газа**. Без охлаждения потоком нити сгорают мгновенно.
{% endhint %}

### <mark style="color:$danger;">Дополнительный параметр детектора ЭЗД, ФИД и др.</mark>

<mark style="color:$danger;">При работе с ЭЗД указывается чувствительность.</mark>&#x20;

{% hint style="info" %}
<mark style="color:$danger;">Чувствительность ЭЗД подбирается в зависимости от анализируемых веществ, таким образом, чтобы произведение коэффициента тока (установленного в Конфигураторе хроматографа во вкладке ЭЗД) и чувствительности ЭЗД не превышало значения 4000.</mark>&#x20;

<mark style="color:$danger;">Как правило, при работе с</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">**капиллярной колонкой**</mark> <mark style="color:$danger;">**коэффициент тока имеет значение от 17 до 20, а чувствительность - 100%.**</mark>&#x20;

<mark style="color:$danger;">При работе с</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">**насадочной колонкой**</mark> <mark style="color:$danger;">**значение коэффициента тока находится в пределах 17-20, чувствительность от 30 до 60%.**</mark>
{% endhint %}

## Температуры

Раздел **Температуры** предназначен для программирования нагрева основных узлов прибора: термостата колонок, испарителей (инжекторов) и детекторов. Настройки этого раздела определяют скорость прохождения пробы через систему и форму получаемых пиков.

**Описание параметров**

* **Колонка, °C** — задание температуры термостата колонок. В случае программирования температуры здесь указывается начальное значение.
* **Детектор (1, 2, 3), °C** — поля для задания рабочих температур установленных детекторов.

{% hint style="info" %}
Примечание: Температура детектора должна исключать конденсацию компонентов пробы и фазы, обычно она на 20-30 °C выше температуры колонки.
{% endhint %}

* **Испаритель (1, 2, 3), °C** — задание температуры узлов ввода пробы. Устанавливается исходя из температур кипения компонентов анализируемой смеси.
* **Рабочие колонки** — блок управления потоками и регистрацией сигналов для конкретных колонок:
  * **Вкл.** — колонка задействована в анализе, поддерживается заданный расход газа-носителя.
  * **Откл.** — колонка деактивирована.

{% hint style="info" %}
При работе с устройством захолаживания термостата колонок (кондиционером) возможен ввод значения температуры термостата колонок равное 0 град. при условии, что в Конфигурации хроматографа включен **переключатель Кондиционер**.&#x20;

При работе с крио клапаном возможен ввод отрицательного значения температуры термостата колонок при условии, что в Конфигурации включен **переключатель Крио клапан**.&#x20;
{% endhint %}

### Программирование температуры колонки

Этот блок предназначен для управления тепловым режимом термостата колонок в ходе анализа. Правильная настройка программы температуры позволяет ускорить выход «тяжелых» компонентов и улучшить разделение сложных смесей.

**Управление**

* **Переключатель «Вкл»:**
  * Если переключатель активен, прибор будет следовать заданной в таблице программе (менять температуру по шагам).
  * Если выключен — термостат будет поддерживать одну постоянную температуру, указанную в параметрах метода (изотерма).



{% hint style="info" %}
_<mark style="color:$primary;">Использование градиента</mark>_ <mark style="color:$primary;"></mark><mark style="color:$primary;">позволяет сократить время анализа и «сжать» пики тяжелых компонентов, делая их более четкими.</mark>
{% endhint %}

<figure><img src="../.gitbook/assets/image (16).png" alt=""><figcaption><p>Рисунок 5.11 - программируемые температуры колонки</p></figcaption></figure>

* Управление ступенями
  * **Кнопка (`+`):** Добавляет новую ступень (линию) в программу. Это позволяет создавать сложные многоступенчатые графики нагрева.
  * **Кнопка (`-`):** Удаляет выбранную ступень из программы.

**Параметры таблицы программирования**

Каждая строка в таблице представляет собой этап (ступень) нагрева или выдержки.

* **Температура (°C):** Начальная температура текущего этапа. Для первой строки это стартовая температура анализа.
* **Конечная температура (°C):** Целевое значение, до которого термостат должен нагреться (или остыть) на данном этапе. Если начальная и конечная температуры совпадают, этот этап будет считаться изотермическим (выдержка).
* **Время (мин.):** Продолжительность этапа. Это время, в течение которого прибор удерживает температуру на заданном уровне (после достижения «Конечной температуры»).
* **Скорость подъема (°C/мин.):** Скорость, с которой прибор переходит от начальной к конечной температуре.
  * Чем выше скорость, тем быстрее нагревается термостат, но важно учитывать технические возможности нагревателя.
* **Рассчитанное время анализа (мин.)** — информационное поле, отображающее суммарную длительность всех этапов программы в минутах.

{% hint style="info" %}
В некоторых методиках необходимо понижение температуры термостата колонок во время анализа. <mark style="color:$danger;">Для этого необходимо указать отрицательную скорость программирования, т. е. со знаком минус. (</mark><mark style="color:$danger;">**надо или не надо я уже не знаю)**</mark>
{% endhint %}

<figure><img src="../.gitbook/assets/image (179).png" alt=""><figcaption></figcaption></figure>

Типовые сценарии настройки:

1. **Линейный нагрев:** Задается начальная температура (например, 50°C), конечная (например, 250°C) и скорость (например, 10°C/мин).
2. **Нагрев с выдержкой:** Сначала задается этап с одинаковыми температурами на 2–3 минуты (чтобы зафиксировать легкие компоненты), а затем добавляется строка с ростом температуры.

{% hint style="info" %}
При настройке программы следите, чтобы «Конечная температура» в любой из строк не превышала максимально допустимую температуру для вашей колонки (указана в её паспорте или на бирке).
{% endhint %}

## Расход газа

{% hint style="warning" %}
Содержимое вкладки зависит от настроек в Конфигурации хроматографа.

Наименования каналов соответствуют именам, заданным в конфигурации прибора. В зависимости от настроек они могут отображаться как «Газ-1», «Сброс» и т.д.
{% endhint %}

Вкладка **«Расходы газов»** предназначена для управления потоками газовых сред внутри хроматографа. Параметры этого раздела определяют скорость движения пробы по колонке, режим ввода пробы в испарителе и подачу газов для работы детекторов.

### Задание расходов газов для насадочных колонок

Для насадочных колонок управление газовыми потоками осуществляется путем прямого задания объемного расхода (мл/мин). В данном режиме система поддерживает установленные значения стабильными на протяжении всего анализа.

**Описание параметров**

* **Газ-1, 2, 3... (мл/мин)** — поля для задания расхода газа-носителя через соответствующие линии хроматографа. В зависимости от конфигурации прибора, это могут быть:
  * **Испаритель** — основной поток газа-носителя через насадочную колонку.
  * **Газ-поддув** — дополнительный поток на выходе из колонки, который компенсирует «мертвый объем» детектора и улучшает форму пика.
  * **Обдув** — поток для продувки оптических узлов.
* **Водород (мл/мин)** — расход горючего газа для детекторов пламенного типа (ПИД, ПФД).
* **Воздух (мл/мин)** — расход газа-окислителя для поддержания пламени в детекторе.

{% hint style="info" %}
**Стабильность потока:** Прибор переходит в режим готовности только после стабилизации всех заданных расходов.

**Герметичность:** Если фактический расход или давление не достигают заданных значений, необходимо проверить систему на наличие утечек (особенно в узле крепления колонки и септы).
{% endhint %}

### Задание расходов газов для капиллярных колонок

Настройка газового режима для капиллярных колонок требует учета аппаратной конфигурации прибора, параметров колонки и настройки системы деления потока (сброса).&#x20;

**Подготовка и базовые настройки**

* **Газовая схема и регуляторы:** Должна быть правильно указана конфигурация электронных регуляторов давления (ЭРД) или расходов (ЭРР).
  * **Типы аппаратных модулей** — перед началом настройки параметров убедитесь, какой тип электронного регулятора давления (ЭРД) установлен в вашей системе:
    * **ЭРД7** — основной тип регулятора. Предназначен для поддержания заданного давления в линии **после себя** (на выходе). Используется для подачи газа в испарители и детекторы.
      * **ЭРД8** — специализированный модуль. Поддерживает давление в линии **до себя** (на входе). Обычно применяется в специальных схемах газовых потоков для сброса избыточного давления.
      * **Ручной регулятор давления** — используется в старых хроматографах.
* **Геометрия колонок:** Длина, внутренний диаметр и толщина фазы. Давление и расход жестко связаны через сопротивление капилляра; ошибка в диаметре даже в 0.05мм приведет к недостоверным результатам расчета.

{% hint style="info" %}
При замене капиллярной колонки на колонку с другим внутренним диаметром или длиной, необходимо обязательно обновить данные о геометрии колонки в методе, иначе расчет давления и расходов будет некорректным.
{% endhint %}

* **Тип газа-носителя:** Физические свойства газов (вязкость, сжимаемость) различаются. Необходимо выбрать актуальный газ: _Азот_, _Аргон,_ _Гелий_ или _Водород_.
  * **Вспомогательные газы:** Для работы пламенных детекторов (**ПИД, ТИД, ПФД**) установите требуемые расходы водорода и воздуха для поддержания пламени.

### Газовый калькулятор

Встроенный **«Газовый калькулятор»** предназначен для моделирования, расчета и автоматической оптимизации параметров газовых потоков в хроматографической системе. В основе расчета лежит математическое моделирование поведения идеального газа в капиллярных колонках с учетом сжимаемости носителя, температурных режимов и геометрии линий.

<figure><img src="../.gitbook/assets/image (186).png" alt=""><figcaption></figcaption></figure>

**Функциональные возможности калькулятора**

В интерфейсе калькулятора параметры **«Давление»** (атм) и **«Расход»** (мл/мин) являются взаимно зависимыми величинами.

* **Приоритет ввода:** Вы можете зафиксировать один из параметров. Например, если ввести целевой расход (оптимальный для текущего диаметра колонки и типа детектора), калькулятор автоматически рассчитает необходимое начальное давление на входе.
* <mark style="color:$danger;">**Линейная скорость:**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">Параллельно вычисляется средняя линейная скорость газа-носителя (см/с), что позволяет оптимизировать разделение по уравнению Ван-Деемтера.</mark>

<figure><img src="../.gitbook/assets/image (185).png" alt=""><figcaption></figcaption></figure>

* В поле **«Коэффициент деления»** введите требуемое соотношение (например, `1:2`).
* Программа, используя ранее вычисленный расход через колонку, автоматически рассчитает и установит абсолютное значение в поле **«Сброс»** (мл/мин).

### **Программирование давления**

При реализации градиентного температурного режима вязкость газа-носителя динамически изменяется при нагреве. Если удерживать давление постоянным, расход через колонку во время анализа будет неизбежно падать.

Для стабилизации хроматографического процесса калькулятор предлагает два режима:

* **Постоянный расход (Constant Flow):** Калькулятор рассчитывает временную программу изменения давления. На основании температурной программы термостата колонок прибор будет плавно и синхронно повышать давление на входе, удерживая объемный расход через колонку на строго постоянном уровне.
* **Программируемое давление (Advanced Pressure):** Позволяет оператору вручную задать ступенчатый или линейный рост давления.

Для задания изменения давления в ходе анализа нажмите кнопку **«Давление»**. В появившемся диалоговом окне настройте следующие параметры:

* **Начальное давление** — давление на входе в колонку в начале анализа.
* **Время изобарического участка** — период времени, в течение которого давление остается постоянным перед началом изменения.
* **Скорость программирования** — скорость подъема или <mark style="color:$danger;">снижения давления.</mark>
  * _<mark style="color:$danger;">Примечание: Допускается</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">**обратное программирование давления**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">(снижение потока) — для этого укажите скорость со знаком «минус».</mark>_
* **Конечное давление** — целевое значение, которое должно быть достигнуто.

Система автоматически отображает расчетное время изменения давления на основе введенных данных. Из этого же окна можно вызвать «Газовый калькулятор» для оперативной корректировки параметров.

{% hint style="warning" %}
После переноса рассчитанных данных в активный метод анализа убедитесь, что фактические показания датчиков давления на панели прибора стабилизировались и соответствуют расчетным.
{% endhint %}

### Расход газа пламенных детекторов

## Продувка&#x20;

Вкладка **«Продувка»** предназначена для автоматизации процесса пост-аналитической очистки колонок от высококипящих компонентов (ВКК), которые не являются объектами качественного или количественного анализа.

**Функциональное назначение**

Режим обеспечивает принудительный перенос соединений с большими временами удерживания из испарителя в колонку и их ускоренный выход к детектору. Использование данной вкладки позволяет:

* Минимизировать время общего цикла анализа;
* Исключить появление пиков «памяти» на последующих хроматограммах;
* Продлить срок службы аналитических колонок.

**Состав и настройка параметров**

Для активации режима необходимо перевести переключатель **«Включить продувку»** в активное состояние и заполнить следующие поля:

{% hint style="warning" %}
Наименования каналов соответствуют именам, заданным в конфигурации прибора. В зависимости от настроек они могут отображаться как «Газ-1», «Сброс» и т.д.
{% endhint %}

1. **Группа «Газы, мл/мин»:**
   * **Газ-1...Газ-N:** Значения расходов газов-носителей через каналы ЭУПГ в фазе очистки. Для эффективной «продувки» рекомендуется устанавливать расходы выше рабочих, обеспечивая максимально допустимую линейную скорость для используемого типа колонки.
2. **Давление на колонке (для капиллярных колонок):**
   * Позволяет задать избыточное давление газа-носителя на входе в колонку. Увеличение давления в этом поле напрямую ускоряет движение ВКК по аналитическому тракту.
3. **Время продувки, мин:**
   * Длительность этапа очистки, которая отсчитывается с момента завершения основного времени анализа.
4. **Температура колонки, град:**
   * Температура термостата в процессе продувки. Для ускорения выхода ВКК температуру обычно устанавливают на уровне финальной точки температурной программы (но не выше предела термостабильности фазы).

{% hint style="info" %}
При настройке параметров продувки убедитесь, что суммарный поток газа через детектор не приведет к его перегрузке или погасанию пламени (в случае ПИД/ПФД).
{% endhint %}

**Алгоритм работы прибора**

* **Инициация:** После окончания регистрации хроматограммы прибор автоматически переключает расходы газов и температуру термостата на значения, указанные во вкладке «Продувка».
* **Исполнение:** В течение заданного времени происходит интенсивное «промывание» аналитического тракта.
* **Завершение:** По истечении времени продувки прибор переходит в режим подготовки к следующему анализу, возвращаясь к исходным уставкам газов и температур, заданным в основных вкладках метода.

## Экономия

{% hint style="warning" %}
Наименования каналов соответствуют именам, заданным в конфигурации прибора. В зависимости от настроек они могут отображаться как «Газ-1», «Сброс» и т.д.
{% endhint %}

Функция **«Экономия»** предназначена для рационального использования газа-носителя без ущерба для качества хроматографического разделения. Данный режим особо актуален при работе с капиллярными колонками в режиме деления потока.

**Преимущества использования**:

* **Снижение затрат:** Расход газа из баллона в режиме ожидания или во время долгого анализа сокращается.
* **Увеличение ресурса фильтров:** Газовые фильтры очистки (от влаги и кислорода) работают значительно дольше за счет уменьшения объема прокачиваемого через них газа.

Режим экономии активируется с помощью переключателей (ползунков) для каждого этапа работы хроматографа отдельно. При включении функции **«Вкл.»** становятся доступными поля для ввода конкретных значений расходов или давления.

**Доступные этапы для настройки**:

1. **Экономия газа на этапе «Подготовка»**
   * _Когда работает:_ В период, когда прибор только выходит на заданные температурные и газовые режимы после включения или смены метода.
   * _Зачем использовать:_ Если выход на режим занимает длительное время, можно установить минимально безопасный поток газа для защиты колонки от перегрева без лишнего расхода.
2. **Экономия газа на этапе «Готовность»**
   * _Когда работает:_ Прибор уже вышел на рабочий режим и ожидает команды оператора или автосамплера на начало анализа («Старт»).
   * _Зачем использовать:_ Позволяет не тратить газ впустую, пока прибор простаивает в ожидании ввода пробы. При нажатии кнопки «Старт», программа автоматически переключит расход на рабочий (указанный в основном методе).
3. **Экономия газа на этапе «Анализ»**
   * _Когда работает:_ Непосредственно во время записи хроматограммы.
   * _Зачем использовать:_ Обычно применяется в испарителях с делением потока (Split/Splitless). Например, через несколько минут после начала анализа (после того как проба попала в колонку) можно автоматически снизить «сброс» газа в атмосферу, не влияя на качество разделения.

При переводе ползунка в положение **«Вкл.»**, рядом с названием этапа появляется возможность задать время включения режима экономии, через которое будут установлены заданные выше значения **расхода (мл/мин)**.

{% hint style="info" %}
При установке параметров экономии всегда следите за тем, чтобы поток газа-носителя через колонку оставался достаточным для предотвращения диффузии воздуха внутрь системы, особенно при высоких температурах термостата.
{% endhint %}

## Сплит

Вкладка **«Сплит»** предназначена для настройки параметров режима **ввода без деления потока** (Splitless). Настройка этой вкладки позволяет задержать включение сброса газа из испарителя, обеспечивая максимальный перенос пробы в колонку.

{% hint style="warning" %}
Наименования каналов соответствуют именам, заданным в конфигурации прибора. В зависимости от настроек они могут отображаться как «Газ-1», «Сброс» и т.д.
{% endhint %}

**Порядок настройки**:

1. **Задайте расходы газов (Газ-1...Газ-N):**\
   Установите значение **0 мл/мин** (или иное необходимое минимальное значение) для каналов сброса. Эти значения будут поддерживаться прибором в течение времени задержки.
2. **Задержка включения рабочих газов, сек:**\
   Укажите время, через которое прибор должен вернуться к основным рабочим параметрам метода. В течение этого интервала расходы будут соответствовать значениям, указанным в пункте 1. В этот период вся проба из испарителя направляется в колонку.
3. **Переключатель «Включить сплит»:**\
   Переведите в активное состояние для автоматического запуска режима при каждом вводе пробы.

**Алгоритм функционирования:**

* **Начало анализа:** После нажатия кнопки «Старт» устанавливаются расходы газов, указанные в данной вкладке (режим ввода без деления потока).
* **Исполнение задержки:** В течение заданного времени («Задержка включения рабочих газов») проба переносится из испарителя в колонку при закрытом или ограниченном сбросе.
* **Переход в рабочий режим:** По окончании времени задержки прибор автоматически устанавливает расходы газов, которые были заданы как основные в настройках метода. Происходит удаление остатков пробы из испарителя и переход к штатному режиму разделения.
