# Работа с хроматограммой

## Файловый менеджер

### Работа с журналом хроматограмм

Окно **«Открыть файл»** (Журнал хроматограмм) является основным инструментом для поиска, систематизации и загрузки ранее записанных данных в программном комплексе **NetChrom**. Система позволяет работать с большими массивами данных благодаря развитой функции фильтрации и предварительного просмотра.

### Интерфейс менеджера файлов

Окно разделено на три основные функциональные зоны:

1. **Панель фильтров** (верхняя часть) — инструменты для быстрого отбора данных по времени, методу и источнику.
2. **Таблица результатов** (центральная часть) — детализированный список хроматограмм, соответствующих выбранным критериям.
3. **Панель выбора хроматографа** (правая верхняя часть) —  переключатель активного хроматографа.

### Панель фильтров и поиска

Панель позволяет мгновенно ограничить список отображаемых файлов.

#### Временной интервал

Позволяет выбрать период, за который проводились анализы. При нажатии на поле открывается список предустановок:

* **Последние 24 часа / 7 / 14 / 30 дней** — быстрая выборка за указанный срок.
* **За все время** — отображение всего архива (может занять время при большом объеме базы данных).
* **Диапазон дат...** — вызов календаря для задания произвольного интервала.

#### Выбор метода

Фильтрует записи по имени методического файла (**\*.metx**). В списке отображаются только те методы, по которым проводились измерения на данном рабочем месте.

#### Источник данных (Основной / Сателлит)

Позволяет разделять данные, полученные с разных физических модулей или приборов в рамках одной системы.

* Выбор осуществляется установкой флажков (чек-боксов).
* Выбранные категории отображаются в заголовке фильтра в виде **тегов**. Нажатие на символ **«х»** на теге быстро сбрасывает данный фильтр.

#### Поле «Поиск...»

Обеспечивает контекстный поиск по мере ввода текста. Поиск ведется одновременно по колонкам: **«Проба»**, **«Оператор»** и **«Комментарий»**.

### Список зарегистрированных хроматограмм

В таблице фиксируется полная информация о каждом анализе. Данные вносятся автоматически в момент старта регистрации сигнала.

| Колонка          | Описание                                                       |
| ---------------- | -------------------------------------------------------------- |
| **Дата и время** | Точное время начала записи сигнала.                            |
| **Проба**        | Название образца (идентификатор), введенное оператором.        |
| **Объем**        | Количественное значение введенной пробы (мл/мкл).              |
| **Оператор**     | ФИО или учетная запись пользователя, выполнившего анализ.      |
| **Назначение**   | Категория замера (Анализ, Градуировка, Поверка).               |
| **Комментарий**  | Дополнительные сведения об условиях анализа.                   |
| **Метод**        | Название файла методики (.metx), использованной для анализа.   |
| **Тип расчета**  | Алгоритм количественного расчета (например, Внешний стандарт). |
| **Канал**        | Указание источника                                             |

{% hint style="info" %}
Клик по заголовку любой колонки позволяет отсортировать список по возрастанию или убыванию.
{% endhint %}

### Идентификация и выбор хроматографа

В правой части окна расположен блок, отвечающий за привязку данных к конкретному оборудованию.

* **Выбор прибора**: С помощью **синего символа стрелки** (рядом с иконкой прибора) можно переключиться между различными хроматографами «Кристаллюкс-4000М», подключенными к станции.
* **ID прибора**: Отображает сетевое имя или серийный номер (например, **3791\_ПИД-ДТП**).
* **Этап**: Указывает статус обработки или номер цикла (например, «Нулевой»).
* **Длительность**: Показывает общее время записи выбранной хроматограммы (ЧЧ:ММ:СС).

{% hint style="warning" %}
**Внимание!** Если при верно настроенных фильтрах список файлов пуст, убедитесь, что в правом углу выбран правильный хроматограф.
{% endhint %}

### Порядок открытия хроматограммы

1. Вызовите окно открытия файла через меню **«Файл»** — **«Открыть»**.
2. Проверьте правильность выбора **Хроматографа** в правой части окна.
3. Настройте **Временной интервал** и, при необходимости, **Фильтр по методу**.
4. Если требуется найти конкретную пробу, введите её номер или название в поле **«Поиск»**.
5. Выделите нужную строку в таблице и нажмите кнопку **«Открыть»** (или выполните двойной клик по строке).

## Окно обработки хроматограмм

\
Данное окно является основным рабочим пространством оператора при анализе результатов. Оно объединяет инструменты графического анализа, средства автоматической обработки и таблицы с расчетными данными.

### Верхняя панель инструментов (Панель управления)

Панель инструментов обеспечивает быстрый доступ к операциям с файлом и методиками расчета.

* **Открыть / Сохранить**: Команды для загрузки хроматограмм из архива и сохранения внесенных изменений (например, после ручной корректировки пиков).
* **Метод хроматограммы**: Открывает параметры метода, которые были сохранены внутри данного файла данных.
  * _Примечание:_ Изменение параметров здесь влияет только на текущую хроматограмму, не затрагивая основной файл рабочего метода.
* **Градуировка**: Переход в модуль построения калибровочных графиков по стандартным образцам.
* **Обработать**: Запуск процедуры поиска пиков (интегрирования) и расчета концентраций. Наличие **синего треугольника** позволяет выбрать режим: только расчет или полная переработка.
* **Поверка**: Специализированный модуль для контроля метрологических характеристик прибора.
* **Внешний расчет**: Передача данных в сторонние модули обработки.
* **Печать**: Формирование протокола анализа.

**Ниже расположена панель вкладок:**\
Она позволяет переключаться между открытыми хроматограммами и методами. Активная вкладка выделяется светлым фоном. Название вкладки хроматограммы обычно содержит дату и время её регистрации.

### Графическое поле (Хроматограмма)

Центральная зона, где отображается аналитический сигнал.

* **Отображение сигнала**: Кривая линия (синего цвета по умолчанию) отображает отклик детектора во времени.
* **Идентификация пиков**:
  * **Цветовая заливка (голубая область)** — указывает на площадь пика, которая была учтена при расчете.
  * **Базовая линия (розовая линия)** — линия, от которой программа отсчитывает высоту и площадь пика.
  * **Подписи** — над вершиной пика указывается его время удерживания и название компонента (например, **0.65; Водород**).
* **Легенда (вверху слева)**: Позволяет включать или отключать отображение каналов (например, **ДТП-1**).

### <mark style="color:$danger;">!!!Правая вертикальная панель (Инструменты графика)</mark>

Кнопки, расположенные вдоль правой границы графика, служат для оперативного управления отображением:



### Нижняя левая панель (Таблицы результатов)

В данной области сосредоточены данные количественного анализа, разбитые по тематическим вкладкам:

* **Пики**: Подробная таблица всех найденных пиков. Содержит колонки: время, детектор, название компонента, высота, площадь, ширина и др.
* **Компоненты**: Итоговая таблица с рассчитанными концентрациями (массовая доля, объемная доля и т.д.).
* **Интегрирование**: Список программных фильтров (пороги, шумы), которые использовались для поиска пиков на этой хроматограмме.
* **Журнал**: Список всех операций, произведенных с данным файлом.

### Паспорт хроматограммы (Нижняя правая панель)

Здесь отображается  информация об анализе, необходимая для формирования отчета.

* **Вкладка «Проба»**:
  * **Номер пробы** и **Номер анализа** — для учета в журнале лаборатории.
  * **Дата и время отбора** — фиксируются для контроля оперативной работы.
  * **Оператор** — ФИО сотрудника, выполнившего анализ.
  * **Описание** — текстовое поле для произвольных пометок.
* Вкладка «Хроматограмма» отображает технические параметры регистрации конкретного файла данных:
  * **Тип хроматографа**: Модель прибора (например, **Кристаллюкс-4000М**).
  * **Модуль детекторов**: Конфигурация установленных детекторов (например, **ПИД-ДТП**).
  * **Хроматограф**: Сетевое имя или серийный номер конкретного прибора.
  * **Дата / Время**: Точные системные данные о моменте начала записи сигнала.
  * **Комментарий**: Текстовое поле для кратких заметок к данному файлу (например, «Определение состава смеси»).
* Вкладка «Колонка» - данная вкладка содержит паспортные данные разделительной колонки, на которой был выполнен текущий анализ. Заполнение этих полей необходимо для корректного ведения лабораторного журнала и соблюдения требований методик выполнения измерений .**Основные параметры:**
* **Название колонки** — полное наименование.
* <mark style="color:$danger;">**Тип колонки**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">— выбор из выпадающего списка.</mark>
* **Производитель** — завод-изготовитель или бренд.
* **Длина (м.)** — фактическая длина колонки в метрах.
* **Диаметр (мм.)** — внутренний диаметр колонки.
* **Описание** — дополнительная информация (например, толщина неподвижной фазы или дата установки колонки в прибор).

### Индикатор состояния (Верхний правый угол)

Отображает информацию о приборе, с которого получены данные:

* **ID прибора** (например, 3791).
* **Этап и Длительность** анализа.
* **Статус сети**: Индикация связи с хроматографом.

{% hint style="info" %}
**Важно!** Любые изменения, внесенные в паспорт пробы или в границы пиков на графике, требуют нажатия кнопки **«Сохранить»** на верхней панели, иначе при закрытии вкладки данные будут утеряны.
{% endhint %}

## !!!Метод хроматограммы

**Метод хроматограммы** — это неотемлемая часть зарегистрированного файла данных (хроматограммы), содержащая полную копию параметров прибора и алгоритмов обработки, которые действовали непосредственно в момент проведения данного измерения.

#### Назначение

Главная цель интеграции метода внутрь хроматограммы — обеспечение принципа целостности данных и воспроизводимости результатов.

* **Защита от изменений:** Если рабочий метод на компьютере будет изменен или удален, хроматограмма сохранит свои первоначальные настройки.
* **Историчность:** Внутренний метод гарантирует, что при открытии файла через месяц или год лаборант увидит именно те условия (температуры, расходы, градуировочные коэффициенты), при которых сигнал был получен и рассчитан.

#### Структура метода хроматограммы

Внутренний метод условно разделен на два блока параметров, доступных для просмотра:

| Блок параметров           | Редактируемость | Описание компонентов                                                                                                                                                                   |
| ------------------------- | --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Аппаратные параметры**  | **Запрещено**   | Фактические температуры испарителя, детекторов и термостата колонок; расходы газов; тип колонки; серийный номер прибора. Записываются жестко в момент анализа и изменению не подлежат. |
| **Параметры обработки**   | **Разрешено**   | Таблица компонентов (названия пиков), времена удерживания, градуировочные коэффициенты, формулы количественного расчета, настройки интегратора (ширина пика, порог).                   |

{% hint style="info" %}
В методе хроматограммы можно изменить любые параметры, кроме параметров разделов Управление и Устройства, то есть тех параметров, которые отвечают за управление хроматографом.
{% endhint %}

#### Порядок работы с методом хроматограммы

При открытии сохраненного файла данных программа автоматически загружает именно его внутренний метод. Оператору доступны следующие операции:

* **Модификация расчета:** Вы можете изменять параметры интеграции, добавлять новые компоненты в таблицу или корректировать градуировку внутри данной конкретной хроматограммы. При этом исходный рабочий метод на диске **не меняется**.
* **Экспорт во внешний файл:** Если обработка хроматограммы была оптимизирована и эти настройки нужно применить для будущих анализов, выберите пункт меню **«Сохранить»** - **«Сохранить как...»**. Программа создаст независимый внешний файл метода.

<figure><img src="../.gitbook/assets/image (189).png" alt=""><figcaption></figcaption></figure>

* **Печать метода:** Программа позволяет вывести полную структуру внутреннего метода хроматограммы на бумажный носитель или в файл формата PDF. Печать метода является обязательным требованием при верификации результатов анализа в аккредитованных лабораториях.

<figure><img src="../.gitbook/assets/image (190).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
При отправке файла хроматограммы или переносе на другой ПК, все параметры градуировки и интеграции передаются внутри самого файла. Наличие файла внешнего метода на целевом компьютере не требуется.
{% endhint %}

## !!!Управление окном хроматограммы (масштабирование и тд)

В случае многоканальной хроматограммы графики каналов располагаются один над другим. При этом один из каналов рисуется поверх остальных и является активным. Подписи осей, относятся только к активному каналу, так как в общем случае масштабы каналов могут не совпадать и вообще иметь разные единицы измерения.&#x20;

Переместить канал наверх и, следовательно, сделать его активным, можно щелкнув мышью на его графике
