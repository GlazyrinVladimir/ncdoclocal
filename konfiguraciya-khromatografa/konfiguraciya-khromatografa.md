# Конфигурация хроматографа

## Конфигурация хроматографа

Установка конфигурации хроматографа (хроматографов, если их несколько) выполняется один раз после **добавления хроматографа** для того, чтобы настроить программу для работы с конкретным хроматографом. В дальнейшем настройки конфигурации могут изменяться пользователем по мере необходимости, чаще всего при замене аналитического модуля, при установке новой версии программы или модернизации хроматографа.&#x20;

Для настройки конфигурации хроматографа откройте "Конфигурацию" на панели управления хроматографом:

### Хроматограф

Данный раздел предназначен для установления связи с прибором и определения его базового состава.

**Параметры связи и идентификации**

* **Выбор COM-порта**: Укажите номер последовательного порта, к которому физически подключен «Кристаллюкс-4000М» (через переходник USB-RS232 или напрямую).
* **Название хроматографа**: Введите произвольное имя (например, «ГХ №1» или «Кристаллюкс\_Лаб2»). Это имя будет отображаться в главном окне и протоколах.
* **Тип хроматографа**: Поле заполнено по умолчанию (**Кристаллюкс-4000М**) и определяет набор доступных программных модулей.

**Сбор данных и автоматизация**

* **Модуль детекторов**: Выберите комбинацию установленных плат детекторов из выпадающего списка (например, ПИД-ПФД-ДТП). Кнопка **«Авто»** позволяет ПО самостоятельно опросить прибор и определить состав модулей.

{% hint style="info" %}
<mark style="color:$danger;">Если выбранный модуль не соответствует реально установленному в хроматограф, то справа от перечня  появится строка В хроматографе с названием модуля, установленного в хроматограф.</mark> Как только тип модуля будет выбран правильно, строка В хроматографе исчезнет.
{% endhint %}

* **Частота измерения**: Установите частоту опроса детектора (стандартно — 25 Гц).
  * _Примечание:_ <mark style="color:$danger;">Частота 10 Гц выбирается при работе с насадочными колонками, при этом шумы АЦП - минимальные. Частота 25 Гц выбирается при работе с капиллярными колонками. Частота 50 Гц выбирается при работе с поликапиллярными колонками, при этом шумы АЦП - максимальные. Изменение частоты доступно на нулевом этапе работы хроматографа.</mark>
* **Автостарт хроматографа без компьютера**: При активации чекбокса прибор сможет начать выполнение программы анализа по нажатию физической кнопки на блоке ГХ, даже если связь с ПК временно отсутствует.

### **Дополнительные температурные зоны**

Настройка количества независимых каналов нагрева. Правильное указание зон необходимо для того, чтобы в методе анализа отобразились соответствующие вкладки настройки температур.

* **Количество детекторов**: Укажите число установленных блоков детектирования (обычно от 1 до 3).
* **Количество испарителей**: Укажите число портов ввода пробы.
* **Количество доп. устройств**: Укажите количество прочих нагреваемых узлов (например, обогреваемые линии отбора).
