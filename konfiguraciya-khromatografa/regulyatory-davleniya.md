# Регуляторы давления

### Регуляторы давления <mark style="color:$danger;">(подробнее?)</mark>

Данная группа параметров предназначена для настройки модулей, обеспечивающих стабильное давление газов в узлах хроматографа. В ПО NetChrom реализована гибкая система переназначения каналов давления в зависимости от сложности газовой схемы.

**Типы используемых модулей (ЭРД)**

Перед настройкой определите тип установленного аппаратного модуля:

* **ЭРД7** — основной тип регулятора. Предназначен для поддержания заданного давления в линии **после себя** (на выходе).
* **ЭРД8** — специализированный модуль. Поддерживает давление в линии **до себя** (на входе).
* **ЭРД6** — версия регулятора с ручным заданием давления при помощи переменного резистора на плате.

### Конфигурирование РД для капиллярных колонок (КК)

При использовании капиллярных колонок управление давлением газа-носителя осуществляется через переназначение существующих аппаратных линий. В отличие от насадочных колонок, где задается прямой расход, для КК программа рассчитывает расход и линейную скорость на основе геометрии колонки и приложенного давления. Для этого используются соответствующие переключатели (чекбоксы) в настройках регулятора:

* **КК1 (Регулятор для первой колонки)** — используется стандартный адрес **РД-1**.
* **КК2 (Регулятор для второй колонки)** — программно активируется чекбокс «Регулятор давления 2» (КК2). Физически этот модуль устанавливается **вместо РД Воздуха**.
* **КК3 (Регулятор для третьей колонки)** — программно активируется чекбокс **Регулятор давления 3**. Физически модуль устанавливается **вместо РД Водорода**.

{% hint style="warning" %}
При использовании КК2 и КК3 соответствующие линии (Воздух/Водород) исключаются из системы автоматического регулирования давления. В этом случае подача вспомогательных газов на детекторы должна осуществляться через внешние механические регуляторы или свободные каналы РРГ.
{% endhint %}

#### Описание параметров настройки

* **Название регулятора** — идентификатор канала в ПО (например, _«РД Кап. колонка 2»_).
* **Линия** — программный адрес порта (РД-1, РД газа-носителя, РД воздуха, РД водорода).
* **Давление (атм.)** — значение давления, которое система будет поддерживать.
  * _Стандартный предел:_ **5 атм** (0,5 МПа).
  * _Специальное исполнение:_ до **10 атм** (только при наличии соответствующих аппаратных модулей).

#### Порядок настройки

* Нажмите **«Добавить регулятор»** для регистрации нового модуля РД.
* Выберите физическую **Линию** согласно пневматической схеме прибора.
* Если настраивается канал для капиллярной колонки (2 или 3), установите соответствующий флаг (**КК2/КК3**).
* Установите значение давления. Помните, что превышение порога в **5 атм** (без спец. модулей) может привести к аварийной остановке прибора.
