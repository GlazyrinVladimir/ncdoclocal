# Регуляторы расхода газа

## Регуляторы

Вкладка используется для формирования логической структуры газовой системы хроматографа «Кристаллюкс-4000М». Здесь выполняется регистрация физически установленных в приборе модулей управления потоками и их распределение по типам (расход/давление).

{% hint style="warning" %}
Настраивается согласно газовой схеме прибора.

Порядок и количество регуляторов в данном окне должны строго соответствовать заводской комплектации вашего прибора или внесённым изменениям.
{% endhint %}

<figure><img src="../.gitbook/assets/image (75).png" alt=""><figcaption></figcaption></figure>

### Регуляторы расхода

В данном блоке перечисляются модули, работающие в режиме поддержания постоянного объемного расхода (мл/мин или см³/мин).

* **Газ-1, Газ-2... Газ-N** — программные имена каналов.
* Используются преимущественно для вспомогательных газов детекторов (водород, воздух) или управления поддувом.
* **Кнопка «Добавить регулятор»** — позволяет добавить новую линию расхода при физическом дооснащении прибора дополнительными модулями РРГ.
* **Редактирование:** Для настройки детальных параметров нажмите на **стрелку развертывания** (v) в правой части строки регулятора.

### Настройка параметров РРГ

При развертывании строки регулятора открываются поля для его идентификации и определения рабочих характеристик. Точность задания этих параметров напрямую влияет на достоверность поддержания потоков.

#### **Описание полей настройки**

* **Название регулятора** — текстовое поле для идентификации канала в интерфейсе программы (например, в окне мониторинга или в паспорте метода).

{% hint style="info" %}
**Наименование:** Рекомендуется давать регуляторам осмысленные имена (например: «Сброс - Обдув ПФД»), чтобы избежать путаницы при наличии нескольких однотипных модулей и упростить дальнейшую работу с прибором.
{% endhint %}

* **Линия** — выпадающий список для выбора физического адреса модуля (номера канала в контроллере прибора).
  * Значение **РРГ-1** соответствует первому физическому порту подключения электронного регулятора на плате управления газовой системой «Кристаллюкс-4000М».
* **Тип газа** — определяет алгоритм расчета расхода.
  * В выпадающем списке выберите фактический газ, подаваемый на вход данного регулятора (**Гелий**, Азот, Водород или Аргон).

{% hint style="warning" %}
Поскольку физические свойства газов (вязкость и теплопроводность) различаются, выбор неверного типа газа приведет к значительной погрешности в установке реального расхода.
{% endhint %}

### Расширенная конфигурация регуляторов

Система управления газовыми потоками хроматографа «Кристаллюкс-4000М» построена по модульному принципу. Несмотря на то, что базово в приборе установлено 3 регулятора расхода (РРГ), их количество и назначение могут быть расширены за счет использования других слотов контроллера.

<figure><img src="../.gitbook/assets/image (172).png" alt=""><figcaption></figcaption></figure>

#### Принцип назначения линий

При выборе значения в поле **«Линия»** пользователь осуществляет программную привязку виртуального регулятора к физическому порту (слоту) на плате управления.

**Важные особенности распределения ресурсов:**

1. **Лимиты контроллера:** Общее количество активных регуляторов ограничено числом физически установленных модулей.
2. **Взаимозаменяемость (РРГ вместо РД):** В случае сложной конфигурации (например, при установке нескольких многоканальных детекторов) дополнительные модули РРГ могут подключаться к портам, стандартно зарезервированным для регуляторов давления (РД).
3. **Приоритет выбора:**
   * **РРГ-1, РРГ-2, РРГ-3** — стандартные порты для управления расходом.
   * **РРГ-Водород / РРГ-Воздух** — специализированные порты. Если в данных слотах установлены модули расхода, они выбираются здесь.
   * **РД Воздух / РД Водород** **/ РД-1** — порты, которые могут быть программно переопределены. Если вместо модуля давления в этот слот установлен модуль расхода, в списке следует выбирать соответствующую строку «РД...».

#### Правила выбора линии при сложной конфигурации

{% hint style="danger" %}
Если физический модуль (РРГ или РД) переставлен в другой слот на приборе, **НЕОБХОДИМО** изменить значение в поле «Линия» на соответствующее новому порту.
{% endhint %}

Если в приборе установлено более 3-х универсальных модулей РРГ, их программная привязка осуществляется по принципу замещения свободных физических адресов. При настройке в поле **«Линия»** следует придерживаться следующей логики соответствия:

* **Для 4-го регулятора:**
  * Добавьте новый регулятор.
  * В поле **«Линия»** выберите **«РД-1»** или **«РРГ-Водород»** (в зависимости от того, к какому физическому разъему на плате управления подключен модуль).
  * В поле **«Название»** укажите фактическую функцию, например: _«РРГ-4 (Обдув детектора 2)»_.
* **Для 5-го регулятора:**
  * Добавьте новый регулятор.
  * В поле **«Линия»** выберите **«РД Газа-носителя»** или **«РД Воздуха»**.
  * В поле **«Название»** укажите назначение, например: _«РРГ-5 (Вспомогательный)»_.

{% hint style="info" %}
При использовании схемы замещения физический модуль регулятора давления (РД) должен быть отключен от соответствующего разъема, а на его место установлен модуль РРГ. Одновременное использование одного и того же адреса (например, РД Воздуха) для управления и давлением, и расходом невозможно.
{% endhint %}

#### Таблица оперативного переназначения адресов

| Целевой программный канал | Допустимая физическая линия (адрес в списке) |
| ------------------------- | -------------------------------------------- |
| **Газ-1, 2, 3**           | РРГ-1, РРГ-2, РРГ-3                          |
| **Газ-4**                 | РД-1 / РРГ-Водород                           |
| **Газ-5**                 | РД Газа-носителя / РД Воздуха                |

### Порядок настройки регуляторов расхода (РРГ)

**Шаг 1. Определение физической конфигурации**

Перед внесением изменений в ПО необходимо точно знать количество установленных модулей РРГ и их подключение к портам на плате управления.

* **Стандартная схема:** использует линии **РРГ-1, 2, 3**.
* **Схема расширения (Газ-4):** модуль подключен к порту **РД-1** или **РРГ-Водород**.
* **Схема расширения (Газ-5):** модуль подключен к порту **РД газа-носителя** или **РД Воздуха**.

**Шаг 2. Добавление регулятора**

1. Перейдите во вкладку **«Регуляторы»** конфигурации прибора.
2. Нажмите кнопку **«+ Добавить регулятор»**. В списке появится новая строка (например, «Газ-4»).
3. Разверните настройки нового регулятора, нажав на стрелку (**v**) в правой части строки.

**Шаг 3. Идентификация и привязка линии**

1. В поле **«Название регулятора»** введите уникальное имя, отражающее его функцию (например, _«РРГ-4 Сброс ПФД»_).
2. В выпадающем списке **«Линия»** выберите физический адрес порта (согласно Шагу 1).
   * Для регулятора «Газ 4»Если в вашей конфигурации предусмотрен четвертый канал расхода, он физически подключается к портам дополнительных устройств.
     * В поле **«Название регулятора»** введите: **Газ 4** (или функциональное имя, например, _«РРГ-4 Сброс»_).
     * В выпадающем списке **«Линия»** выберите один из двух адресов (в зависимости от того, куда физически подключен шлейф модуля):
       * **РД-1** — если используется слот первого дополнительного регулятора давления.
       * **РРГ-Водород** — если используется слот, штатно зарезервированный под расход водорода.
   * Для регулятора «Газ 5»Пятый канал расхода использует адреса основных технологических магистралей.
     * В поле **«Название регулятора»** введите: **Газ 5** (или функциональное имя, например, _«РРГ-5 Поддув»_).
       *   В выпадающем списке **«Линия»** выберите один из двух адресов:

           **РД Газа-носителя** — если модуль установлен вместо основного регулятора давления.
       * **РД Воздуха** — если модуль установлен вместо регулятора давления воздуха.

**Шаг 4. Установка параметров газа**

1. В поле **«Тип газа»** выберите фактический газ, подаваемый в данную линию (**Гелий, Азот, Водород** или **Воздух**).
   * **ВНИМАНИЕ!** Ошибка в типе газа приведет к неверному расчету массового расхода и нестабильности базовой линии.
2. Убедитесь, что давление на входе в прибор (от баллона или генератора) превышает максимальное рабочее давление в линии минимум на **1–2 атм**.

**Шаг 5. Проверка**

1. Перейдите в окно **«Монитор»** или **«Состояние»**.
2. Задайте тестовое значение расхода (например, 30 мл/мин).
3. Убедитесь, что фактическое значение расхода достигло заданного и стабилизировалось в течение 1–2 минут.
