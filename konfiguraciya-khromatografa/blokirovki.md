# Блокировки

## Блокировки

Данная вкладка предназначена для установки аппаратно-программных ограничений, обеспечивающих защиту узлов хроматографа от перегрева и предотвращение выхода из строя дорогостоящих расходных материалов (колонок, сорбентов, сенсоров детекторов).

**Общий алгоритм защиты**

Система непрерывно сравнивает текущие значения датчиков с установленными пределами. Если температура любого из узлов превысит заданный максимум:

1. Выводится аварийное сообщение: **«Температура превышает максимум»**.
2. Прибор автоматически переходит в **«Нулевой этап»** (отключение нагрева).
3. Включается принудительное охлаждение (открытие заслонок термостата, работа вентиляторов).

#### Описание параметров

1\. **Максимальная температура колонок**

Устанавливает верхний предел нагрева термостата колонок.

* <mark style="color:$danger;">**Диапазон:**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">от 70 до 425 °С.</mark>
* **Настройка:** Определяется максимальной рабочей температурой сорбента или фазы используемой колонки.
* **Рекомендация:** Во избежание деградации неподвижной фазы рекомендуется устанавливать предел на **10–20 °С ниже** паспортного значения максимальной температуры колонки.

{% hint style="info" %}
Предельная температура термостата колонок определяется максимальной температурой сорбентов используемых колонок.
{% endhint %}

2\. **Максимальная температура испарителя**

Определяется химической природой анализируемых веществ и конструктивными особенностями узла ввода.

* **Стандартное значение:** обычно не выше **350 °С**.
* **Особые случаи:**
  * Для испарителей с покрытием **Sulfinert**: предел **250 °С**.
  * При использовании **кранов-дозаторов** в качестве узла ввода: предел **100 °С**.

3\. **Максимальная температура детекторов**

Предельное значение устанавливается исходя из конструкционной термостойкости конкретного типа детектора:

для ПИД, ТИД, ЭЗД - не выше 400°С.

для ДТП, ПФД, ФИД - не выше 250°С.

#### Порядок установки блокировок

1. Определите паспортные ограничения для всех установленных колонок и узлов.
2. Введите значения в соответствующие поля вкладки «Блокировки».
3. &#x20;Данные будут записаны автоматически.
