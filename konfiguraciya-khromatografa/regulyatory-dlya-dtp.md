# Регуляторы для ДТП

## Подключение ДТП

Подмодуль **«Подключение ДТП»** расположен в нижней части общего диалогового окна конфигурации регуляторов пневматики. Данный программный блок предназначен для аппаратно-логической привязки измерительных ячеек детектора по теплопроводности (ДТП / катарометра) к конкретным электронным регуляторам расхода газа (РРГ).

### Назначение функции защитной блокировки

Детектор по теплопроводности крайне чувствителен к наличию непрерывного и стабильного потока газа-носителя через измерительные элементы.

{% hint style="danger" %}
Если электрический ток моста будет подан на вольфрамовые спирали детектора в условиях отсутствия или внезапного прекращения потока газа, чувствительные элементы мгновенно перегреются и необратимо выйдут из строя (перегорят) под воздействием кислорода воздуха.
{% endhint %}

Программный блок «Подключение ДТП» реализует **автоматическую систему перекрестного блокировочного контроля**. На основании заданных здесь настроек ядро программы «NetChrom 3.0» непрерывно опрашивает телеметрию именно тех линий РРГ, которые обеспечивают газовое питание измерительных ячеек. При падении расхода в этих линиях ниже критического предела система мгновенно и принудительно отключает ток питания моста ДТП, предотвращая аварию.

### Порядок конфигурации параметров подключения

Интерфейс модуля отображает список всех физически объявленных в системе блоков (например, **ДТП-1**, **ДТП-2**). Для каждого детектора предусмотрена пара интерактивных полей выпадающего типа:

* **Верхнее поле выбора (Рабочий канал):** Предназначено для программного назначения регулятора (например, `РРГ-2`), обеспечивающего непрерывную подачу газа-носителя в _измерительную ячейку_ детектора.
* **Нижнее поле выбора (Канал сравнения):** Предназначено для назначения регулятора, подающего газ в _сравнительную ячейку_ детектора (стабилизация нулевой линии).

### Особенности конфигурирования для различных газовых схем:

* **Раздельное питание ячеек:** Если измерительный канал и канал сравнения физически подключены к разным газовым линиям и управляются независимыми блоками пневматики, укажите в верхнем и нижнем полях соответствующие им разные номера регуляторов (например, верхнее поле — `РРГ-2`, нижнее поле — `РРГ-3`).
* **Совмещенное питание (Одноканальная схема):** Если, согласно заводской пневматической схеме хроматографа «Кристаллюкс-4000М», обе ячейки (измерительная и сравнительная) запитаны параллельно от одной общей входной газовой магистрали, **в обоих полях выбора необходимо жестко указать один и тот же номер регулятора** (например, выставить `РРГ-3` как для измерительного канала, так и для канала сравнения).
