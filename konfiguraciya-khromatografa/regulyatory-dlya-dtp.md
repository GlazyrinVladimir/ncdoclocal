# Регуляторы для ДТП

### Подключение ДТП

Для обеспечения безопасности и сохранности чувствительных элементов (спиралей) детектора по теплопроводности (ДТП) реализована система перекрестного контроля потоков.

<figure><img src="../.gitbook/assets/image (76).png" alt=""><figcaption></figcaption></figure>

**Назначение функции защиты**

ДТП крайне чувствителен к наличию постоянного потока газа-носителя. Если ток моста будет подан на спирали в отсутствие потока, они мгновенно перегреются и выйдут из строя. Программный блок «Подключение ДТП» заставляет систему контролировать наличие расхода именно на тех линиях, которые питают ячейки детектора.

**Описание параметров**

В окне отображается список установленных модулей (например, **ДТП-1**), для каждого из которых предусмотрено два поля выбора:

* **Верхнее поле выбора:** Предназначено для назначения РРГ, подающего газ в **измерительную ячейку** (рабочий канал).
* **Нижнее поле выбора:** Предназначено для назначения РРГ, подающего газ в **сравнительную ячейку** (канал сравнения).

{% hint style="info" %}
Если обе ячейки (измерительная и сравнительная) запитаны от одной газовой линии, в обоих полях следует выбрать один и тот же номер регулятора (например, РРГ-3, как показано на рисунке).
{% endhint %}
