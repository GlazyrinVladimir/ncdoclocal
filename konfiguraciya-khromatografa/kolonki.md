# Колонки

## Колонки

В данном разделе описываются физические характеристики используемых хроматографических колонок. Эти данные необходимы системе NetChrom для автоматического расчета давления на входе, линейной скорости газа-носителя и обеспечения стабильности времени удерживания.

Колонки хроматограф не может обнаружить сам и оператору необходимо **добавить их вручную**. Одновременно с колонками добавляются и связанные с ними регуляторы.

**Управление списком колонок**

* **Кнопка «Удалить колонку»**: Исключает текущую колонку из конфигурации прибора.
* **Кнопка «Добавить колонку»**: Позволяет сконфигурировать вторую (параллельную) колонку, если прибор оснащен соответствующим количеством испарителей и детекторов.

**Описание колонки**

* **Название**: Введите идентификатор колонки (например, _HP-5_ или _Solgel-1ms_). Данное имя будет использоваться при выборе колонки в методе анализа.
* **Тип колонки**: Согласно газовой схеме укажите тип - **Капиллярная** или **Насадочная**.
  * При выборе типа колонки автоматически задаются и регуляторы: регулятор газа носителя для насадочной колонки и регуляторы сброса и поддува для капиллярной.
  * Выбор типа определяет, какие математические формулы будут применяться для расчета газового режима.

**Схемы подключения газов (Настройка РРГ)**

При конфигурировании колонок система NetChrom автоматически переопределяет функции **регуляторов расхода газа (РРГ)**. Для корректной работы ЭУПГ необходимо строго соблюдать соответствие программных настроек физической схеме подключения газов.

{% hint style="info" %}
Перед установкой значений в данных полях убедитесь, что соответствующие регуляторы уже активированы во вкладке **«Регуляторы»**. Если номер нужного РРГ отсутствует в выпадающем списке, проверьте общую конфигурацию оборудования.
{% endhint %}

### **Для насадочных колонок**

Если выбрано значение **«Насадочная»**, система переходит в режим прямого управления потоком через слой сорбента:

* **Газ-носитель:** В выпадающем списке выберите номер РРГ, через который осуществляется подача газа-носителя непосредственно в колонку. В этом режиме регулятор поддерживает заданный объемный расход (мл/мин) в течение всего анализа.

### **Для капиллярных колонок**

Если в поле **«Тип колонки»** выбрано значение **«Капиллярная»**, регуляторы работают в режиме управления делением потока и обеспечения стабильности работы детектора:

* **Сброс пробы:** Выберите из выпадающего списка номер РРГ, который физически подключен к линии сброса испарителя (линия _Split_). Этот регулятор будет отвечать за поддержание коэффициента деления пробы.
* **Поддув:** Выберите номер РРГ, обеспечивающего подачу газа поддува в детектор (_Makeup gas_). Это необходимо для оптимизации объема детектора и улучшения формы пиков.

<figure><img src="../.gitbook/assets/image (84).png" alt=""><figcaption></figcaption></figure>

* **Производитель / Модель**: Справочные поля для заполнения паспортных данных колонки (не влияют на расчеты, но важны для ведения журнала).
  *   <mark style="color:$danger;">Для ускорения процесса заполнения параметров колонок вы можете воспользоваться их каталогом. Для этого нажмите на кнопку Каталог. На экране появится диалоговое окно Каталог колонок:</mark>

      В <mark style="color:$danger;">данном диалоговом окне выберите необходимую колонку из иерархического списка, отсортированного по производителю колонок, и нажмите на кнопку ОК. Каталог содержит далеко не все колонки, выпускаемые в данный момент различными компаниями. Если вы не обнаружили в списке необходимую Вам колонку, введите ее параметры вручную.</mark>
* **Геометрические и эксплуатационные параметры**
  * **Длина (м)**: Фактическая длина колонки.
  * **Диаметр (мм)**: Внутренний диаметр колонки.

{% hint style="info" %}
Эта информация позволит прибору рассчитать поток в колонке. Это дает огромные преимущества при использовании капиллярных колонок, поскольку делает возможным следующее:

* Непосредственный ввод соотношений деления, а также подсчет и установка прибором соответствующих скоростей потоков;
* Ввод скорости потока, давления на входе или средней линейной скорости. Прибор рассчитывает давление, необходимое для достижения скорости потока или линейной скорости, и устанавливает его.
{% endhint %}

{% hint style="warning" %}
Ошибка в указании диаметра (даже на 0.05 мм) приведет к некорректному расчету давления и невоспроизводимости времен удерживания.
{% endhint %}

* **Максимальная температура (°С)**: Укажите предельную рабочую температуру согласно паспорту колонки. ПО заблокирует нагрев термостата выше этого значения для предотвращения порчи неподвижной фазы.
