# Устройства

## Устройства

Данный раздел предназначен для активации и конфигурирования дополнительного навесного оборудования, расширяющего аналитические возможности хроматографа «Кристаллюкс-4000М».

#### **Автосамплеры (Дозаторы)**

В верхней части окна выберите тип используемого автоматического дозатора из выпадающего списка. Поддерживаются следующие модели:

* **Жидкостные дозаторы**: Серия **HT300A**, **HT3000A**, **HT3100A/AM**, **HT3200A**, а также отечественный **ДАЖ-4000**. Предназначены для автоматического ввода жидких проб.
* **Парофазные дозаторы**: Модели **HT2000H**, **HT280T HeadSpace**, **HT2800T HeadSpace**. Используются для анализа летучих компонентов в твердых и жидких матрицах.
* **Комбинированные системы**: Модели серии **HT280T/2800T Liquid**, позволяющие переключаться между вводом жидкости и паровой фазы.

{% hint style="info" %}
Если автоматический дозатор не используется, установите значение **«Нет»**.
{% endhint %}

#### **Дополнительные модули и узлы**

Активация устройств осуществляется путем установки флажка (чекбокса) напротив соответствующего наименования:

* **Системы ввода и подготовки**:
  * **ДАГ**: Дозатор автоматический газовый.
  * **Программируемый испаритель**: Позволяет изменять температуру порта ввода в процессе анализа.
  * **Термодесорбер**: Устройство для анализа проб, сконцентрированных на сорбентах.
  * **Дозатор равновесного пара / сжиженного газа**: Специализированные блоки для ввода сложных сред.
* **Управление потоками**:
  * **Переключающий кран / Кран обратной продувки**: Для реализации сложных схем разделения и защиты колонок от тяжелых компонентов.
  * **Газовый коммутатор**: Для автоматического выбора между несколькими линиями подачи пробы.
* **Реакторы и детекторы**:
  * **Метанатор**: Каталитический реактор для превращения CO и CO2 в метан (для анализа на ПИД).
  * **Сателлит**: Дополнительный измерительный блок или второй хроматограф в системе.
* **Вспомогательные узлы**:
  * **Второй термостат**: Если прибор оснащен независимым термостатом колонок.
  * **Кондиционер / Датчик уровня жидкости / Клапан Динса**: Узлы тонкой настройки и контроля состояния системы.

### Особенности интеграции устройств в рабочий процесс

Активация оборудования в окне конфигурации меняет структуру интерфейса в других разделах ПО:

* **Интеграция в «Метод»**: Параметры управления (температуры модулей, времена переключения кранов, работа метанатора) становятся доступны для редактирования в соответствующих вкладках **Метода**.
* <mark style="color:$danger;">**Интеграция в «Планировщик»**</mark><mark style="color:$danger;">: Управление</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">**Автосамплерами**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">вынесено в отдельный модуль планирования серийных анализов. Это позволяет задавать последовательность проб, количество промывок шприца и объемы ввода независимо от настроек температурного режима прибора.</mark>

### Работа с Сателлитом <mark style="color:$danger;">(4ый детектор добавить)</mark>

Функция **«Сателлит»** позволяет расширить количество каналов обработки данных. В ПО NetChrom реализовано два варианта использования данной функции:

**Внутренний сателлит**

Используется как **второй параллельный канал** обработки информации основного хроматографа.

* **Назначение**: Одновременная регистрация и обработка сигналов с двух детекторов.
* **Условие работы**: Применяется в случаях, когда параметры управления (температурные режимы, расходы газов) для обоих каналов полностью совпадают.
* **Особенности**: Для корректной работы системы необходимо создать и активировать отдельный **Метод для сателлита**, который будет работать синхронно с основным методом.

<mark style="color:$danger;">**Внешний сателлит**</mark>

<mark style="color:$danger;">Используется для подключения к системе</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">**стороннего неавтоматизированного оборудования**</mark><mark style="color:$danger;">.</mark>

* <mark style="color:$danger;">**Назначение**</mark><mark style="color:$danger;">: Оцифровка и обработка аналогового сигнала, поступающего с выхода детектора или усилителя другого хроматографа.</mark>
* <mark style="color:$danger;">**Принцип подключения**</mark><mark style="color:$danger;">: Выход стороннего прибора подключается к свободному каналу АЦП «Кристаллюкс-4000М».</mark>
* <mark style="color:$danger;">**Возможности**</mark><mark style="color:$danger;">: Система поддерживает до</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">**трех независимых каналов измерения**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">включительно. В этом режиме NetChrom выполняет роль многоканального регистратора данных (интегратора).</mark>
