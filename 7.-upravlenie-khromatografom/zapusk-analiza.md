# Запуск анализа

## Панель «Запуск анализа»

Панель **«Запуск анализа»** расположена в левой части окна управления хроматографом и предназначена для окончательной подготовки прибора к выполнению измерений. Функционал панели позволяет связать аппаратные ресурсы хроматографа с конкретным файлом аналитической методики и метаданными об исследуемом образце.

## Параметры запуска

Рабочая зона панели разделена на две идентичные зоны для независимого управления аналитическими каналами. Отображение зон зависит от текущей аппаратной конфигурации прибора:

* **Основной метод (Старт 1)** — активен всегда, используется для управления главным аналитическим каналом.
* **Метод сателлита (Старт 2)** — отображается в интерфейсе **исключительно при условии**, что двухканальный режим работы или наличие сателлитного модуля предварительно активированы в общей конфигурации газового хроматографа. Если сателлит не используется, данная зона автоматически скрывается.

В каждой активной зоне оператору доступны следующие элементы управления:

* **Выпадающий список выбора метода** — предназначен для выбора подготовленного файла аналитической методики (формат `*.metx`).
* **Управляющие кнопки**:
  * `Открыть метод` — быстрый переход к просмотру и редактированию параметров выбранной методики.
  * `Создать метод` — запуск файлового менеджера для создания нового файла методики с нулевыми базовыми установками.
  * `Передать` (`Передать (основной)`) — отправка физических параметров выбранного метода (температур зон, расходов газов-носителей, токов детекторов) в контроллер хроматографа для исполнения.

{% hint style="info" %}
До тех пор, пока параметры метода не «переданы» в прибор нажатием кнопки `Передать`, хроматограф продолжает функционировать на установках предыдущего метода или в режиме ожидания. Только после инициации передачи прибор начинает изменять температуры и расходы под новую аналитическую задачу. Рекомендуется создавать отдельные файлы методик для каждого типа анализа (например, «Анализ природного газа», «Градуировка спиртов»). Это исключает ошибки ручного ввода параметров перед пуском.
{% endhint %}

* **Назначение хроматограммы** — выпадающий список, определяющий статус выполняемого анализа для последующей автоматической сортировки и обработки данных в базе:
  * **Анализ** — выполнение стандартного рабочего измерения пробы.
  * **Градуировка** — выполнение анализа с целью построения или корректировки калибровочных зависимостей.
  * **Поверка** — контрольное измерение для проверки метрологических характеристик хроматографа.
  * **Холостой анализ** — проверка чистоты газовой системы, качества разделения и стабильности нулевой линии без ввода целевого компонента.

## Паспорт пробы

Блок **«Паспорт пробы»** содержит текстовые и цифровые метаданные, которые будут неразрывно связаны с файлом регистрируемой хроматограммы. Вкладки **«Основной»** и **«Сателлит»** в верхней части блока позволяют задавать раздельные описания для параллельных потоков данных при двухканальном анализе.

* **Проба/Анализ** — поле для ввода краткого наименования или шифра исследуемого образца.
* **Номер пробы / Номер анализа** — цифровые индексы для ведения сквозной нумерации в журнале лаборатории.
* **Дата отбора / Время отбора** — информационные поля. По умолчанию заполняются текущим системным временем ПК, при необходимости доступны для ручной корректировки оператором.
* **Оператор** — ФИО сотрудника, выполняющего измерение.
* **Место отбора пробы** — дополнительная текстовая информация о происхождении образца.
* **Комментарий** — поле для произвольных заметок оператора (например, указание веса навески, условий пробоподготовки или особенностей ввода).

## Порядок подготовки и запуска единичного анализа

Для выполнения измерений в режиме ручного ввода пробы (или при использовании автоматических дозаторов в режиме единичного пуска) оператор обязан строго соблюдать следующую последовательность действий:

**1. Инициализация рабочей области и выбор режима**

На верхней ленте панели управления нажмите на стрелку рядом с кнопкой `Выполнить анализ` для вызова выпадающего списка. Выберите пункт **«Одиночный анализ»**.&#x20;

**2. Выбор и передача методики обсчета**

1. В поле **«Основной метод (Старт 1)»** из выпадающего списка выберите требуемый рабочий метод. При необходимости нажмите кнопку `Открыть метод` для визуальной верификации параметров.
2. Если в системе активирована двухканальная схема, в поле **«Метод сателлита (Старт 2)»** укажите метод для ведомого канала. Если сателлит не используется, оставьте поле незаполненным.
3. Нажмите кнопку `Передать` для отправки настроек в контроллер хроматографа.

**3. Задание параметров регистрации данных и идентификация пробы**

1. Задайте объем вводимой пробы в поле **«Объем пробы (мкл)»**.
2. В поле **«Назначение хроматограммы»** выберите целевой статус анализа.
3. Заполните информационные поля блока **«Паспорт пробы»** (Наименование, Номер анализа, Оператор). Внесите дополнительные данные в поля места отбора и комментариев, если это требуется внутренней инструкцией.

**4. Выход хроматографа на рабочий режим**

Переведите внимание на правую панель во вкладку **«Состояние»**:

1. Убедитесь, что фактические значения температур зон и расходов газов в таблицах сравнения полностью совпадают с заданными уставками метода.
2. Проверьте нижнюю панель **«Параметры вне нормы»**. В ней должно отображаться сообщение: **«Все параметры в норме»**.

**5. Контроль стабильности нулевой линии**

Нажмите кнопку `Видеосамописец` на верхней ленте. Оцените графический сигнал детектора: базовая линия должна быть стабильной, без резких скачков (рывков), дрейфа и выраженного высокочастотного шума.

**6. Активация записи и ввод пробы**

1. Дождитесь выхода аналитического комплекса на рабочий режим. Признаками полной готовности системы к старту измерения являются:
   * Переход текущего технологического этапа в статус **«Ввод пробы»** (отображается в строке состояния ПО «NetChrom»);
   * Непрерывное свечение физического светодиодного индикатора **«Готов»** на передней панели корпуса хроматографа «Кристаллюкс-4000М».
2. Осуществите ввод исследуемого образца в газовый тракт прибора вручную микрошприцом, либо в соответствии с типом используемого периферийного оборудования.
