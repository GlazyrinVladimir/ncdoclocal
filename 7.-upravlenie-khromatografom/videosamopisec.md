# !Видеосамописец

### Видеосамописец

Диалоговое окно **«Видеосамописец»** предназначено для графического отображения аналитических сигналов детекторов в режиме реального времени. Непрерывный мониторинг динамики сигнала необходим на всех ключевых этапах работы с газовым хроматографом:

* **Этап подготовки** — для контроля выхода детекторов на стабильный рабочий режим;
* **Выход на режим** — для оперативной оценки стабильности нулевой линии и текущего уровня высокочастотных шумов;
* **Выполнение анализа** — для визуального мониторинга процесса разделения компонентов и регистрации хроматограммы.

#### Основные элементы управления

**1. Панель выбора каналов**

Расположена в верхней части диалогового окна и содержит переключатели активных измерических каналов (например, **ПИД-1**, **ПФД-1**, **ДТП-1**). Элементы панели обеспечивают:

* **Тумблер (Вкл/Откл)** — позволяет отображать или скрывать график сигнала конкретного детектора в общем рабочем поле.
* **Цветовая индикация** — каждый графический трек имеет индивидуальный цвет, строго соответствующий маркеру рядом с наименованием детектора.

**2. Поле графиков**

Предназначено для визуализации зависимости амплитуды аналитического сигнала от времени:

* **Ось Y (Вертикальная)** — амплитуда сигнала, выраженная в милливольтах (мВ). Масштаб оси может быть изменен оператором для детального изучения шумов нулевой линии или корректного просмотра высокоинтенсивных пиков.
* **Ось X (Горизонтальная)** — текущее время регистрации в формате `чч:мм:сс`.
* **Сетка рабочего поля** — вспомогательный элемент, помогающий оператору визуально оценивать дрейф базовой линии и наличие периодических помех.

**3. Функциональные возможности интерактивного анализа**

* **Масштабирование** — выделение определенной области графического поля с помощью мыши для детального осмотра структуры шумов или формы пиков.
* **Панорамирование (Смещение)** — перемещение рабочей области вдоль временной оси для просмотра ранее зарегистрированных фрагментов сигнала до момента инициализации официальной регистрации хроматограммы.
* **Автоматическая подстройка** — непрерывный автоматический выбор оптимального масштаба по оси Y, гарантирующий постоянное нахождение текущего сигнала в зоне видимости оператора.

#### Элементы управления отображением

Данная группа кнопок предназначена для оперативной настройки геометрии и области видимости графиков без изменения физических параметров текущего аналитического метода.

* **Кнопка «Диапазон»** — позволяет вручную установить фиксированные границы отображения по осям времени и амплитуды. Используется в случаях, когда необходимо жестко закрепить «окно» просмотра в заданных пределах (например, в диапазоне от 10 до 20 мВ).
* **Меню «Показать»** — инструментарий для быстрого позиционирования графиков в рабочей области. При нажатии на стрелку открывается выпадающий список команд автоматического масштабирования:
  * _Показать все_ — устанавливает масштаб, при котором все зарегистрированные от начала записи данные по всем детекторам полностью умещаются в окне;
  * _Начало / Конец по времени_ — мгновенный переход к самому первому или к самому последнему зарегистрированному временному отсчету;
  * _Верх / Низ по высоте_ — смещение текущей области видимости к максимальному или минимальному значению амплитуды сигнала соответственно;
  * _Все по времени_ — растягивает график на всю доступную ширину окна по горизонтальной оси;
  * _Все по высоте_ — выполняет вертикальный «автомасштаб», подбирая высоту отображения таким образом, чтобы пиковый сигнал занимал 80–90% высоты рабочего окна.
* **Кнопка «Очистить»** — выполняет полное удаление накопленных графических данных с экрана видеосамописца.

{% hint style="info" %}
**Примечание**\
Команда «Очистить» удаляет исключительно визуальную историю в текущем окне мониторинга. Она не влияет на запись текущего анализа и не приводит к потере данных из файла хроматограммы, если в данный момент запущен процесс официальной регистрации. Кнопка используется для «обнуления» экрана перед выходом прибора на новый рабочий режим.
{% endhint %}

#### Управление масштабом и навигация

Для детального изучения хроматограммы в окне Видеосамописца предусмотрены инструменты интерактивного масштабирования. По умолчанию система автоматически устанавливает масштаб, при котором в окне виден весь график от начала записи.

**Управление с помощью мыши**

* **Произвольное увеличение (зум)** — для увеличения интересующего участка нажмите левую кнопку мыши в одном из углов целевой области и, не отпуская её, растяните прямоугольник выделения в противоположный угол. После отпускания кнопки мыши выбранная область развернется на все рабочее поле.
* **Быстрый возврат (Сброс масштаба)** — для возврата к исходному отображению всего графика выполните двойной щелчок левой кнопкой мыши в любом месте графического поля (вне границ зарегистрированных пиков).

**Использование клавиатуры (Курсорные кнопки)**

Инструменты мыши удобно сочетать с клавиатурой для быстрого кратного изменения масштаба. Ориентируйтесь на стрелки управления курсором:

* `↑` (Вверх) — растянуть график в два раза по вертикали (увеличение амплитудного разрешения сигнала);
* `↓` (Вниз) — сжать график в два раза по вертикали;
* `→` (Вправо) — растянуть график в два раза по горизонтали (увеличение временного разрешения);
* `←` (Влево) — сжать график в два раза по горизонтали.

#### Диагностика состояния ГХ по сигналу видеосамописца

Визуальный анализ «живого» сигнала позволяет оператору превентивно выявить ряд аппаратных неисправностей хроматографа или ошибок в подготовке газовой системы:

| Вид аналитического сигнала                             | Вероятная причина отклонения параметров                                                                                                                |
| ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Хаотичные высокочастотные всплески («пики-иголки»)     | Наличие электрических наводок в питающей сети или критическое загрязнение электродов детектора.                                                        |
| Резкий одномоментный уход базовой линии вверх или вниз | Скачкообразное изменение расхода газа-носителя или резкий температурный сдвиг в термостатируемой зоне.                                                 |
| Широкие волны (циклический дрейф нулевой линии)        | Нестабильность давления на входе в прибор (неисправность регулятора) или выраженные колебания температуры воздуха в лаборатории (работа кондиционера). |
| Выраженная ступенька сигнала (вертикальный сдвиг)      | Момент розжига пламени (для детекторов ПИД/ПФД) или механическое переключение кранов-дозаторов.                                                        |

{% hint style="info" %}
**Важная рекомендация**\
Перед запуском любого типа анализа в обязательном порядке убедитесь по Видеосамописцу, что нулевая линия полностью стабильна и видимый дрейф отсутствует. Выполнение этого правила гарантирует правильный расчет площадей хроматографических пиков при последующей автоматической обработке результатов.
{% endhint %}

#### Алгоритмы действий оператора при обнаружении нестабильного сигнала

Если в окне **«Видеосамописец»** зафиксировано аномальное поведение базовой линии, оператор обязан приостановить подготовку к анализу и выполнить следующие компенсирующие действия:

**Действия при появлении хаотичных всплесков («пиков-иголок»):**

1. Проверьте надежность подключения контура защитного заземления к корпусу хроматографа и персонального компьютера.
2. Убедитесь, что в одну силовую розетку (или сетевой фильтр) с хроматографом не подключено мощное силовое оборудование (компрессоры, центрифуги, климатические системы).
3. Если всплески продолжаются, выполните процедуру кондиционирования детектора (прокалка) в соответствии с Инструкцией по эксплуатации на конкретный тип детектора.

**Действия при резком уходе базовой линии вверх или вниз:**

1. Немедленно проверьте показания манометров/расходомеров на блоке управления газами. Убедитесь в отсутствии критического падения давления на входе в прибор (на входе в регуляторы давления).
2. Проверьте герметичность газовой магистрали, обратив особое внимание на затяжку гаек крепления колонки и состояние резиновой прокладки (септы) испарителя. При необходимости замените септу.
3. Проверьте по вкладке **«Состояние»**, не происходит ли в данный момент запланированный методом выход термостата колонок на новую температурную уставку.

**Действия при циклическом дрейфе («широких волнах»):**

1. Проверьте стабильность давления на выходных редукторах газовых баллонов или на выходах генераторов газов (водорода, чистого воздуха). Признаком неисправности является колебание стрелки редуктора.
2. Проверьте условия окружающей среды в помещении лаборатории. Направленные потоки холодного воздуха от кондиционера или радиаторов отопления на корпус прибора должны быть исключены.
3. Убедитесь, что время стабилизации прибора после включения составило не менее 1–1,5 часов для выхода на тепловой баланс.

**Действия при обнаружении «ступеньки» сигнала:**

1. Если используется детектор ПИД/ПФД, убедитесь по току поджига или визуально, что пламя стабильно горит. Ступенька вверх — штатный признак успешного розжига.
2. Если ступенька появилась спонтанно в режиме ожидания, проверьте журнал событий ПО **«NetChrom»** на предмет несанкционированного или ошибочного переключения электромагнитных клапанов или кранов-дозаторов.

#### Практические рекомендации по анализу сложных пиков

* **Просмотр малых пиков** — если целевой компонент присутствует в пробе в малой концентрации, используйте клавишу `↑` несколько раз до тех пор, пока шум нулевой линии не станет отчетливо виден. Это позволит оператору визуально оценить критическое соотношение «сигнал/шум».
* **Оценка разделения (разрешения)** — в случаях, когда два соседних пика выходят из колонки близко друг к другу, воспользуйтесь выделением интересующей области мышью или комбинацией клавиш для детального осмотра «долины» между ними и определения степени их разделения.









## Видеосамописец

* **Этап подготовки:** для контроля выхода детектора на рабочий режим.
* **Выход на режим:** для оценки стабильности нулевой линии и уровня шумов.
* **Выполнение анализа:** для визуального мониторинга процесса разделения компонентов.

#### Рекомендации по анализу пиков

* **Просмотр малых пиков:** Если целевой компонент имеет малую концентрацию, используйте клавишу **↑** несколько раз, пока шум нулевой линии не станет отчетливо виден. Это позволит визуально оценить соотношение «сигнал/шум».
* **Оценка разделения:** Если два пика выходят слишком близко друг к другу, воспользуйтесь выделением мышью или клавишей **→** для детального осмотра «долины» между ними.
* **Панорамирование:** При сильном увеличении используйте полосы прокрутки для перемещения вдоль хроматограммы без изменения масштаба.
