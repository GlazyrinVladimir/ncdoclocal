# !Настройки NetChrom

## Первоначальная настройка

После установки и первого запуска NetChrom необходимо выполнить начальную настройку программного обеспечения. <mark style="color:$primary;">Данная операция проводится однократно для синхронизации параметров программы с конфигурацией конкретного хроматографа (или группы хроматографов).</mark>

{% hint style="info" %}
Примечание. Все параметры, введенные при первоначальной настройке, могут быть изменены пользователем в любой момент времени в процессе дальнейшей эксплуатации системы.
{% endhint %}

Для перехода к настройкам нажмите кнопку **«Настройки»** (символ ⚙️), расположенную в правой верхней части главного окна программы.

## Настройки

Окно **«Настройки»** предназначено для конфигурации программной среды NetChrom, управления параметрами автоматизации анализа и контроля ресурсов системы хранения данных.

Интерфейс окна настроек состоит из двух основных областей:

1. **Левая панель:** иерархический список разделов конфигурации.
2. **Правая панель:** активная вкладка, содержащая параметры и элементы управления выбранного пункта списка.

При вызове диалогового окна конфигурации верхнего уровня первым на экран выводится раздел **«Главная»**. Эта вкладка является базовой и используется для верификации лицензии, управления системной логикой приложения и контроля дисковой памяти.

Данный раздел содержит базовые сведения о лицензии и системные параметры работы.

### Информационный блок лицензирования

В верхней части окна приведена системная информация о текущей версии ПО и параметрах лицензирования (Ключ лицензии, Организация-лицензиат). Данные сведения необходимы при обращении в службу технической поддержки.

* Строка **«NetChrom версия...»** — указывает текущий номер программной сборки и дату её релиза.
* Строка **«Ключ лицензии»** — отображает уникальный цифровой идентификатор установленной копии продукта.
* Строка **«Лицензирована для...»** — содержит официальное наименование предприятия-заказчика и его региональное расположение.
* Поле нормативной документации — информационная область, отображающая перечень поддерживаемых ГОСТов и методик измерений.

### Рекомендуемые параметры

Группа содержит настройки, определяющие логику выполнения анализа и способы математической обработки результатов. Использование данных параметров позволяет стандартизировать условия проведения измерений.

* **Автостарт метода после запуска**\
  Обеспечивает автоматическую загрузку и запуск последнего метода анализа, использованного в диалоговом окне «Запуск метода». Запуск происходит при старте программы или при повторном входе в неё после выхода.

{% hint style="info" %}
**Важное условие:** для корректной работы функции хроматограф должен быть включен.
{% endhint %}

* **Округлять температуры до сотых**\
  Специализированный параметр для хроматографов серии **К-4000М**. При активации точность отображения температур различных зон прибора в «Окне состояния» повышается до сотых долей градуса (0.01 °C).
* **Выравнивать результаты до 100%**\
  Режим автоматического математического выравнивания (нормировки) результатов анализа. При включении система приводит сумму всех определенных компонентов к 100%.
* **Контроль t° колонки и детектора**\
  Управляет системной проверкой соотношения температурных зон.
  * **Назначение:** Позволяет отключить стандартное условие безопасности.
  * **Применение:** Опция используется в случаях, когда методикой предусмотрен нагрев колонок до температур, превышающих температуру детектора.

### Хранилище

Обеспечивает мониторинг дискового пространства и настройку политик архивирования данных.

* **Индикатор заполнения** — отображает текущий объем базы данных (ГБ) и процент использования доступного пространства.
* **Автоматическая очистка** — позволяет установить регламентированный срок хранения хроматограмм и записей системного журнала. По истечении заданного интервала система производит безвозвратное удаление устаревших данных.
* **Очистка хранилища** — функция принудительного удаления временных файлов и оптимизации структуры базы данных в ручном режиме.

#### Управление конфигурацией

* **\[Применить]** — сохранение текущей конфигурации и вступление настроек в силу без перезагрузки приложения.
* **\[По умолчанию]** — восстановление "заводских" (базовых) установок параметров для текущего раздела.
