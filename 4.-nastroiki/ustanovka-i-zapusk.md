# !Установка и запуск

## Установка и иницализация ПО "NetChrom 3.0"

Настоящий раздел содержит пошаговый алгоритм развертывания программного комплекса «NetChrom 3.0» на управляющей рабочей станции под управлением ОС Windows 10/11, включая процедуру проверки безопасности и автоматическую установку сопутствующих драйверов.

### Верификация цифровых сертификатов безопасности

Современные требования ИТ-безопасности промышленных предприятий требуют обязательной проверки подлинности программного обеспечения перед его запуском в операционной среде.

1. Перенесите дистрибутив программы (инсталляционный пакет) на целевой ПК.
2. В случае распространения ПО через систему авторизованных электронных ключей, перед запуском убедитесь, что в ОС Windows импортированы и активированы актуальные **цифровые сертификаты безопасности** компании-разработчика.
3. При первом запуске инсталлятора системная служба контроля учетных записей (UAC) Windows выдаст запрос на подтверждение операции. Убедитесь, что в поле «Проверенный издатель» указана легитимная подпись, и нажмите кнопку **\[Да]**.

### Стандартный процесс установки

Процесс развертывания полностью автоматизирован и выполняется с помощью мастера установки.

1. Запустите исполняемый файл установщика (`NetChrom30_Setup.exe`) **от имени Администратора**.
2. В открывшемся диалоговом окне приветствия нажмите кнопку **\[Далее]**.
3. **Лицензионное соглашение:** Ознакомьтесь с условиями использования ПО, установите флаг в положение _«Я принимаю условия соглашения»_ и нажмите **\[Далее]**.
4. **Выбор директории:** По умолчанию программа предлагает установку в системный каталог `C:\Program Files\NetChrom30`.

**Важная рекомендация:** Если на предприятии действуют жесткие доменные политики ограничения прав пользователей (лаборантов), рекомендуется изменить путь установки на коренной каталог диска, например: `C:\NetChrom30\`. Это исключит проблемы с правами доступа при записи временных файлов базы данных.

5. Нажмите кнопку **\[Установить]** для запуска процесса копирования исполняемых модулей и создания структуры папок.

### Автоматическое развертывание сопутствующего ПО и драйверов

Программный комплекс «NetChrom 3.0» для своей корректной работы использует внешние системные компоненты и средства аппаратной защиты. На финальной стадии установки инсталлятор автоматически инициализирует развертывание сопутствующих модулей:

**1. Среда выполнения Microsoft .NET Framework**

Программа проверяет наличие в ОС библиотек **.NET Framework** (версий 3.5 и 4.8). В случае их отсутствия или повреждения, инсталлятор в фоновом режиме запустит процесс их конфигурации и интеграции в систему.

**2. Драйверы электронных ключей защиты Guardant**

Для предотвращения несанкционированного копирования и верификации лицензии ПО использует USB-ключи Guardant.

* В процессе установки инсталлятор автоматически развернет актуальную версию системной службы и драйверов **Guardant Driver**;
* Во время этого процесса на экране может на несколько секунд появиться консольное окно инициализации драйвера защиты — не закрывайте его вручную;
* По завершении процесса установщик выдаст системное сообщение об успешной интеграции драйверов.

{% hint style="warning" %}
Категорически запрещается подключать физический USB-ключ Guardant к порту компьютера ДО полного завершения работы инсталлятора. Подключайте ключ только после финальной перезагрузки системы.
{% endhint %}

### Завершение установки

После наката всех компонентов мастер установки выведет финальное диалоговое окно.

1. Установите флаг в поле _«Выполнить перезагрузку компьютера сейчас»_ (критически важно для корректного запуска службы Guardant и применения сетевых политик).
2. Нажмите кнопку **\[Готово]**.

После перезагрузки ПК на Рабочем столе появится ярлык **«NetChrom 3.0»** для запуска программного комплекса.

## Порядок запуска ПО и инициализации связи с прибором

В данном разделе приведены пошаговые инструкции по запуску программного обеспечения **«NetChrom 3.0»** в операционной среде Windows 10/11, а также алгоритм установления сетевого и логического соединения между управляющей рабочей станцией (ПК) и газовым хроматографом «Кристаллюкс-4000М».

### Способы инициализации приложения в среде Windows

Запуск программного комплекса «NetChrom 3.0» осуществляется с использованием стандартных интерфейсов операционной системы. Предусмотрены следующие варианты инициализации:

* **Запуск с Рабочего стола:** Выполните двойной щелчок левой кнопкой мыши по ярлыку приложения **«NetChrom 3.0»**.
* **Запуск из главного меню «Пуск»:** Откройте меню «Пуск», перейдите в программную группу «NetChrom» и выберите соответствующий исполняемый модуль, либо воспользуйтесь системной строкой поиска Windows, введя наименование программы.
* **Быстрый запуск с Панели задач:** При необходимости оперативного доступа запустите программу любым из вышеперечисленных способов, после чего закрепите иконку приложения на панели задач (вызов контекстного меню нажатием правой кнопки мыши по значку активной программы -> команда _«Закрепить на панели задач»_).
* **Автоматический запуск (Автозагрузка):** При использовании хроматографического комплекса в режиме непрерывного потокового мониторинга рекомендуется настроить автоматический запуск ПО. Для этого добавьте ярлык исполняемого файла `NetChrom.exe` в системный каталог автозагрузки Windows (программа будет инициализироваться автоматически при авторизации пользователя в операционной системе).

{% hint style="info" %}
Для корректной инициализации всех фоновых математических потоков и модулей опроса АЦП прибора рекомендуется производить запуск ПО с правами Администратора или от имени пользователя, входящего в локальную группу с расширенными правами доступа.
{% endhint %}

**Основные способы добавления приложения в автозапуск:**

1. **Папка автозагрузки (надежный метод):**
   * Нажмите клавиши `Win + R` на клавиатуре.
   * Введите `shell:startup` и нажмите Enter. Откроется папка автозагрузки.
   * Скопируйте ярлык нужной программы (с рабочего стола или из папки установки) и вставьте его в эту папку.
2. **Через параметры системы:**
   * Откройте **Параметры** (значок шестеренки в меню «Пуск» или `Win+I`).
   * Перейдите в раздел **Приложения** -> **Автозагрузка**.
   * Найдите программу в списке и переведите переключатель в положение **Вкл**.
3.  **Диспетчер задач:**

    * Нажмите `Ctrl+Shift+Esc`, чтобы открыть Диспетчер задач.
    * Перейдите на вкладку **Автозагрузка**.
    * Нажмите правой кнопкой мыши на приложении и выберите «Включить».

    **Как убрать программу из автозагрузки:**\
    Зайдите в **Диспетчер задач** -> **Автозагрузка**, нажмите правой кнопкой мыши на программу и выберите «Отключить».

## Конфигурирование физического интерфейса связи

Взаимодействие между управляющей рабочей станцией и газовым хроматографом «Кристаллюкс-4000М» осуществляется через последовательный интерфейс передачи данных.

#### Интерфейс RS-232 (Последовательный COM-порт)

Для настройки локального обмена данными выполните следующие действия:

1. **Физическое подключение:** Подключите штатный интерфейсный кабель из комплекта поставки прибора к свободному физическому COM-порту персонального компьютера (или к виртуальному COM-порту при использовании сертифицированного преобразователя интерфейсов USB-to-RS232).
2. **Выбор режима:** В главном диалоговом окне конфигурации оборудования ПО «NetChrom 3.0» переведите переключатель режима связи в положение **«COM»**.
3. **Идентификация порта:** В выпадающем списке параметров укажите точный порядковый номер используемого COM-порта (например, `COM1` или `COM2`), соответствующий подключенному кабелю хроматографа в Диспетчере устройств Windows.

{% hint style="warning" %}
Подключение и отключение интерфейсного кабеля RS-232 необходимо производить строго при выключенном питании хроматографа и персонального компьютера во избежание выхода из строя приемопередатчиков портов.
{% endhint %}
