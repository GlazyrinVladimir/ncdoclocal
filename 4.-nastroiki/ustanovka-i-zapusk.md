## Порядок запуска ПО и инициализации связи с прибором

В данном разделе приведены пошаговые инструкции по запуску программного обеспечения **«NetChrom 3.0»** в операционной среде Windows 10/11, а также алгоритм установления сетевого и логического соединения между управляющей рабочей станцией (ПК) и газовым хроматографом «Кристаллюкс-4000М».

### **Способы запуска в среде Windows 10**

Для запуска NetChrom используйте один из стандартных интерфейсов операционной системы:

* **Классический запуск:** Выполните двойной щелчок левой кнопкой мыши по ярлыку «NetChrom» на Рабочем столе.  <img src="../.gitbook/assets/Безымянный (1).png" alt="" data-size="line">
* **Меню «Пуск»:** Нажмите клавишу `Win`, введите в строке поиска «NetChrom» или найдите соответствующий пункт в программной группе «NetChrom»..
* **Панель задач:** Для оперативного доступа рекомендуется закрепить иконку приложения на панели задач (ПКМ по значку запущенной программы -> «Закрепить на панели задач»).
* **Автозапуск:** Если комплекс используется для непрерывного мониторинга, добавьте приложение NetChrom  в автозагрузку Windows 10, чтобы программа запускалась автоматически при входе пользователя в систему.

**Основные способы добавления приложения в автозапуск:**

1. **Папка автозагрузки (надежный метод):**
   * Нажмите клавиши `Win + R` на клавиатуре.
   * Введите `shell:startup` и нажмите Enter. Откроется папка автозагрузки.
   * Скопируйте ярлык нужной программы (с рабочего стола или из папки установки) и вставьте его в эту папку.
2. **Через параметры системы:**
   * Откройте **Параметры** (значок шестеренки в меню «Пуск» или `Win+I`).
   * Перейдите в раздел **Приложения** -> **Автозагрузка**.
   * Найдите программу в списке и переведите переключатель в положение **Вкл**.
3.  **Диспетчер задач:**

    * Нажмите `Ctrl+Shift+Esc`, чтобы открыть Диспетчер задач.
    * Перейдите на вкладку **Автозагрузка**.
    * Нажмите правой кнопкой мыши на приложении и выберите «Включить».

    **Как убрать программу из автозагрузки:**\
    Зайдите в **Диспетчер задач** -> **Автозагрузка**, нажмите правой кнопкой мыши на программу и выберите «Отключить».

## Конфигурирование физического интерфейса связи

Взаимодействие между управляющей рабочей станцией и газовым хроматографом «Кристаллюкс-4000М» осуществляется через последовательный интерфейс передачи данных.

#### Интерфейс RS-232 (Последовательный COM-порт)

Для настройки локального обмена данными выполните следующие действия:

1. **Физическое подключение:** Подключите штатный интерфейсный кабель из комплекта поставки прибора к свободному физическому COM-порту персонального компьютера (или к виртуальному COM-порту при использовании сертифицированного преобразователя интерфейсов USB-to-RS232).
2. **Выбор режима:** В главном диалоговом окне конфигурации оборудования ПО «NetChrom 3.0» переведите переключатель режима связи в положение **«COM»**.
3. **Идентификация порта:** В выпадающем списке параметров укажите точный порядковый номер используемого COM-порта (например, `COM1` или `COM2`), соответствующий подключенному кабелю хроматографа в Диспетчере устройств Windows.

{% hint style="warning" %}
Подключение и отключение интерфейсного кабеля RS-232 необходимо производить строго при выключенном питании хроматографа и персонального компьютера во избежание выхода из строя приемопередатчиков портов.
{% endhint %}
