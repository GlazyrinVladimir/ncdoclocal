# Настройка отображения хроматограмм

Настоящий раздел руководства предназначен для конфигурирования параметров графического отображения аналитических сигналов (хроматограмм) в окне оперативного мониторинга и при формировании отчетной документации. Раздел также содержит описание инструментов интерактивного взаимодействия оператора с графическим полем.

Перед началом регистрации данных и выполнения анализов необходимо выполнить базовую настройку графической панели.

## Настройки хроматограммы

Переход к конфигурированию параметров отображения осуществляется через главное меню программы.&#x20;

### Доступные параметры конфигурации

Внутри вкладки **«Хроматограмма»** управление параметрами разделено на три независимых блока:

1. **Параметры вывода и взаимодействия** — задание критериев автоматического масштабирования осей времени удерживания и амплитуды сигнала, настройка чувствительности мыши при интерактивном выделении (кадрировании) областей графика, установка шага сетки.
2. **Линии и надписи** — конфигурация отображения подписей идентифицированных компонентов (название, время удерживания), разметки характерных точек пиков (старт, вершина, конец), а также толщины и сглаживания линии базового сигнала.
3. **Цвета** — выбор контрастной цветовой схемы для элементов графика (линия хроматограммы, базовая линия, подложка, координатная сетка).

При подготовке хроматограмм к выводу на монохромную печать рекомендуется использовать предустановленную контрастную черно-белую цветовую схему в блоке «Цвета».

### Параметры вывода и взаимодействия

Данная группа настроек определяет внешний вид графического поля и логику подсветки объектов при анализе данных.

* **Тип сетки** (выпадающий список)\
  Позволяет выбрать стиль отрисовки координатной сетки на поле хроматограммы.&#x20;
* **Толщина вертикальной линии** (выпадающий список)\
  Задает толщину линий сетки или маркеров времени. Настройка «Тонкая» рекомендуется для обеспечения лучшей читаемости графиков с высокой плотностью пиков.
* **Выделять активный сигнал** (переключатель)\
  При включении система визуально акцентирует (изменяет толщину) линию текущего активного канала, что упрощает работу в многоканальных режимах.
* **Заливка выбранного пика** (переключатель)\
  Активирует цветовое заполнение площади под кривой для пика, выбранного пользователем в таблице результатов или на графике. Помогает визуально оценить границы интегрирования.
* **Заливка выбранной группы пиков** (переключатель)\
  Аналогично предыдущему пункту, включает заливку для всей группы компонентов, объединенных по какому-либо признаку (например, при расчете суммы фракций).
* **Заливка специальных пиков** (переключатель)\
  Включает индикацию пиков, имеющих специальный статус в методе (например, контрольные или калибровочные компоненты).

### Линии и надписи

Данная группа параметров позволяет настроить отображение расчетных линий и текстовых меток над пиками хроматограммы. Большинство настроек управляются переключателями типа «Вкл/Откл».

#### Графические элементы (Линии)

Параметры этой группы позволяют визуально контролировать процесс обработки сигнала.

* **Вывод всех детекторов** — параметр, управляющий одновременным отображением графиков со всех активных каналов системы в одном рабочем поле.
* **Базовая линия** — отображение расчетной нулевой линии. Позволяет оценить качество разделения пиков и стабильность работы детектора.
* **Маркеры пика** — вертикальные отметки в точках начала и окончания интегрирования. Необходимы для визуальной проверки корректности определения границ пика.

#### Количественные и идентификационные данные

Эти параметры определяют, какие данные из таблицы результатов будут продублированы текстом над вершиной каждого пика.

* **Идентификация:**
  * **Имя компонента** — название вещества (согласно таблице идентификации).
  * **Имя группы** — название группы, если компонент в неё включен.
  * **Номер пика** — порядковый номер в списке найденных объектов.
* **Метрологические параметры:**
  * **Время удерживания** — время выхода максимума пика (в минутах).
  * **Площадь / Высота** — количественные характеристики сигнала.
  * **Концентрация / Относительный объем** — расчетные значения содержания вещества в пробе.

#### Окна идентификации и зоны контроля

Параметры для визуализации «коридоров» допуска, заданных в методе анализа.

* **Окно идентификации / неидентификации** — графическое выделение временных интервалов, в которых программа производит поиск пиков.
* **Окно реперных компонентов** — выделение зон выхода опорных веществ, по которым калибруется шкала времени.
* **Границы группы компонентов** — маркеры, объединяющие серию пиков в единый расчетный блок.

#### Служебная информация и диагностика

Инструменты для оперативного контроля процесса анализа.

* **Ошибки идентификации** — вывод предупреждений прямо на хроматограмме при нарушении условий идентификации (например, выход за пределы временного окна).
* **Специальные компоненты** — включение отображения меток для пиков со специфическим статусом (контрольные пробы, внутренние стандарты).

### Цвета

Подраздел предназначен для индивидуальной настройки цветовой схемы графического окна. Цветовое кодирование облегчает визуальный анализ многоканальных систем и контроль границ интегрирования.

**Сигналы детекторов**

В верхней части списка располагаются настройки для каждого измерительного канала системы (от 1 до 8)<mark style="color:$danger;">.</mark>

* **Сигнал детектора (1–8)** — индивидуальный цвет линии хроматограммы для каждого физического или расчетного канала. Рекомендуется выбирать контрастные цвета для удобства сравнения графиков при их наложении.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (106).png" alt=""><figcaption><p>Выбор цвета сигнала детектора</p></figcaption></figure></div>

**Линии и маркеры обработки**

Цвета вспомогательных графических элементов, отвечающих за обсчет пиков.

* **Базовая линия** — цвет расчетной нулевой линии.
* **Маркеры пиков** — цвет вертикальных линий, обозначающих границы интегрирования (начало и конец) для всех найденных пиков.
* **Маркеры выбранного пика** — определяет цвет отображения трех ключевых точек активного пика: **начала**, **конца** и **вершины**. Эти точки являются интерактивными узлами управления и используются оператором для визуального контроля и выполнения операций ручного редактирования (корректировки границ интегрирования или положения вершины) непосредственно на графике.
* **Маркеры групп** — цветовое обозначение границ для серий пиков, объединенных в одну расчетную группу.

**Информационные зоны и надписи**

Настройка цветовой индикации для инструментов идентификации и диагностики.

* **Окно идентификации / неидентификации** — цвета заливки или рамок для временных интервалов, в которых происходит поиск веществ.
* **Ошибки идентификации** — цвет предупреждающих заливки или рамок для временных интервалов  в случае нарушения логики распознавания пиков.
* **Подписи компонентов** (неидентифицированного, специального) — настройка цвета шрифта для текстовых меток над пиками соответствующих типов.

**Заливка областей**

Параметры цветового заполнения площадей под кривой хроматограммы.

* **Заливка выбранного пика** — цвет заполнения площади для текущего активного компонента.
* **Заливка специального пика** — выделение цветом площади контрольных или калибровочных компонентов.
* **Заливка выбранной группы** — цвет заполнения площади для всех пиков, входящих в выделенную группу.
