# Конфигурация хроматографа

## Конфигурация хроматографа

Установка и первичное конфигурирование параметров хроматографа выполняется однократно после добавления устройства в сетевой контур программы.

#### Регламент проведения настройки

Конфигурирование системы выполняется в следующих случаях:

* Первичный запуск аналитического комплекса после инсталляции ПО;
* Обновление версий программного обеспечения управляющей рабочей станции;
* Аппаратная модернизация хроматографа «Кристаллюкс-4000М»;
* Замена стационарного аналитического модуля детекторов или ремонт электронных плат ГХ.

Для перехода к панели глобальных параметров выберите элемент **«Конфигурация хроматографа»** на верхней контекстной ленте управления прибором.

### Параметры связи, идентификации и автоматизации

Интерфейсная форма корневого подраздела «Хроматограф» предназначена для установления стабильного низкоуровневого обмена данными и автоматического распознавания архитектуры прибора.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (169).png" alt=""><figcaption></figcaption></figure></div>

#### Параметры связи и идентификации

* **Выбор COM-порта:** Из выпадающего списка укажите порядковый номер физического или виртуального (при подключении через адаптер-преобразователь USB-RS232) последовательного порта, к которому подключен интерфейсный кабель ГХ.
* **Название хроматографа:** Ввод произвольного буквенно-цифрового идентификатора прибора для отображения в главном рабочем пространстве, экранных формах мониторинга и отчетных протоколах (например, `ГХ №1_Лаб2`).
* **Тип хроматографа:** Системное поле, заблокированное для редактирования. По умолчанию установлено значение **«Кристаллюкс-4000М»**, определяющее базовый набор доступных программных модулей, библиотек и математических алгоритмов.

#### Сбор данных и автоматизация

* **Модуль детекторов:** Выбор аппаратной комбинации измерительных каналов, физически установленных на приборе. Задание параметра осуществляется выбором строки из выпадающего списка серийных модификаций аналитических модулей (см. Таблицу 19.1).
  * **Функция интерактивного опроса \[Авто]:** Нажатие кнопки инициализирует принудительный аппаратный опрос контроллеров хроматографа. Программа самостоятельно считывает прошивку и определяет текущий состав измерительных плат.

{% hint style="warning" %}
Если выбранный в программе тип модуля не соответствует физически установленному в хроматографе «железу», справа от поля ввода инициализируется предупреждающая строка: _«В хроматографе: \[название реального модуля]»_. Сбор данных блокируется. Предупреждение исчезает только после выбора корректной аппаратной конфигурации.
{% endhint %}

* **Частота измерения:** Задание частоты опроса АЦП активного детектора. Доступны три фиксированных режима (изменение параметра возможно строго на нулевом этапе работы, до начала записи хроматограммы):
  * **`25 Гц` (Стандартный режим):** Оптимальная частота для работы со стандартными капиллярными колонками;
  * **`10 Гц`:** Выбирается при использовании насадочных (набивных) колонок. Обеспечивает минимальный уровень флуктуационных шумов АЦП;
  * **`50 Гц`:** Режим для работы с высокоскоростными поликапиллярными колонками. Характеризуется максимальным уровнем аппаратного шума АЦП.
* **Автостарт хроматографа без компьютера:** Активация чекбокса разрешает прибору автономный запуск встроенной технологической программы анализа по нажатию физической кнопки «Пуск» на корпусе ГХ, даже если логическое соединение с управляющим ПК временно разорвано или компьютер выключен.

### Дополнительные температурные зоны

Панель предназначена для программного распределения независимых физических каналов нагрева и термостатирования. Корректное объявление количества зон критически важно: на основе этих данных система динамически формирует структуру и вкладки параметров в узле _«Управление -> Температуры»_ каждого рабочего метода.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (206).png" alt=""><figcaption></figcaption></figure></div>

#### Конфигурируемые аппаратные параметры:

* **Количество детекторов:** Установка числа независимых блоков детектирования, требующих поддержания рабочей температуры (доступный диапазон ввода: от 1 до 3);
* **Количество испарителей:** Указание числа установленных на приборе портов ввода пробы (испарительных камер), снабженных индивидуальными нагревательными элементами;
* **Количество доп. устройств:** Задание числа прочих вспомогательных узлов хроматографического комплекса, требующих автономного регулирования температуры (например, обогреваемые газовые краны-дозаторы, линии отбора проб, термостаты пробоподготовки).
