# Сеть хроматографов

## Выбор и добавление хроматографа

Модуль предназначен для централизованного мониторинга и добавления аналитического оборудования, подключенного к управляющей рабочей станции. Перед началом настройки внутренней конфигурации аналитических модулей прибора, его необходимо зарегистрировать в локальной программной сети комплекса **«NetChrom 3.0»**.



#### Сетевое подключение и регистрация приборов

Перед началом настройки внутренней конфигурации аналитических модулей газового хроматографа «Кристаллюкс-4000М», прибор необходимо зарегистрировать в локальной программной сети комплекса **«NetChrom»**.

**Порядок вызова диспетчера сети**

Регистрация и администрирование подключенного оборудования осуществляются через специализированный интерфейс **«Сеть хроматографов»**. Вызов данного окна доступен из двух зон программного комплекса:

1. **На стартовой странице программы** — в правом верхнем углу главного приветственного интерфейса.
2. **В окне «Управление хроматографом»** — на верхней панели инструментов (используется для оперативного переключения между приборами в процессе работы).

**Алгоритм добавления хроматографа в систему**

Для внесения нового физического прибора в список доступного оборудования выполните следующие действия:

1. Инициализируйте окно «Сеть хроматографов» нажатием на соответствующую пиктограмму (изображение четырех связанных узлов с индикатором выпадающего списка).
2. В нижней части открывшейся панели нажмите кнопку **«Добавить хроматограф»**.
3. В появившемся диалоговом окне введите сетевые параметры прибора (IP-адрес или уникальный идентификатор контроллера в локальной сети) и подтвердите ввод.
4. В общем списке панели отобразится новая строка с графической иконкой хроматографа и присвоенным по умолчанию сетевым именем (например, _«Хроматограф 1»_).

**Назначение активного хроматографа и переход к управлению**

Для переключения программного комплекса на работу с конкретным хроматографом из общего списка зарегистрированного оборудования и перехода в окно управления оператор может использовать один из двух алгоритмов:

* **Через иконку прибора:** Нажмите левой клавишей мыши непосредственно на графическую иконку или текстовое наименование требуемого хроматографа внутри панели списка. В результате выбранный прибор назначится активным, информационное табло в верхнем правом углу главного окна программы мгновенно обновится, а система переведет интерфейс со стартовой страницы в главное окно **«Управление хроматографом»**.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (251).png" alt=""><figcaption><p>Активный хроматограф</p></figcaption></figure></div>

* **Через командную кнопку:** Нажмите на кнопку **«Управлять прибором»** (пиктограмма с изображением стрелки), расположенную в правой части строки целевого хроматографа. Программный комплекс автоматически переведет интерфейс в главное окно контроля, телеметрии и настройки параметров данного физического блока.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (252).png" alt=""><figcaption><p>Прямой переход к окну Управление хроматографом</p></figcaption></figure></div>

После успешного перехода в окно **«Управление хроматографом»** на верхней панели инструментов активируется кнопка **«Конфигурация хроматографа»**, открывающая доступ к детальной настройке аппаратных модулей, измерительных плат и технологических лимитов прибора.

## Конфигурация сети

Для изменения состава лабораторного оборудования или оперативного переключения между приборами используется функциональный элемент **«Сеть хроматографов»**.

#### **Доступные операции управления списком:**

* **Развертывание сетевой структуры:** Нажатие на интерактивный маркер сети (изображение четырех связанных узлов с индикатором выпадающего списка) позволяет мгновенно свернуть или развернуть подробный древовидный список подключенных физических устройств и их измерительных каналов.
* **Добавление нового прибора:** Если целевой хроматограф отсутствует в перечне, нажмите кнопку **«Добавить хроматограф»**, расположенную на нижней навигационной панели всплывающего окна,. В общем списке отобразится новая строка с графической иконкой хроматографа и присвоенным по умолчанию сетевым именем.
* **Исключение из конфигурации:** Для удаления неиспользуемого или находящегося на техническом обслуживании оборудования выберите прибор в списке и нажмите кнопку **«Удалить из сети»** (пиктограмма корзины). Хроматограф будет исключен из общего контура оперативного мониторинга. Данная операция не стирает архивные данные хроматограмм, а лишь разрывает активную программную связь с физическим контроллером.

#### Оперативный диспетчерский контроль

Встроенная система телеметрии выводит критически важную статусную информацию непосредственно в интерфейс сетевого модуля под уникальным именем каждого прибора:

* **Текущий технологический этап** (например, _«Этап: Нулевой»_, _«Этап: Охлаждение»_, _«Выход на режим»_, _«Готовность»_);
* **Временная метка** (общее время в формате `ЧЧ:ММ:СС` с момента инициализации текущего процесса или старта сессии физического анализа).

Данное архитектурное решение позволяет дежурному инженеру или начальнику лаборатории осуществлять сквозной визуальный контроль состояния всех аналитических комплексов предприятия в рамках единого рабочего окна.
