# Запуск анализа

## Запуск анализа

Данная панель предназначена для окончательной подготовки прибора к выполнению измерений. Она позволяет связать аппаратные ресурсы хроматографа с конкретной аналитической методикой и данными об исследуемом образце.

### Параметры запуска

Блок разделен на две идентичные зоны для независимого управления каналами: **Основной метод (Старт 1)** и **Метод сателлита (Старт 2)** (если включен в конфигурации).

* **Выбор метода:** Выпадающий список для выбора подготовленного файла методики (.met).
* **Управляющие кнопки:**
  * **Открыть метод:** Быстрый переход к редактированию выбранной методики.
  * **Создать метод:** Запуск файлового менеджера для создания новой методики «с нуля».
  * **Передать:** Отправка параметров выбранного метода в контроллер хроматографа (установка температур, расходов и токов).
* **Назначение хроматограммы:** Определяет статус выполняемого анализа для автоматической сортировки в базе данных. Доступны варианты:
  * **Анализ** — рабочее измерение пробы.
  * **Градуировка** — выполнение анализа для построения или корректировки калибровочного графика.
  * **Поверка** — контрольное измерение для проверки метрологических характеристик.
  * **Холостой анализ** — проверка чистоты системы и стабильности нулевой линии.

{% hint style="info" %}
Рекомендуется создавать отдельные методы для каждого типа анализа (например, «Анализ природного газа», «Градуировка спиртов»). Это исключит ошибки ручного ввода параметров перед каждым пуском.
{% endhint %}

#### Паспорт пробы

Содержит метаданные, которые будут неразрывно связаны с файлом хроматограммы. Вкладки **«Основной»** и **«Сателлит»** позволяют задавать разные описания для параллельных потоков данных.

* **Проба/Анализ:** Краткое наименование образца.
* **Номер пробы / Номер анализа:** Цифровые индексы для ведения сквозной нумерации в журнале лаборатории.
* **Дата и Время отбора:** Информационные поля (по умолчанию заполняются текущим системным временем).
* **Оператор:** ФИО сотрудника, проводящего измерения.
* **Место отбора пробы:** Дополнительная информация о происхождении образца.
* **Комментарий:** Поле для произвольных заметок (например, указание веса навески или условий пробоподготовки).

#### Порядок выполнения запуска анализа

1. Выберите требуемый файл в поле **«Основной метод»**.
   1. Если для текущей задачи в списке **«Выберите метод»** отсутствует подходящий файл, воспользуйтесь инструментами быстрого создания на панели «Параметры запуска»:
      1. **Создать метод с нуля**
      2. **Отредактировать существующий метод:**
         * Если у вас есть похожий метод, выберите его в списке и нажмите **«Открыть метод»**.
         * Внесите необходимые правки в параметры.
2. Нажмите кнопку **«Передать»** и дождитесь выхода прибора в режим готовности (все индикаторы в панели мониторинга должны стать зелеными).
3. Установите **Назначение хроматограммы**.
4. Заполните поля в блоке **«Паспорт пробы»**.
5. Выполните ввод пробы и нажмите кнопку **«Пуск»** на панели инструментов или корпусе прибора.

{% hint style="info" %}
Пока параметры не «переданы», хроматограф продолжает работать на установках предыдущего метода или в режиме ожидания. Только после нажатия кнопки «Передать» прибор начнет изменять температуры и расходы под новую задачу.
{% endhint %}
