# !Запуск анализа

## Запуск анализа

Данная панель предназначена для окончательной подготовки прибора к выполнению измерений. Она позволяет связать аппаратные ресурсы хроматографа с конкретной аналитической методикой и данными об исследуемом образце.

### Параметры запуска

Блок разделен на две идентичные зоны для независимого управления каналами: **Основной метод (Старт 1)** и **Метод сателлита (Старт 2)** (если включен в конфигурации).

* **Выбор метода:** Выпадающий список для выбора подготовленного файла методики (.met).
* **Управляющие кнопки:**
  * **Открыть метод:** Быстрый переход к редактированию выбранной методики.
  * **Создать метод:** Запуск файлового менеджера для создания новой методики «с нуля».
  * **Передать:** Отправка параметров выбранного метода в контроллер хроматографа (установка температур, расходов и токов).
* **Назначение хроматограммы:** Определяет статус выполняемого анализа для автоматической сортировки в базе данных. Доступны варианты:
  * **Анализ** — рабочее измерение пробы.
  * **Градуировка** — выполнение анализа для построения или корректировки калибровочного графика.
  * **Поверка** — контрольное измерение для проверки метрологических характеристик.
  * **Холостой анализ** — проверка чистоты системы и стабильности нулевой линии.

{% hint style="info" %}
Рекомендуется создавать отдельные методы для каждого типа анализа (например, «Анализ природного газа», «Градуировка спиртов»). Это исключит ошибки ручного ввода параметров перед каждым пуском.
{% endhint %}

#### Паспорт пробы

Содержит метаданные, которые будут неразрывно связаны с файлом хроматограммы. Вкладки **«Основной»** и **«Сателлит»** позволяют задавать разные описания для параллельных потоков данных.

* **Проба/Анализ:** Краткое наименование образца.
* **Номер пробы / Номер анализа:** Цифровые индексы для ведения сквозной нумерации в журнале лаборатории.
* **Дата и Время отбора:** Информационные поля (по умолчанию заполняются текущим системным временем).
* **Оператор:** ФИО сотрудника, проводящего измерения.
* **Место отбора пробы:** Дополнительная информация о происхождении образца.
* **Комментарий:** Поле для произвольных заметок (например, указание веса навески или условий пробоподготовки).



#### Порядок выполнения запуска анализа

1. Выберите требуемый файл в поле **«Основной метод»**.
   1. Если для текущей задачи в списке **«Выберите метод»** отсутствует подходящий файл, воспользуйтесь инструментами быстрого создания на панели «Параметры запуска»:
      1. **Создать метод с нуля**
      2. **Отредактировать существующий метод:**
         * Если у вас есть похожий метод, выберите его в списке и нажмите **«Открыть метод»**.
         * Внесите необходимые правки в параметры.
2. Нажмите кнопку **«Передать»** и дождитесь выхода прибора в режим готовности (все индикаторы в панели мониторинга должны стать зелеными).
3. Установите **Назначение хроматограммы**.
4. Заполните поля в блоке **«Паспорт пробы»**.
5. Выполните ввод пробы и нажмите кнопку **«Пуск»** на панели инструментов или корпусе прибора.

{% hint style="info" %}
Пока параметры не «переданы», хроматограф продолжает работать на установках предыдущего метода или в режиме ожидания. Только после нажатия кнопки «Передать» прибор начнет изменять температуры и расходы под новую задачу.
{% endhint %}























### 18.4 Пошаговые технологические алгоритмы для оператора

#### 📥 Сценарий 1. Порядок подготовки и запуска единичного анализа

Для выполнения измерения в режиме ручного ввода пробы (или при использовании автоматических дозаторов в режиме единичного пуска) оператор обязан строго соблюдать следующую последовательность действий:

1. **Инициализация рабочей области:** На панели управления (верхняя лента) нажмите кнопку **«Выполнить анализ»**. Программа активирует левую панель «Запуск анализа».
2. **Контроль режима:** Убедитесь, что ползунковый переключатель _«Режим серии анализов»_ находится в положении **«Выключен»**.
3. **Выбор методики обсчета:**
   * В поле **«Основной метод (Старт 1)»** из выпадающего списка выберите требуемый рабочий метод (`*.metx`). При необходимости нажмите под полем кнопку _«Открыть метод»_ для визуальной верификации параметров;
   * Если в системе активирована двухканальная схема детектирования, в поле **«Метод сателлита (Старт 2)»** укажите метод для ведомого канала. Если сателлит не используется, оставьте поле пустым.
4. **Задание параметров регистрации данных:**
   * В поле **«Объем пробы (мкл)»** укажите вводимое количество образца;
   * В поле **«Назначение хроматограммы»** выберите целевую папку базы данных и задайте имя файла (рекомендуется использовать лаконичные буквенно-цифровые маски без пробелов).
5. **Идентификация пробы (Паспорт):** Заполните информационные поля блока **«Паспорт пробы»** (Наименование, Номер анализа). Системные поля _«Дата отбора»_ и _«Время отбора»_ программа заполнит автоматически на основании системных часов ПК, однако при необходимости их можно скорректировать вручную. Внесите ФИО в строку _«Оператор»_.
6. **Выход хроматографа на рабочий режим:** Переведите взгляд на правую панель во вкладку **«Состояние»**.
   * Убедитесь, что фактические значения температур зон и расходов газов в таблицах сравнения полностью совпадают с заданными уставками;
   * Проверьте нижнюю панель _«Параметры вне нормы»_. В ней должно отображаться сообщение: **«Все параметры в норме»**;
   * На верхней ленте индикатор _«Неисправности»_ должен иметь нейтральный цвет.
7. **Контроль стабильности нулевой линии:** Нажмите кнопку **«Видеосамописец»** на верхней ленте. Оцените графический сигнал детектора: линия должна быть стабильной, без резких скачков (рывков) и выраженного высокочастотного шума.
8. **Активация записи и ввод пробы:**
   * Дождитесь, пока кнопка **«Старт анализа»** на верхней ленте станет активной (снимется программная аппаратная блокировка прибора);
   * Выполните физический ввод пробы в испаритель хроматографа (с помощью микрошприца или нажатием кнопки пуска на автосамплере);
   * Синхронно с вводом пробы нажмите левой кнопкой мыши кнопку **«Старт анализа»** (или клавишу **Пробел** на клавиатуре ПК) для инициализации процесса записи хроматограммы.

#### 🔍 Сценарий 2. Действия оператора при срабатывании телеметрической защиты

Если при подготовке комплекса или в процессе выхода на режим правая панель сигнализирует об отклонениях, выполните проверку по следующему алгоритму:

**Ситуация А: Индикатор «Неисправности» горит красным**

1. Нажмите на кнопку **«Неисправности»** на верхней ленте для вызова структурированного окна кодов ошибок.
2. Локализуйте узел сбоя по текстовому сообщению (например, _«Детектор 1 (ПИД): пламя отсутствует»_).
3. Перейдите во вкладку _«Состояние»_, найдите целевой узел и выполните регламентную процедуру (для ПИД — нажмите кнопку повторного автоматического поджига пламени в методе, предварительно проверив расходы водорода и воздуха).

**Ситуация Б: Резкое падение давлений или расходов во вкладке «Состояние»**

1. Если фактический расход газа через колонку (например, _Водород ПИД_ или _Сброс 1_) равен нулю, а во вкладке **«Диагностика»** в блоке **«Напряжение клапанов (В)»** значение по данному каналу достигло технологического максимума, **немедленно приостановите подготовку прибора**.
2. Перекройте подачу электропитания на нагревательные элементы (кнопка _«Режим сна»_ или _«Режим охлаждения»_), чтобы не допустить термической деструкции фазы хроматографической колонки в отсутствие потока газа-носителя.
3. Проверьте остаточное давление в баллонах (или на выходе генератора газов) и герметичность соединительных гаек на входе в хроматограф «Кристаллюкс-4000М». After восстановления давления в магистрали перезапустите метод кнопкой **«Выполнить анализ»**.







#### 🛠️ Сценарий 3. Порядок проведения инженерно-диагностического контроля комплексов

Процедура углубленного контроля электронных и пневматических узлов хроматографа «Кристаллюкс-4000М» проводится в рамках планового технического обслуживания, при валидации (поверке) прибора, а также при возникновении скрытых метрологических дефектов (повышенный дрейф базовой линии, нестабильность площадей пиков).

Для проведения сквозной цифровой диагностики без демонтажа панелей прибора выполните следующие действия:

1. **Переход в режим диагностики:** В правой панели мониторинга и статуса переключитесь на вкладку **«Диагностика»**.
2. **Верификация стабильности электропитания:** Локализуйте блок **«Параметры напряжения (В)»** и оцените стабильность внутренних шин:
   * **Поле «Контроль АЦП»:** Проверьте фактическое значение напряжения (номинал должен строго соответствовать паспортным данным АЦП, например, `2.49 В` или `5.00 В`). Флуктуации в пределах сотых долей вольта указывают на критический износ стабилизатора платы или помехи по заземлению, что приводит к росту шума на хроматограмме.
   * **Поле «Питающая сеть»:** Проверьте уровень напряжения на входе (норма: `220.00 В` $\pm$ 10%). При фиксации значений ниже `198 В` или выше `242 В` оператор обязан приостановить анализы и подключить управляющий комплекс через лабораторный стабилизатор напряжения во избежание сбоев в работе нагревателей термостата.
3. **Экспресс-тест герметичности и проходимости газовых линий:** Переведите взгляд на блок **«Напряжение клапанов (В)»**:
   * Сравните значения вольтажа на пропорциональных электромагнитных клапанах регуляторов пневматики (Сброс, Поддув, Водород и др.);
   * **Проверка на засор/перекрытие:** Если по любому из измерительных каналов напряжение находится в верхнем технологическом лимите (максимальный вольтаж открытия), а фактический расход во вкладке «Состояние» равен нулю — зафиксируйте факт отсутствия газа на входе или критического засора магистрали.
   * **Проверка на утечку (течь):** Если расход газа в норме, но напряжение на клапане аномально высокое по сравнению со стандартными рабочими логами метода, это указывает на скрытую разгерметизацию (утечку) в газовом тракте после регулятора (например, износ прокладки испарителя или плохая затяжка гайки колонки). Система вынуждена переоткрывать клапан, чтобы компенсировать потерю давления.
4. **Проверка целостности измерительных элементов детекторов:** Перейдите к блоку **«Ток детекторов (мА)»**:
   * Оцените энергопотребление активных детекторов (в первую очередь детектора по теплопроводности ДТП-1 / ДТП-2);
   * Если в активном рабочем методе задан рабочий ток питания (например, `100 мА`), а поле «Измеренное» на вкладке «Диагностика» стабильно отображает значение `0.00 мА`, зафиксируйте аппаратный отказ. Это однозначно указывает на **физический обрыв вольфрамовых спиралей (нитей)** в измерительной ячейке катарометра либо на срабатывание встроенной электронной защиты (автоматическое отключение тока при падении расхода газа-носителя во избежание перегорания нитей на воздухе).
5. **Документирование результатов:** По завершении проверки перейдите на верхнюю ленту интерфейса, откройте вкладку **«Журнал»** и убедитесь, что все зафиксированные в процессе диагностики аппаратные отклонения корректно залогированы системой с присвоением внутренних кодов ошибок.
