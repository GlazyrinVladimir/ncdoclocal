# Файловый менеджер

## Файловый менеджер (Журнал хроматограмм)

Диалоговое окно **«Открыть файл»** (Журнал хроматограмм) является основным инструментом для поиска, систематизации, предварительного просмотра и загрузки ранее записанных аналитических данных в программном комплексе **«NetChrom»**. Благодаря развитой функции многокритериальной фильтрации и контекстного поиска, система обеспечивает эффективное администрирование больших массивов данных.

### Интерфейс менеджера файлов

Рабочая область диалогового окна разделена на три основные функциональные зоны:

1. **Панель инструментов и фильтров (верхняя часть)** — элементы для быстрого отбора архивных данных по временным интервалам, применяемым аналитическим методикам и источникам измерительных сигналов.
2. **Таблица результатов (центральная часть)** — детализированный хронологический список зарегистрированных хроматограмм, соответствующих установленным критериям фильтрации.
3. **Панель выбора хроматографа и дерева папок (правая часть)** — интерактивный переключатель для привязки отображаемого массива данных к конкретному физическому оборудованию и навигации по встроенным каталогам системы.

### Панель инструментов, фильтров и поиска

Инструменты панели позволяют оператору мгновенно ограничить список отображаемых файлов, отсекая нецелевые массивы данных.

**1. Временной интервал**

Предназначен для выбора календарного периода, за который проводились хроматографические измерения. При нажатии на поле открывается выпадающий список предустановок:

* **Последние 24 часа / 7 дней / 14 дней / 30 дней** — быстрая выборка хроматограмм за указанный прошедший срок (на Рисунке 1 активирован фильтр _«Последние 30 дней»_).
* **За все время** — отображение всего накопленного архива базы данных (при значительном объеме логов загрузка списка может занять продолжительное время).
* **Диапазон дат...** — вызов интерактивного календаря для ручного задания произвольного временного интервала (с указанием точных дат начала и окончания периода).

**2. Фильтр «Выбор метода»**

Обеспечивает сортировку записей по наименованию файла аналитической методики (формат `*.metx`). В раскрывающемся списке отображаются исключительно те методы обсчета и калибровки, по которым физически проводились измерения на данном рабочем месте (например, метод `3816-ПФД_ПФД.metx`).

**3. Фильтр «Источник данных» (Основной / Сателлит)**

Позволяет разделять потоки хроматографических данных, полученные от разных физических модулей или независимых каналов в рамках одной аналитической системы:

* Выбор целевых измерительных потоков осуществляется установкой соответствующих флажков (чек-боксов).
* Текущие выбранные категории визуализируются в заголовке фильтра в виде динамических **тегов** (например, теги `Основной х` и `Сателлит х`). Нажатие на графический символ **«х»** на теле тега мгновенно сбрасывает данный критерий отбора.

**4. Интерактивное поле «Поиск...»**

Обеспечивает прецизионный контекстный поиск по мере ввода текста оператором (символ за символом). Поиск является сквозным и выполняется одновременно по трем информационным колонкам: **«Проба»**, **«Оператор»** и **«Комментарий»**.

### Список зарегистрированных хроматограмм

В центральной таблице фиксируется полная метаинформация о каждом выполненном анализе. Данные вносятся в базу в автоматическом режиме непосредственно в момент старта регистрации аналитического сигнала хроматографа.

**Структура метаданных таблицы:**

| Наименование колонки | Информационное наполнение                                                                           |
| -------------------- | --------------------------------------------------------------------------------------------------- |
| **Дата и время**     | Точное астрономическое время начала записи аналитического сигнала детектора.                        |
| **Проба**            | Краткое наименование или цифровой шифр образца (идентификатор), введенный оператором перед пуском.  |
| **Объем**            | Количественное значение объема введенной в испаритель пробы (в мл или мкл в зависимости от метода). |
| **Оператор**         | ФИО или уникальная учетная запись пользователя, выполнившего данное измерение.                      |
| **Назначение**       | Метрологическая категория замера (**Анализ**, **Градуировка**, **Поверка**, **Холостой**).          |
| **Комментарий**      | Дополнительные текстовые сведения об особенностях пробоподготовки или условиях проведения анализа.  |
| **Метод**            | Наименование файла аналитической методики (`*.metx`), использованной для регистрации и обсчета.     |
| **Тип расчета**      | Базовый алгоритм количественного расчета (например, _Внешний стандарт_, _Внутренняя нормализация_). |
| **Канал**            | Указание физического источника измерительного сигнала (_Основной_ или _Сателлит_).                  |

{% hint style="info" %}
Клик левой кнопкой мыши по заголовку любой колонки таблицы позволяет мгновенно отсортировать весь отображаемый список хроматограмм по возрастанию или по убыванию выбранного параметра.
{% endhint %}

### Идентификация хроматографа и навигация по каталогам

В правой части окна расположен специализированный блок, отвечающий за аппаратную привязку данных к конкретному хроматографическому оборудованию и структурирование файлов методик.

**Блок выбора прибора**

* **Выбор прибора:** С помощью **синего символа стрелки** (рядом с иконкой прибора) оператор может переключаться между различными газовыми хроматографами серии «Кристаллюкс-4000М», заведенными на данную рабочую станцию.
* **ID прибора:** Отображает уникальное сетевое имя прибора или его модель (на Рисунке 1 активен хроматограф `3816 2ПФД`).
* **Этап:** Информационное поле, указывающее текущий технологический статус прибора (например, _«Этап: Нулевой»_).
* **Длительность:** Фиксирует общее паспортное время физической записи выбранной хроматограммы.

**Дерево каталогов и встроенных папок**

Ниже блока прибора расположена иерархическая структура встроенного файлового менеджера, которая позволяет оперативно переключаться между целевыми системными директориями при помощи кнопок развертывания **(+)** и **(-)**:

* **Папка «Методы»** — хранилище для глобальных файлов рабочих аналитических методик (`*.metx`), привязанных к данному прибору (например, `3816-ПФД-ПФД.metx`). Клик по файлу в этом дереве позволяет быстро просмотреть его свойства.
* **Папка «Паспорта СО»** — специализированный каталог для хранения файлов калибровочных данных, паспортов стандартных образцов и градуировочных графиков (формат `*.grd`, например: `10-10-10.grd`, `14-10-7.grd`).
* **Папка «Архив»** — директория для долгосрочного хранения устаревших или защищенных от редактирования хроматограмм прошлых отчетных периодов.

{% hint style="warning" %}
Если при верно настроенных параметрах временных интервалов и фильтров методов центральный список файлов остается полностью пустым, убедитесь, что в правом блоке окна выбран правильный (активный в данный момент) хроматограф и открыта целевая рабочая папка.
{% endhint %}

## Порядок открытия хроматограммы

Для загрузки файла хроматограммы из архива в рабочее окно обработки оператор обязан строго соблюдать следующую последовательность действий:

1. Вызовите диалоговое окно открытия файла через главное системное меню программы: **«Файл»** — **«Открыть»**.
2. Верифицируйте правильность выбора целевого **Хроматографа** и активной папки в правой функциональной зоне окна.
3. Настройте требуемый **Временной интервал** поиска (например, _Последние 30 дней_) и, при необходимости, укажите жесткий **Фильтр по методу**.
4. Если требуется найти конкретную точечную пробу, введите ее шифр, ФИО лаборанта или шифр партии в интерактивное поле **«Поиск...»**.
5. Выделите целевую строку в сформированной таблице результатов левой кнопкой мыши и нажмите кнопку **«Открыть»** (либо выполните быстрое двойное нажатие левой кнопкой мыши по строке файла). Программа автоматически загрузит графический сигнал детектора и локальный метод в главное рабочее окно обработки.
