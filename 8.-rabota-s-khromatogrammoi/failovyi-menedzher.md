# !Файловый менеджер

### Работа с журналом хроматограмм

Окно **«Открыть файл»** (Журнал хроматограмм) является основным инструментом для поиска, систематизации и загрузки ранее записанных данных в программном комплексе **NetChrom**. Система позволяет работать с большими массивами данных благодаря развитой функции фильтрации и предварительного просмотра.

### Интерфейс менеджера файлов

Окно разделено на три основные функциональные зоны:

1. **Панель фильтров** (верхняя часть) — инструменты для быстрого отбора данных по времени, методу и источнику.
2. **Таблица результатов** (центральная часть) — детализированный список хроматограмм, соответствующих выбранным критериям.
3. **Панель выбора хроматографа** (правая верхняя часть) —  переключатель активного хроматографа.

### Панель фильтров и поиска

Панель позволяет мгновенно ограничить список отображаемых файлов.

#### Временной интервал

Позволяет выбрать период, за который проводились анализы. При нажатии на поле открывается список предустановок:

* **Последние 24 часа / 7 / 14 / 30 дней** — быстрая выборка за указанный срок.
* **За все время** — отображение всего архива (может занять время при большом объеме базы данных).
* **Диапазон дат...** — вызов календаря для задания произвольного интервала.

#### Выбор метода

Фильтрует записи по имени методического файла (**\*.metx**). В списке отображаются только те методы, по которым проводились измерения на данном рабочем месте.

#### Источник данных (Основной / Сателлит)

Позволяет разделять данные, полученные с разных физических модулей или приборов в рамках одной системы.

* Выбор осуществляется установкой флажков (чек-боксов).
* Выбранные категории отображаются в заголовке фильтра в виде **тегов**. Нажатие на символ **«х»** на теге быстро сбрасывает данный фильтр.

#### Поле «Поиск...»

Обеспечивает контекстный поиск по мере ввода текста. Поиск ведется одновременно по колонкам: **«Проба»**, **«Оператор»** и **«Комментарий»**.

### Список зарегистрированных хроматограмм

В таблице фиксируется полная информация о каждом анализе. Данные вносятся автоматически в момент старта регистрации сигнала.

| Колонка          | Описание                                                       |
| ---------------- | -------------------------------------------------------------- |
| **Дата и время** | Точное время начала записи сигнала.                            |
| **Проба**        | Название образца (идентификатор), введенное оператором.        |
| **Объем**        | Количественное значение введенной пробы (мл/мкл).              |
| **Оператор**     | ФИО или учетная запись пользователя, выполнившего анализ.      |
| **Назначение**   | Категория замера (Анализ, Градуировка, Поверка).               |
| **Комментарий**  | Дополнительные сведения об условиях анализа.                   |
| **Метод**        | Название файла методики (.metx), использованной для анализа.   |
| **Тип расчета**  | Алгоритм количественного расчета (например, Внешний стандарт). |
| **Канал**        | Указание источника                                             |

{% hint style="info" %}
Клик по заголовку любой колонки позволяет отсортировать список по возрастанию или убыванию.
{% endhint %}

### Идентификация и выбор хроматографа

В правой части окна расположен блок, отвечающий за привязку данных к конкретному оборудованию.

* **Выбор прибора**: С помощью **синего символа стрелки** (рядом с иконкой прибора) можно переключиться между различными хроматографами «Кристаллюкс-4000М», подключенными к станции.
* **ID прибора**: Отображает сетевое имя или серийный номер (например, **3791\_ПИД-ДТП**).
* **Этап**: Указывает статус обработки или номер цикла (например, «Нулевой»).
* **Длительность**: Показывает общее время записи выбранной хроматограммы (ЧЧ:ММ:СС).

{% hint style="warning" %}
**Внимание!** Если при верно настроенных фильтрах список файлов пуст, убедитесь, что в правом углу выбран правильный хроматограф.
{% endhint %}

### Порядок открытия хроматограммы

1. Вызовите окно открытия файла через меню **«Файл»** — **«Открыть»**.
2. Проверьте правильность выбора **Хроматографа** в правой части окна.
3. Настройте **Временной интервал** и, при необходимости, **Фильтр по методу**.
4. Если требуется найти конкретную пробу, введите её номер или название в поле **«Поиск»**.
5. Выделите нужную строку в таблице и нажмите кнопку **«Открыть»** (или выполните двойной клик по строке).
