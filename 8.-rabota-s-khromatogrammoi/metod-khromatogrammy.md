# Метод хроматограммы

Метод хроматограммы — это неотъемлемая часть зарегистрированного файла данных (хроматограммы), содержащая полную копию параметров аналитического прибора и математических алгоритмов обработки, которые действовали непосредственно в момент проведения конкретного измерения.

При открытии любого сохраненного файла данных (`*.chrx`) из архива через Файловый менеджер, программный комплекс «NetChrom» автоматически загружает именно его внутренний локальный метод.

## Назначение и фундаментальные свойства

Главная цель интеграции (вшивания) метода внутрь файла хроматограммы — обеспечение принципа целостности данных и воспроизводимости результатов химического анализа:

* **Защита от изменений:** Если глобальный рабочий метод на жестком диске компьютера будет случайно изменен, перезаписан или полностью удален другим сотрудником лаборатории, открытая хроматограмма гарантированно сохранит свои первоначальные настройки.
* **Историчность данных:** Внутренний метод гарантирует, что при открытии файла через месяц, год или несколько лет оператор увидит именно те условия (температуры узлов, расходы газов, градуировочные коэффициенты), при которых аналитический сигнал был физически получен и рассчитан. Это полностью исключает потерю архивных данных при обновлении парка методик лаборатории.

## Структура параметров метода хроматограммы

Для обеспечения метрологической достоверности внутренний метод условно разделен программным комплексом на два независимых блока параметров, доступных для просмотра оператором:

| Блок параметров          | Редактируемость в окне обработки | Описание входящих компонентов и ограничений                                                                                                                                                                                                                        |
| ------------------------ | -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Аппаратные параметры** | **Запрещено**                    | Фактические температуры испарителя, детекторов и термостата колонок; расходы и давления газов; тип установленной колонки; серийный номер прибора. Данные записываются жестко в момент анализа физическим контроллером ГХ и изменению в открытом файле не подлежат. |
| **Параметры обработки**  | **Разрешено**                    | Таблица компонентов (названия пиков), времена удерживания, градуировочные коэффициенты, формулы количественного расчета, настройки интегратора (ширина пика, пороги чувствительности, события интеграции).                                                         |

{% hint style="info" %}
В методе хроматограммы можно свободно изменять любые параметры, кроме параметров разделов «Управление» и «Устройства». Программа намертво блокирует ручное редактирование любых полей, отвечающих за физическое управление газовым хроматографом на момент вкола. Это защищает архивный файл от фальсификации условий проведения хроматографического разделения.
{% endhint %}

## Порядок работы с методом хроматограммы

В процессе исследования и доводки графического сигнала оператору доступны следующие регламентированные операции с локальным методом:

**1. Модификация расчета (Работа в «песочнице»)**

Оператор имеет право изменять любые параметры интеграции, добавлять новые компоненты в таблицу, двигать временные ворота идентификации или корректировать градуировку внутри данного конкретного файла хроматограммы.

При выполнении этих действий исходный глобальный рабочий метод (`*.metx`), хранящийся на диске ПК, **не меняется**. Это позволяет лаборанту свободно экспериментировать с разметкой пиков без риска испортить общую рабочую методику.

**2. Экспорт во внешний файл и фиксация эталона**

Если обработка текущей хроматограммы была полностью оптимизирована (подобраны идеальные пороги, события интегратора и верно подписаны пики) и эти настройки необходимо применить ко всем будущим анализам, оператор выполняет экспорт.

Для этого нажмите на стрелку рядом с кнопкой **«Сохранить»** на верхней панели инструментов. Система откроет выпадающий список команд:

* **Сохранить** (или `Ctrl+S`) — записывает внесенные изменения разметки внутрь текущего файла хроматограммы.
* **Сохранить как** — позволяет сохранить текущий файл данных под новым уникальным именем в архиве.
* **Сохранить как рабочий метод** — ключевой инструмент «обучения» системы. Программа экспортирует все настроенные параметры обработки из текущей хроматограммы и создаст (либо перезапишет) независимый внешний файл глобального рабочего метода (`*.metx`). Все последующие пуски хроматографа по этой методике будут автоматически использовать эти калибровочные настройки.

**3. Печать параметров метода**

При необходимости оператор может мгновенно вывести полную структуру и все математические уставки внутреннего метода хроматограммы на бумажный носитель или экспортировать в PDF-файл. Для этого достаточно нажать кнопку **«Печать метода»** на верхней панели управления.

Этого функционала полностью достаточно для выполнения обязательных регламентных требований при верификации результатов химического анализа в аккредитованных испытательных лабораториях, а также при прохождении процедур внешнего государственного аудита.

## Абсолютная автономность файлов хроматограмм

Важнейшей софтверной особенностью архитектуры базы данных «NetChrom» является полная контейнеризация файлов данных.

При отправке файла хроматограммы по электронной почте, копировании на внешний USB-носитель или переносе на любой другой удаленный ПК, все настроенные параметры градуировки, калибровочные графики, коэффициенты и уставки интеграции **автоматически передаются внутри самого файла хроматограммы**. Наличие исходного файла внешнего рабочего метода (`*.metx`) на целевом компьютере не требуется. Хроматограмма откроется на любом рабочем месте в первозданном виде и со всеми расчетными данными.

Несмотря на полную автономность файла данных, при его переносе на сторонний компьютер убедитесь, что установленная там версия ПК «NetChrom» является аналогичной или более новой. Попытка открыть современный файл хроматограммы на устаревшей версии софта может привести к некорректному считыванию специфических отраслевых моделей расчета результатов (например, модулей для обсчета природного газа или спиртов).
