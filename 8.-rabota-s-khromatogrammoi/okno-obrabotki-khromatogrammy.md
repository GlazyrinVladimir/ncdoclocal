# !Окно обработки хроматограммы

\
Данное окно является основным рабочим пространством оператора при анализе результатов. Оно объединяет инструменты графического анализа, средства автоматической обработки и таблицы с расчетными данными.

### Верхняя панель инструментов (Панель управления)

Панель инструментов обеспечивает быстрый доступ к операциям с файлом и методиками расчета.

* **Открыть / Сохранить**: Команды для загрузки хроматограмм из архива и сохранения внесенных изменений (например, после ручной корректировки пиков).
* **Метод хроматограммы**: Открывает параметры метода, которые были сохранены внутри данного файла данных.
  * _Примечание:_ Изменение параметров здесь влияет только на текущую хроматограмму, не затрагивая основной файл рабочего метода.
* **Градуировка**: Переход в модуль построения калибровочных графиков по стандартным образцам.
* **Обработать**: Запуск процедуры поиска пиков (интегрирования) и расчета концентраций. Наличие **синего треугольника** позволяет выбрать режим: только расчет или полная переработка.
* **Поверка**: Специализированный модуль для контроля метрологических характеристик прибора.
* **Внешний расчет**: Передача данных в сторонние модули обработки.
* **Печать**: Формирование протокола анализа.

**Ниже расположена панель вкладок:**\
Она позволяет переключаться между открытыми хроматограммами и методами. Активная вкладка выделяется светлым фоном. Название вкладки хроматограммы обычно содержит дату и время её регистрации.

### Графическое поле (Хроматограмма)

Центральная зона, где отображается аналитический сигнал.

* **Отображение сигнала**: Кривая линия (синего цвета по умолчанию) отображает отклик детектора во времени.
* **Идентификация пиков**:
  * **Цветовая заливка (голубая область)** — указывает на площадь пика, которая была учтена при расчете.
  * **Базовая линия (розовая линия)** — линия, от которой программа отсчитывает высоту и площадь пика.
  * **Подписи** — над вершиной пика указывается его время удерживания и название компонента (например, **0.65; Водород**).
* **Легенда (вверху слева)**: Позволяет включать или отключать отображение каналов (например, **ДТП-1**).

### Нижняя левая панель (Таблицы результатов)

В данной области сосредоточены данные количественного анализа, разбитые по тематическим вкладкам:

* **Пики**: Подробная таблица всех найденных пиков. Содержит колонки: время, детектор, название компонента, высота, площадь, ширина и др.
* **Компоненты**: Итоговая таблица с рассчитанными концентрациями (массовая доля, объемная доля и т.д.).
* **Интегрирование**: Список программных фильтров (пороги, шумы), которые использовались для поиска пиков на этой хроматограмме.
* **Журнал**: Список всех операций, произведенных с данным файлом.

### Паспорт хроматограммы (Нижняя правая панель)

Здесь отображается  информация об анализе, необходимая для формирования отчета.

* **Вкладка «Проба»**:
  * **Номер пробы** и **Номер анализа** — для учета в журнале лаборатории.
  * **Дата и время отбора** — фиксируются для контроля оперативной работы.
  * **Оператор** — ФИО сотрудника, выполнившего анализ.
  * **Описание** — текстовое поле для произвольных пометок.
* Вкладка «Хроматограмма» отображает технические параметры регистрации конкретного файла данных:
  * **Тип хроматографа**: Модель прибора (например, **Кристаллюкс-4000М**).
  * **Модуль детекторов**: Конфигурация установленных детекторов (например, **ПИД-ДТП**).
  * **Хроматограф**: Сетевое имя или серийный номер конкретного прибора.
  * **Дата / Время**: Точные системные данные о моменте начала записи сигнала.
  * **Комментарий**: Текстовое поле для кратких заметок к данному файлу (например, «Определение состава смеси»).
* Вкладка «Колонка» - данная вкладка содержит паспортные данные разделительной колонки, на которой был выполнен текущий анализ. Заполнение этих полей необходимо для корректного ведения лабораторного журнала и соблюдения требований методик выполнения измерений .**Основные параметры:**
* **Название колонки** — полное наименование.
* <mark style="color:$danger;">**Тип колонки**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">— выбор из выпадающего списка.</mark>
* **Производитель** — завод-изготовитель или бренд.
* **Длина (м.)** — фактическая длина колонки в метрах.
* **Диаметр (мм.)** — внутренний диаметр колонки.
* **Описание** — дополнительная информация (например, толщина неподвижной фазы или дата установки колонки в прибор).

### Индикатор состояния (Верхний правый угол)

Отображает информацию о приборе, с которого получены данные:

* **ID прибора** (например, 3791).
* **Этап и Длительность** анализа.
* **Статус сети**: Индикация связи с хроматографом.

{% hint style="info" %}
**Важно!** Любые изменения, внесенные в паспорт пробы или в границы пиков на графике, требуют нажатия кнопки **«Сохранить»** на верхней панели, иначе при закрытии вкладки данные будут утеряны.
{% endhint %}
