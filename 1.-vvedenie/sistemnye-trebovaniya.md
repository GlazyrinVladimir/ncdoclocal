# Системные требования

## Системно-аппаратные требования к персональному компьютеру

Для обеспечения стабильного функционирования программного обеспечения **«NetChrom 3.0»**, корректного сбора данных в реальном времени и выполнения ресурсоемких математических расчетов, управляющая рабочая станция должна соответствовать приведенным ниже аппаратным и системным конфигурациям.

### Минимальные системные требования

Конфигурация, обеспечивающая запуск ПО и сбор данных с одного хроматографического канала.

* **Процессор (CPU):** Многоядерный процессор (Intel Core, AMD Ryzen или аналогичный) с тактовой частотой не менее **2.0 ГГц**.
* **Оперативная память (RAM):** Не менее **8 ГБ**.
* **Дисковая подсистема:** Свободное пространство на накопителе не менее **80 ГБ** (для установки ПО и локального архива данных).
* **Видеоподсистема и монитор:** Разрешение экрана не менее **1280×1024 пикселей** (глубина цвета 24/32 бит).
* **Интерфейсы связи с хроматографом:**
  * Свободный сетевой порт **Ethernet (RJ-45)** со скоростью 10/100 Мбит/с (рекомендуется);
  * Порт **USB 2.0 / USB 3.0** (для подключения через виртуальный COM-порт или преобразователи интерфейсов);
  * Физический последовательный порт **COM (RS-232C)**.
* **Устройства ввода:** Стандартная клавиатура и манипулятор типа «мышь».
* **Периферийные устройства:** Любой принтер с установленным в ОС драйвером печати.
* **Операционная система:** Лицензионная версия **Microsoft Windows 10 (64-bit)** с установленными актуальными пакетами обновлений.

### Рекомендуемые системные требования

Конфигурация для комфортной многозадачной работы, ведения больших баз данных и одновременного управления несколькими хроматографами.

* **Процессор (CPU):** Высокопроизводительный процессор уровня **Intel Core i3 / i5 / i7** или **AMD Ryzen 3 / 5 / 7** с тактовой частотой от **2.5 ГГц** и выше.
* **Оперативная память (RAM):** **16 ГБ** и более (минимизирует задержки интерфейса при параллельной обработке многоканальных серий анализов).
* **Дисковая подсистема:** Накопитель типа **SSD / NVMe** объемом от **240 ГБ** (существенно ускоряет инициализацию СУБД, чтение и запись больших массивов хроматограмм).
* **Видеоподсистема и монитор:** Разрешение экрана **Full HD (1920×1080) или выше** (рекомендуются широкоформатные мониторы для одновременного панорамного обзора хроматограммы, дерева объектов и таблицы результатов).
* **Интерфейсы связи:**
  * Выделенный сетевой адаптер **Ethernet (LAN)** на материнской плате ПК для интеграции прибора в локальную сеть лаборатории;
  * Свободные высокоскоростные порты **USB 3.0/3.2** (при использовании сертифицированных преобразователей интерфейсов USB-COM);
  * Наличие сетевого коммутатора (свитча) в случае одновременного подключения пула хроматографов к одной рабочей станции по протоколу TCP/IP.
* **Периферийные устройства:** Лазерный принтер со скоростью печати не менее 20 стр/мин для потокового вывода паспортов хроматограмм и отчетов.
* **Операционная система:** Лицензионная версия **Microsoft Windows 11 (64-bit).**

{% hint style="info" %}
Для корректной работы сетевых компонентов ПО «NetChrom 3.0» учетная запись пользователя в операционной системе Windows должна обладать правами **Администратора** или явными правами на запись в рабочую директорию базы данных. В сетевом экране Windows должны быть открыты соответствующие порты для обмена данными по протоколу TCP/IP.
{% endhint %}

## Рекомендуемые настройки операционной системы и правила эксплуатации

Для обеспечения бесперебойной регистрации хроматографических данных в реальном времени и корректного отображения интерфейса ПО «NetChrom 3.0», операционную систему Windows необходимо предварительно настроить в соответствии с нижеописанными правилами.

### Настройка виртуальных и физических COM-портов

* **Фиксация номеров портов:** При использовании преобразователей интерфейсов (USB-to-UART/RS-232) портам в Диспетчере устройств Windows должны быть принудительно назначены постоянные порядковые номера (например, `COM1`, `COM2`). Автоматическая динамическая переназначение номеров портов операционной системой при перезапуске ПК не допускается.
* **Аппаратное согласование:** При использовании многопортовых плат расширения (PCI-Express RS-232/RS-422) тип контроллера и версии драйверов должны быть согласованы с заводом-изготовителем хроматографа для исключения конфликтов прерываний и потери пакетов данных.

### Электропитание и энергосбережение

* **Запрет спящего режима:** В настройках схемы управления питанием Windows («Панель управления» -> «Электропитание») для текущего плана должен быть установлен режим **«Высокая производительность»**. Переход компьютера в спящий или ждущий режим должен быть полностью отключен (**параметр «Никогда»**).
* **Энергосбережение USB-контроллеров:** В Диспетчере устройств для всех корневых USB-концентраторов, к которым подключен хроматограф, необходимо во вкладке «Управление электропитанием» **снять флажок** с пункта _«Разрешить отключение этого устройства для экономии энергии»_. В противном случае ОС может разорвать связь с прибором во время длительного анализа.

### Настройки графической подсистемы и интерфейса

* **Масштабирование (DPI):** Интерфейс ПО «NetChrom 3.0» оптимизирован под стандартное масштабирование Windows (100%). При использовании мониторов высокого разрешения (FullHD и выше) и включении системного масштабирования (125% или 150%) рекомендуется в свойствах исполняемого файла программы (`.exe`) во вкладке «Совместимость» настроить _«Переопределение масштабирования с высоким разрешением»_ с управлением со стороны приложения. Это исключит размытие шрифтов и наложение таблиц.
* **Качество цветопередачи:** В настройках дисплея должна быть установлена максимальная глубина цвета (32 бита / True Color) для корректного рендеринга векторных графиков хроматограмм.

### Особенности настройки печати отчетов

* **Оптимизация под монохромную печать:** Поскольку большинство лабораторных отчетов печатается на черно-белых лазерных принтерах, в настройках графического движка ПО «NetChrom 3.0» рекомендуется выбрать контрастную цветовую схему (например, черная линия хроматограммы на белом фоне). Это обеспечит четкую видимость базовой линии и пиков без потери качества при копировании документов.
* **Режим компоновки отчета:** При серийных анализах рекомендуется использовать функцию вывода графического поля хроматограммы на отдельный лист (или в верхнюю половину листа формата А4), оставляя нижнюю часть под структурированные таблицы результатов и паспорт анализа.

### Общие правила безопасной эксплуатации

* **Приоритет фонового процесса:** Во время выполнения сессии анализа и сбора данных в реальном времени категорически запрещается запуск стороннего программного обеспечения, создающего пиковые нагрузки на центральный процессор или дисковую подсистему (архивация больших массивов данных, сканирование антивирусом, 3D-графика, игры).
* **Блокировка изменения разрешения:** Не допускается запуск приложений, способных принудительно или динамически изменять разрешение экрана управляющей рабочей станции в процессе работы «NetChrom 3.0».
