# Справка для самых маленьких

## Элементы диалога

Ниже приведён оформленный для справочной системы раздел «Элементы диалога» программы «NetChrom 3.0». Формат соответствует стилю справочных материалов Windows и готов к встраиванию в CHM или аналогичный help-файл.

### Элементы диалога <a href="#undefined" id="undefined"></a>

Программа использует следующие элементы диалога для взаимодействия с пользователем:

### Панель инструментов

<figure><img src="../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

В программе «NetChrom V3.0» для удобства работы с различными функциональными областями предусмотрена панель инструментов (тулбар). На каждом рабочем окне (view) программы отображается собственный набор кнопок панели инструментов, соответствующий выполняемым в этом окне действиям. Такой подход обеспечивает быстрый и удобный доступ к наиболее часто используемым командам и функциям в контексте текущей задачи, повышая эффективность работы пользователя.

### Кнопка - ?

Объёмный прямоугольник с текстом, указывающим назначение. Для активации нажмите левой кнопкой мыши. Кнопка по умолчанию обведена чёрным прямоугольником.

### Строка ввода

<figure><img src="../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

Вдавленный прямоугольник с надписью слева или сверху. Для ввода данных щёлкните левой кнопкой мыши — появится мигающий курсор. Введите данные с клавиатуры и нажмите клавишу ввода.

### Переключатель

<figure><img src="../.gitbook/assets/image (3).png" alt=""><figcaption></figcaption></figure>

Вдавленный квадрат с надписью справа. Имеет два состояния: включено (галочка внутри) и выключено. Для изменения состояния нажмите левой кнопкой мыши.

### Полоса прокрутки

Содержит две кнопки-стрелки и ползунок. Используется для просмотра информации, которая не помещается на экране. Чтобы переместить ползунок, наведите на него курсор, нажмите и удерживайте левую кнопку мыши, затем передвиньте ползунок и отпустите кнопку.

### Кнопка панели инструментов

Объёмный квадрат с пиктограммой, дублирующей команду меню. При наведении курсора отображается всплывающая подсказка с названием команды. Нажатие запускает соответствующую команду.

### Выпадающий список

Вдавленный прямоугольник с кнопкой справа в виде перевёрнутого треугольника. Для выбора значения нажмите кнопку, в открывшемся списке щёлкните по нужному варианту. Список свернётся, выбранное значение отобразится в прямоугольнике.

### Иерархический список

Вдавленный прямоугольник, отображающий данные в виде дерева с узлами. Узлы с дочерними элементами имеют слева квадрат с «+» для раскрытия подузлов. Щелчок «+» разворачивает список, меняя знак на «–». Повторный щелчок сворачивает список. Щелчок по элементу вызывает действие.

### Выпадающее меню

Вызывается нажатием правой кнопки мыши по элементам интерфейса (например, панель графиков). Для выбора пункта наведите курсор и щёлкните левой кнопкой мыши. Выполнение пункта отображено в его названии.

### Орган управления ползункового типа

Шкала с делениями и ползунком-стрелкой. Ползунок можно передвигать мышью или пошагово, нажимая слева или справа от него.

### Кнопка ОК

Подтверждает изменения и закрывает диалоговое окно.

### Кнопка Отмена

Отменяет изменения и закрывает диалог. Аналогично срабатывает клавиша Esc.

### Условные обозначения и аббревиатуры <a href="#undefined" id="undefined"></a>

Данные сведения носят ознакомительный характер и помогают эффективнее организовать работу.

### Используемые аббревиатуры

* ДТП — детектор по теплопроводности
* ПИД — пламенно-ионизационный детектор
* ЭЗД — электронозахватный детектор
* ТИД — термоионный детектор
* ПФД — пламенно-фотометрический детектор
* ФИД — фотоионизационный детектор
* АЦП — аналого-цифровой преобразователь
* ПК — персональный компьютер
* РРГ — регулятор расхода газа
* РД — регулятор давления газа
* ИРГ — измеритель расхода газа
* ДАГ — дозатор автоматический газовый
* ДАЖ — дозатор автоматический жидкостный
* ДРП — дозатор равновесного пара
* СКО — среднеквадратичное отклонение
* ВУФ — вакуумная ультрафиолетовая лампа
