# !!!!Расчёты

## !!!!!!!!!!!!!!!!Расчёты&#x20;

Вкладка "Количественный расчет"

<mark style="color:$primary;">В данном разделе выбирается математическая модель, по которой программа будет преобразовывать площадь или высоту пика в значение концентрации компонента.</mark>

{% hint style="warning" %}
Содержимое вкладки зависит от выбранного типа расчета.
{% endhint %}

<figure><img src="../.gitbook/assets/image (61).png" alt=""><figcaption></figcaption></figure>

### !!!!!!!!!Нормализация

Тип расчета - [**Нормализация**](../) не предусматривает заполнения дополнительных данных для расчета результатов анализа.



<mark style="color:$primary;">Обязательно проверьте, чтобы для всех компонентов были введены</mark> <mark style="color:$primary;"></mark><mark style="color:$primary;">**относительные чувствительности (поправочные коэффициенты)**</mark><mark style="color:$primary;">. Без них расчет будет верным только для смеси очень похожих веществ. Если пик какого-то вещества не идентифицирован, метод нормализации может «наврать», распределив его долю между остальными.</mark>

<br>

### !!!!!!!Внутренняя нормализация

Метод [**Внутренней нормализации**](../) предусматривает ряд дополнительных расчетов согласно различным нормативным документам.&#x20;

<mark style="color:$primary;">Метод критически чувствителен к</mark> <mark style="color:$primary;"></mark><mark style="color:$primary;">**объему ввода**</mark><mark style="color:$primary;">. Если вы используете ручной ввод шприцем, любая ошибка в микролитрах даст ошибку в результате. Для повышения точности в настройках метода рекомендуется использовать среднее значение по нескольким калибровочным вводам.</mark>





<figure><img src="../.gitbook/assets/image (45).png" alt=""><figcaption></figcaption></figure>

В зависимости от нормативного документа выберите из списка "Дополнительный расчет":&#x20;

<figure><img src="../.gitbook/assets/image (154).png" alt=""><figcaption></figcaption></figure>

Размерность выбирается в %об. или %масс.

### !!!!!!!!!Внешний стандарт

Метод [**Внешнего стандарта**](../) предусматривает ряд дополнительных расчетов согласно различным нормативным документам.&#x20;

<figure><img src="../.gitbook/assets/image (44).png" alt=""><figcaption></figcaption></figure>

В зависимости от нормативного документа выберите из списка "Дополнительный расчет":

<figure><img src="../.gitbook/assets/image (48).png" alt=""><figcaption></figcaption></figure>

Размерность во всех дополнительных типах расчета устанавливается автоматически.&#x20;



<mark style="color:$danger;">Если дополнительный расчет не задан (выбран пункт "Нет расчета"), укажите размерности объема вводимой пробы и результата расчета.</mark>&#x20;

Для дополнительного пересчета результатов анализа в другую размерность, включите переключатель Дополнительно Х (где Х- новая размерность).

| Нет расчета | <p>мкл</p><p>мл</p><p>г</p><p>л</p> | <p>% об.</p><p>% мас.</p><p>мкг/мл,мг/мл, мг/л, мг/м3,г/100м3, мг/кг,мкг/л,%0об.,%0мас.</p><p>ppm об.</p><p>ppm мас.</p> | <p>% мас.</p><p>% об.</p><p>-</p><p>ppm мас.</p><p>ppm об..</p> |
| ----------- | ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------- |

### !!!!!!!!Внутренний стандарт

Метод [**Внутреннего стандарта**](../) предусматривает ряд дополнительных расчетов согласно различным нормативным документам.&#x20;

<mark style="color:$primary;">В таблице компонентов одно из веществ должно быть помечено как «Внутренний стандарт». Важно следить, чтобы время выхода стандарта не совпадало с пиками компонентов пробы. Это самый надежный метод для минимизации «человеческого фактора».</mark>

<br>

<figure><img src="../.gitbook/assets/image (46).png" alt=""><figcaption></figcaption></figure>

В зависимости от нормативного документа выберите из списка "Дополнительный расчет":

<figure><img src="../.gitbook/assets/image (47).png" alt=""><figcaption></figcaption></figure>

Размерность во всех дополнительных типах расчета устанавливается автоматически.&#x20;



<mark style="color:$danger;">Если дополнительный расчет не задан (выбран пункт Нет расчета), установите размерность стандарта, пробы, а также размерность результата расчета. Для дополнительного пересчета результатов анализа в другую размерность, включите переключатель Дополнительно Х (где Х- новая размерность).</mark>

| Стандарт            | Проба        | Результат    | Дополнительный результат                                                       |                                                              |
| ------------------- | ------------ | ------------ | ------------------------------------------------------------------------------ | ------------------------------------------------------------ |
| Нет расчета         | г, мл, %     | г, мл        | <p>% об.</p><p>% мас.</p><p>мг/мл, мг/л, мг/м3</p><p>ppm об.</p><p>ppm мас</p> | <p>% мас.</p><p>% об</p><p>-</p><p>ppm мас</p><p>ppm об.</p> |
| Бензол              | по умолчанию | по умолчанию | по умолчанию                                                                   | по умолчанию                                                 |
| Стандартная добавка | по умолчанию | по умолчанию | по умолчанию                                                                   | по умолчанию                                                 |
| Алкоголь СМЭ        | по умолчанию | по умолчанию | по умолчанию                                                                   | по умолчанию                                                 |

Для исключения внутреннего стандарта включите соответствующий переключатель.
