# Расчёты

## Количественный расчет

В данном подразделе рабочго метода (`*.metx`) настраивается математическая модель (тип расчета), на основании которой программный комплекс «NetChrom» осуществляет автоматическое преобразование геометрических параметров пика (площади или высоты) в физические значения концентраций определяемых компонентов.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (61).png" alt=""><figcaption><p>Выбор типа расчёта</p></figcaption></figure></div>

{% hint style="info" %}
**Важное архитектурное правило**\
Содержимое и набор доступных полей вкладки «Количественный расчет» динамически изменяются в строгой зависимости от выбранного оператором типа расчета.
{% endhint %}

## Нормализация

Тип расчета **«Нормализация»** основан на допущении, что сумма площадей всех зарегистрированных и идентифицированных на хроматограмме пиков соответствует 100% содержания компонентов в исследуемом образце. Данный метод является автономным и не требует предварительного построения многоточечных калибровочных графиков.

**Критические метрологические требования к методу:**

* **Учет относительной чувствительности:** Оператор в обязательном порядке должен проверить, чтобы в таблице компонентов аналитической методики для каждого вещества были корректно введены относительные чувствительности (поправочные коэффициенты). Без учета данных индивидуальных коэффициентов расчет будет обладать высокой погрешностью и останется достоверным исключительно для узкого спектра смесей, состоящих из близких по химической природе веществ.
* **Идентификация всех компонентов пробы:** Если на хроматограмме присутствует пик неизвестного или неидентифицированного вещества, метод простой нормализации распределит его условную долю между остальными целевыми компонентами. Это приведет к фатальному искажению итоговых результатов анализа («метод наврет»).

## Внутренняя нормализация

Метод **«Внутренней нормализации»** (также известный как метод абсолютной калибровки с нормированием) предусматривает выполнение ряда специализированных дополнительных расчетов в строгом соответствии с требованиями отраслевых нормативных документов (ГОСТ, ISO, ASTM).

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (45).png" alt=""><figcaption><p>Внутренняя нормализация</p></figcaption></figure></div>

**Особенности метрологического контроля и ручного ввода:**

* **Зависимость от объема пробы:** Данный метод расчета обладает критически высокой чувствительностью к физическому **объему вводимой пробы**. При использовании ручного ввода микрошприцем любая случайная ошибка или невоспроизводимость в микролитрах со стороны лаборанта приведет к прямой линейной ошибке в итоговом расчете концентрации.
* **Повышение прецизионности:** Для минимизации погрешности ручного дозирования в настройках метода рекомендуется использовать математически усредненное значение объема пробы, рассчитанное на основании серии из нескольких предварительных калибровочных (градуировочных) вводов.

**Элементы управления и настройки параметров:**

* **Поле «Результат»:** Выпадающий список, определяющий размерность итоговой концентрации. Доступны варианты: **% об.** (объемные проценты) или **% масс.** (массовые проценты).
*   **Поле «Дополнительный расчет»:** Интерактивное меню для выбора специализированного алгоритма пересчета фракционных и физико-химических свойств сред в соответствии с нормативной документацией лаборатории. Доступны следующие типы моделей:

    * _Нет расчета_ — стандартный прямой расчет концентрации;
    * _Сухой газ / Сжиженный газ_ — расчет компонентного состава и физических свойств углеводородных газов;
    * _Бензин/Нефть / Бензол_ — расчет фракционного состава и массовых долей аренов;
    * _Дистилляция / Отдувки_ — специализированные технологические модели;
    * _Газ сжиженный / Газ нефтезаводской_ — отраслевые алгоритмы расчета для объектов нефтепереработки.



    <div data-with-frame="true"><figure><img src="../.gitbook/assets/image (154).png" alt=""><figcaption><p>Дополнительный расчёт</p></figcaption></figure></div>

## Внешний стандарт

Метод **«Внешнего стандарта»** (метод абсолютной градуировки) основан на прямом сопоставлении площади (или высоты) пика определяемого компонента с градуировочной зависимостью, построенной ранее по искусственным стандартным смесям с известной концентрацией веществ.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (44).png" alt=""><figcaption><p>Внешний стандарт</p></figcaption></figure></div>

**Элементы управления и настройки параметров:**

* **Поле «Объем пробы»** — выпадающий список для выбора единицы измерения объема дозируемого образца.
* **Поле «Результат»** — задает целевую размерность итоговой концентрации.
* **Поле «Дополнительный расчет»** — меню выбора узкоспециализированных математических моделей и алгоритмов обработки данных в соответствии с требованиями МВИ и ГОСТ (модули: _Этиловый спирт_, _Меркаптаны_, _Горючий газ_, _Нефтяной газ_, _Трансформаторное масло_, семейство моделей _Газ природный_ по ГОСТ).

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (48).png" alt=""><figcaption><p>Дополнительные типы расчёта</p></figcaption></figure></div>

**Важные метрологические правила настройки:**

* **Автоматическое назначение размерностей:** При выборе любого специализированного типа из списка **«Дополнительный расчет»**, все необходимые единицы измерения для объема вводимой пробы и результирующей концентрации выставляются программным комплексом **автоматически** в строгом соответствии с требованиями выбранного нормативного документа.
* **Ручное назначение размерностей:** Если дополнительный пересчет не требуется (в поле установлен пункт **«Нет расчета»**), оператор обязан **вручную** заполнить матрицу доступных размерностей.

**Матрица доступных размерностей для метода «Внешний стандарт» (в режиме «Нет расчета»)**

| Дополнительный расчет | Объем пробы   | Результат                                                                                          | Дополнительный результат         |
| --------------------- | ------------- | -------------------------------------------------------------------------------------------------- | -------------------------------- |
| **Нет расчета**       | мкл, мл, г, л | % об., % мас., мкг/мл, мг/мл, мг/л, мг/м3, г/100м3, мг/кг, мкг/л, %Ооб., %Омас., ppm об., ppm мас. | % мас., % об., ppm мас., ppm об. |

* **Конвертация единиц измерения:** Для проведения автоматического дополнительного пересчета финальных результатов анализа в альтернативную размерность, активируйте переключатель **«Дополнительно Х»** (где вместо **Х** программный комплекс подставит наименование выбранной новой размерности из крайнего правого столбца таблицы).

## Внутренний стандарт

Метод **«Внутреннего стандарта»** является наиболее надежным и метрологически защищенным алгоритмом количественного анализа. Он разработан для полной минимизации случайных погрешностей лаборанта при ручном вводе пробы микрошприцем, испарении растворителя или нестабильности деления потока в инжекторе.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (46).png" alt=""><figcaption><p>Внутренний стандарт</p></figcaption></figure></div>

**Принцип функционирования метода**

В исследуемую пробу перед проведением анализа вручную вносится строго известное количество специального индивидуального вещества, которое гарантированно отсутствует в исходном образце — **внутреннего стандарта**. Программа рассчитывает концентрации целевых компонентов не по абсолютным площадям их пиков, а по отношению площади пика компонента к площади пика этого стандарта. При изменении объема вкола (например, лаборант ввел 0.9 мкл вместо 1.0 мкл) площади всех пиков уменьшатся пропорционально, но их отношение останется неизменным.

**Критические правила настройки в ПО «NetChrom»:**

1. **Маркировка в таблице компонентов:** Оператор обязан открыть общую таблицу компонентов текущей методики и явно пометить одно конкретное вещество статусом **«Внутренний стандарт»**.
2. **Контроль разделения пиков:** На этапе разработки методики необходимо строго следить за тем, чтобы время удерживания выбранного внутреннего стандарта не совпадало с временами удерживания целевых компонентов пробы. Пик стандарта на хроматограмме должен быть полностью отделен («разрешен») от соседних пиков.

**Элементы управления и настройки параметров:**

* **Поле «Стандарт»** — выпадающий список для выбора единицы измерения массы или объема внесенного эталона.
* **Поле «Проба»** — выпадающий список единиц измерения исходного исследуемого образца.
* **Поле «Результат»** — целевая размерность выходных концентраций.
* **Поле «Дополнительный расчет»** — меню выбора специализированных нормативных алгоритмов (модули: _Нет расчета_, _Бензол_, _Стандартная добавка_, _Алкоголь СМЭ_, _Дистилляция_).

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (47).png" alt=""><figcaption><p>Доп. расчёт</p></figcaption></figure></div>

**Важные метрологические правила настройки:**

* **Автоматическое назначение размерностей:** При выборе любого специализированного типа из списка **«Дополнительный расчет»**, все необходимые единицы измерения для стандарта, пробы и результирующей концентрации выставляются программным комплексом **«по умолчанию»** автоматически в соответствии с нормативным документом.
* **Ручное назначение размерностей:** Если дополнительный пересчет не требуется (выбран пункт **«Нет расчета»**), оператор обязан **вручную** заполнить матрицу доступных размерностей.

**Матрица доступных размерностей для метода «Внутренний стандарт» (в режиме «Нет расчета»)**

| Дополнительный расчет | Стандарт | Проба | Результат                                                    | Дополнительный результат         |
| --------------------- | -------- | ----- | ------------------------------------------------------------ | -------------------------------- |
| **Нет расчета**       | г, мл, % | г, мл | % об., % мас., мкг/мл, мг/мл, мг/л, мг/м3, ppm об., ppm мас. | % мас., % об., ppm мас., ppm об. |

* **Конвертация единиц измерения:** Для проведения автоматического дополнительного пересчета финальных результатов анализа в альтернативную размерность, активируйте переключатель **«Дополнительно Х»** (где вместо **Х** программный комплекс подставит наименование выбранной новой размерности).

**Исключение внутреннего стандарта из расчета результатов**

В случаях, когда при обсчете хроматограммы необходимо рассчитать концентрации компонентов без учета поправки на внутренний стандарт (например, для проверки абсолютных площадей пиков или экспресс-оценки общей нормализации), оператор может временно деактивировать этот алгоритм.

* **Действие:** Включите соответствующий переключатель (чекбокс) **«Исключить внутренний стандарт»** в интерфейсе настройки. Программа пересчитает таблицу результатов, игнорируя привязку к пику эталона, но сохранит все калибровочные коэффициенты в самом файле методики.

{% hint style="warning" %}
Активация переключателя «Исключить внутренний стандарт» переводит математическую модель в режим простого расчета, что может привести к резкому росту погрешности, если объем ручного ввода пробы нестабилен. Используйте эту функцию только для диагностических целей и не забывайте отключать её перед выдачей финальных протоколов анализа.
{% endhint %}
