# Файловая структура

## Структура папок

В программном обеспечении «NetChrom» (версия 3.0) управление массивами аналитических данных осуществляется посредством встроенного менеджера файлов. Данный программный модуль предназначен для навигации, систематизации, копирования, архивирования и экспорта рабочих файлов без использования средств операционной системы (системного проводника).

При первичном запуске программного обеспечения в директории пользователя `...\Мои документы` автоматически инициализируется корневой каталог **«NetChrom V.3.0»**.

Для обеспечения логического разделения данных внутри корневого каталога для каждого зарегистрированного хроматографа «Кристаллюкс-4000М» создается индивидуальная папка, имя которой соответствует сетевому или заводскому идентификатору прибора. Внутренняя иерархическая структура папок измерительного комплекса приведена ниже:

{% code title="Схема организации иерархической структуры данных" %}
```
📁 NetChrom V.3.0 (Корневой каталог)
┗ 📁 [Идентификатор_Хроматографа]
┣ 📁 Хроматограммы
┣ 📁 Градуировки
┣ 📁 Методы
┗ 📁 Архив
```
{% endcode %}

Внутренние подкаталоги имеют строгое функциональное назначение:

* **Подкаталог «Хроматограммы»** — предназначен для централизованного хранения зарегистрированных аналитических сигналов и результатов их обработки. Файлы сохраняются в защищенном формате `*.chrx`.
* **Подкаталог «Градуировки»** — служит для размещения градуировочных таблиц и параметров калибровочных зависимостей компонентов в формате `*.grd`.
* **Подкаталог «Методы»** — предназначен для хранения файлов рабочих методов (инструментальных профилей и программ интеграции) в формате `*.metx`.
* **Подкаталог «Архив»** — специализированная директория для временного размещения файлов различных категорий, выведенных оператором из текущего аналитического цикла или подлежащих последующему удалению.

### **Функциональные возможности менеджера файлов**

Все операции с файлами и папками выполняются через встроенный файловый менеджер программы. Это значительно упрощает навигацию и работу с данными, устраняя необходимость использования сторонних файловых менеджеров, таких как Проводник.&#x20;

Интерфейс встроенного менеджера файлов обеспечивает выполнение следующих регламентированных операций:

* Навигация по дисковому пространству и визуализация дерева каталогов.
* Копирование, перемещение и удаление файлов данных.
* Пакетный экспорт хроматограмм и отчетов в сторонние форматы.
* Быстрая загрузка выбранного рабочего метода в операционную память хроматографа.

{% hint style="warning" %}
Модификация, перемещение или удаление файлов, находящихся в рабочей директории программы, средствами сторонних файловых менеджеров (например, «Проводник» ОС Windows) не рекомендуется. Это может привести к нарушению целостности базы данных, потере метрологической прослеживаемости и блокировке функций автоматического поиска файлов программным обеспечением «NetChrom».
{% endhint %}
