# Виды файлов

Программное обеспечение «NetChrom» осуществляет хранение, обработку и систематизацию аналитической информации с использованием строго определенных типов файлов собственной структуры. Разделение данных по типам расширений обеспечивает модульность системы, защиту метрологически значимых данных от несанкционированного изменения и удобство экспорта/импорта настроек.

Основные типы файлов, используемые при работе с хроматографом «Кристаллюкс-4000М», приведены в таблице.

### **Типы файлов ПО «NetChrom»**

| Расширение файла | Тип (компонент) данных | Назначение файла                                                         |
| ---------------- | ---------------------- | ------------------------------------------------------------------------ |
| **`*.metx`**     | Рабочий метод          | Хранение инструментальных параметров прибора и алгоритмов обработки.     |
| **`*.chrx`**     | Файл хроматограммы     | Запись зарегистрированного аналитического сигнала и результатов расчета. |
| **`*.grd`**      | Файл градуировки       | Хранение калибровочных зависимостей и параметров компонентов.            |

### **Описание назначений и структуры файлов**

* **Файлы рабочих методов (`*.metx`)**\
  Являются конфигурационными файлами, регламентирующими весь цикл проведения хроматографического анализа. Файл содержит в себе паспортные метаданные, температурные программы термостатов, настройки электронных регуляторов расходов и давлений газов (РРГ/РДГ), параметры управления детекторами, а также временные таблицы интегрирования. Файл метода может быть назначен как для выполнения текущего измерения в реальном времени, так и для пакетной переобработки ранее зарегистрированных сигналов.
* **Файлы хроматограмм (`*.chrx`)**\
  Представляют собой защищенный массив данных, содержащий первичный цифровой сигнал (хроматографический профиль), зарегистрированный измерительными каналами детекторов хроматографа «Кристаллюкс-4000М». Помимо самого сигнала, в файл `*.chrx` автоматически встраивается полная копия рабочего метода, при котором данный анализ был выполнен. Это обеспечивает абсолютную прослеживаемость и защищает результаты измерений (изменение параметров интеграции в файле хроматограммы не изменяет исходный глобальный файл метода).
* **Файлы градуировки (`*.grd`)**\
  Специализированные метрологические файлы, содержащие градуировочные (калибровочные) таблицы. В них аккумулируются данные о качественной идентификации целевых веществ (названия компонентов, времена удерживания, границы окон идентификации), а также математические коэффициенты калибровочных кривых, полученные при анализе градуировочных стандартных образцов. Файлы с расширением `*.grd` могут динамически подключаться к различным рабочим методам `*.metx` для выполнения количественного расчета концентраций однотипных аналитических задач.
