# Управление хроматографом



Данное окно является основным рабочим пространством оператора и предназначено для мониторинга текущего состояния узлов, настройки параметров запуска и оперативного управления анализом.

## Панель управления хроматографом (Верхняя лента)

Данная панель позволяет запускать анализы, отслеживать текущее состояние прибора и управлять его режимами работы.

<figure><img src="../.gitbook/assets/image (72).png" alt=""><figcaption><p>Панель управления хроматографом</p></figcaption></figure>

**Группа основных функций**

* **Выполнить анализ:** Открывает диалоговое окно для запуска измерения. Здесь выбирается метод, вводится название пробы и данные оператора.
* **Видеосамописец:** Открывает окно мониторинга сигнала в реальном времени. Позволяет визуально оценить стабильность нулевой линии перед запуском анализа.
* <mark style="color:$danger;">**Неисправности:**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">Индикатор состояния прибора. При возникновении ошибок (например, отсутствие поджига или перегрев) иконка меняет цвет. Нажатие открывает список активных ошибок.</mark>
* **Журнал:** Переход к журналу событий, где фиксируются все действия пользователя и системные сообщения прибора.
* **Конфигурация хроматографа:** Сервисный раздел. Позволяет настраивать состав оборудования (детекторы, испарители, краны) и параметры связи.

**Группа оперативного управления**

* **Старт анализа:** Команда запуска записи хроматограммы (если прибор готов).
* **+5 минут к анализу:** Позволяет оперативно продлить время текущего анализа, если целевые компоненты еще не вышли из колонки.
* **Режим охлаждения:** Принудительно включает вентиляторы термостата для быстрого снижения температуры (например, при подготовке к следующему анализу с низкой начальной температурой).
* <mark style="color:$danger;">**Продувка колонки:**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">Включает режим интенсивной очистки колонок при повышенных температурах и расходах газа-носителя.</mark>
* **Режим сна:** Переводит прибор в экономичный режим (снижение температур и расходов газов) на время простоя.

{% hint style="info" %}
Кнопка **«Старт анализа»** становится активной только после того, как хроматограф выйдет на заданный режим (все параметры температуры и давления станут стабильными).
{% endhint %}

## Область «Запуск анализа» (Левая часть)

Здесь формируется задание на выполнение конкретного измерения.

### <mark style="color:$danger;">Планировщик (Режим серии анализов)</mark>

<mark style="color:$danger;">В верхней части окна управления расположен слайдер</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">**«Режим серии анализов»**</mark><mark style="color:$danger;">.</mark>

* <mark style="color:$danger;">**Назначение:**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">Переключает интерфейс из режима единичного измерения в режим</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">**Планировщика**</mark><mark style="color:$danger;">.</mark>
* <mark style="color:$danger;">**Функционал:**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">При активации слайдера открывается таблица-очередь, где оператор может задать последовательность из десятков анализов с использованием разных методов, автосемплера или циклического ввода проб.</mark>
* <mark style="color:$danger;">**Автоматизация:**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">Планировщик позволяет полностью автономно выполнять серию измерений с последующим автоматическим переходом прибора в режим экономии газа или выключения после завершения всей очереди.</mark>

## Группа «Параметры запуска»

Позволяет выбрать методику для каждого из каналов управления:

* **Основной метод (Старт 1):** Выбор основного файла метода (.met) для ГХ «Кристаллюкс-4000М».
* **Метод сателлита (Старт 2):** Используется для раздельного управления каналами детектирования.

{% hint style="info" %}
**Конфигурация:** возможность использования Сателлита активируется в конфигурации хроматографа. Если функция включена, в окне управления появляется дополнительная область для выбора метода и ввода паспорта пробы второго канала.
{% endhint %}

* **Назначение хроматограммы:** Поле для ввода имени файла или выбора папки сохранения (по умолчанию — «Анализ»).

### Группа «Паспорт пробы»

Предназначена для ввода метаданных, которые будут сохранены вместе с хроматограммой:

* **Проба/Анализ:** Название исследуемого объекта.
* **Номер пробы / Номер анализа:** Порядковые данные для автоматизации отчетности.
* **Оператор / Комментарий:** Текстовая информация о лице, проводившем анализ, и дополнительных условиях.

## Панель мониторинга и статуса (Правая часть)

Позволяет в реальном времени контролировать «здоровье» прибора.

### Вкладка «Состояние»

Отображает текущие физические параметры:

* **Температуры:** Сравнение заданных и измеренных значений для колонок, детекторов и испарителей.
* **Расход газов:** Текущие потоки по всем настроенным каналам РРГ/РД (Газ 1, Газ 2, Водород и т.д.).
* **Давление:** отображаются текущие значения давления на капиллярных колонках.
* **Статус готовности.**
* **Баланс ДТП:** Интерактивная шкала для контроля разбаланса моста детектора по теплопроводности.

### Вкладка «Диагностика»

Вкладка «Диагностика» предназначена для углубленного контроля состояния аппаратных узлов хроматографа. Она позволяет оператору или сервисному инженеру проверить корректность работы электроники и исполнительных механизмов без разборки прибора.

### Параметры напряжения (В)

Отображает стабильность питания основных электронных модулей:

* **Контроль АЦП:** напряжение питания аналогово-цифрового преобразователя. Стабильность этого параметра напрямую влияет на уровень шума и дрейф нулевой линии.
* **Питающая сеть:** контроль входного напряжения. Позволяет выявить просадки в электросети лаборатории.
* **Питание детектора:** рабочее напряжение, подаваемое на электронные платы детектирования.

### Напряжение клапанов (В)

В этом блоке отображается напряжение, подаваемое на электромагнитные клапаны регуляторов (РРГ/РД).

* **Диагностический признак:** если расход газа равен нулю, а напряжение на клапане максимально — это сигнализирует о засоре в газовой линии или об отсутствии давления на входе в прибор.
* Список клапанов полностью соответствует настроенной конфигурации (Газ-1...5, Водород, Воздух).

### Ток детекторов (мА)

Отображает реальное потребление тока силовыми узлами детекторов (например, **Ток ДТП-1**). Нулевое значение при включенном методе может указывать на обрыв спирали или срабатывание защиты по потоку.

### Панель «Параметры вне нормы»

Расположена в нижней правой части. Если прибор не может выйти в режим готовности, здесь будет выведен список конкретных причин (например, «Испаритель 1: низкая температура»).
