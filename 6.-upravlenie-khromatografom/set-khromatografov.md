# Сеть хроматографов

## **Выбор и добавление хроматографа**

После запуска ПО NetChrom откроется главное окно со списком доступного оборудования. Для взаимодействия с газовым хроматографом (ГХ) «Кристаллюкс-4000М» необходимо выбрать его из списка подключенных приборов или добавить новый.

**Выбор активного прибора**

Если нужный хроматограф уже зарегистрирован в системе, он отображается в списке в центральной части окна (например, «3791\_ПИД-ДТП»).

**Для начала работы:**

1. Найдите прибор в списке по его сетевому имени или серийному номеру.
2. Нажмите кнопку **«Управлять прибором»** (иконка со стрелкой вверх-вправо), расположенную напротив названия ГХ.

{% hint style="info" %}
После нажатия откроется панель управления режимами, детекторами и сбором данных.
{% endhint %}

Если в списке отсутствует необходимый хроматограф: нажмите кнопку **«Добавить хроматограф»** на синей панели в нижней части окна.

## **Управление списком (Сеть хроматографов)**

Для быстрого доступа к хроматографам или изменения состава оборудования используйте кнопку **«Сеть хроматографов»** в верхнем правом углу окна.

* Нажатие на иконку сети позволяет развернуть/свернуть подробный список устройств.
* Кнопка **«Удалить из сети»** (иконка корзины) используется для исключения неиспользуемого оборудования из общего списка мониторинга.

{% hint style="info" %}
Текущий статус работы («Этап: Нулевой») и время с начала процесса отображаются непосредственно под названием каждого прибора, что позволяет контролировать состояние всей лаборатории из одного окна.
{% endhint %}
