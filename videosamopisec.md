# Видеосамописец

## Видеосамописец

Диалоговое окно **«Видеосамописец»** предназначено для графического отображения сигналов детекторов в реальном времени. Наблюдение за динамикой сигнала необходимо на всех этапах работы с хроматографом:

* **Этап подготовки:** для контроля выхода детектора на рабочий режим.
* **Выход на режим:** для оценки стабильности нулевой линии и уровня шумов.
* **Выполнение анализа:** для визуального мониторинга процесса разделения компонентов.

### Основные элементы управления

#### 1<mark style="color:$danger;">. Панель выбора каналов</mark>

В верхней части окна расположены переключатели активных детекторов (например, <mark style="color:$warning;">**●**</mark>**&#x20;ПИД-1**, <mark style="color:$success;">**●**</mark>**&#x20;ПФД-1**, <mark style="color:$danger;">**●**</mark> <mark style="color:$danger;">**ДТП-3**</mark><mark style="color:$danger;">):</mark>

* **Тумблер (Вкл/Откл):** позволяет отображать или скрывать график конкретного детектора на общем поле.
* **Цветовая индикация:** каждый график имеет свой цвет, соответствующий маркеру рядом с названием детектора.

#### 2. Поле графиков

Отображает зависимость сигнала (мВ) от времени:

* **Ось Y (Вертикальная):** амплитуда сигнала в милливольтах. Масштаб может быть изменен для детального изучения шумов или просмотра высоких пиков.
* **Ось X (Горизонтальная):** время в формате _чч:мм:сс_.
* **Сетка:** помогает визуально оценить дрейф и периодические помехи.

#### <mark style="color:$danger;">Функциональные возможности (дополнить???)</mark>

* **Масштабирование:** С помощью мыши (выделение области) можно увеличить любой участок видеозаписи для детального осмотра нулевой линии.
* **Смещение (Панорамирование):** Позволяет перемещаться вдоль временной оси и просматривать уже записанный фрагмент сигнала до начала официальной регистрации хроматограммы.
* **Автоматическая подстройка:** Функция автоматического выбора масштаба по оси Y, чтобы сигнал всегда находился в поле видимости.

#### Элементы управления отображением

Данная группа кнопок предназначена для оперативной настройки области видимости графиков без изменения параметров самого метода.

<figure><img src=".gitbook/assets/image (176).png" alt=""><figcaption></figcaption></figure>

**Кнопка «Диапазон»**

Позволяет вручную задать границы отображения по осям времени и амплитуды. Используется, когда необходимо зафиксировать «окно» просмотра в определенных пределах (например, от 10 до 20 мВ).

**Меню «Показать»**

Инструментарий для быстрого позиционирования графиков в рабочей области. При нажатии на стрелку открывается список команд автоматического масштабирования:

* **Показать все** — устанавливает масштаб таким образом, чтобы все данные от начала записи до текущего момента по всем детекторам уместились в окне.
* **Начало / Конец по времени** — мгновенный переход к самому первому или самому последнему зарегистрированному отсчету.
* **Верх / Низ по высоте** — перемещение области видимости к максимальному или минимальному значению сигнала.
* **Все по времени** — растягивает график на всю ширину окна по горизонтали.
* **Все по высоте** — выполняет вертикальный «автомасштаб», подбирая высоту так, чтобы сигнал занимал 80–90% высоты окна.

**Кнопка «Очистить»**

Выполняет удаление накопленных данных с экрана видеосамописца.

{% hint style="info" %}
Данная команда очищает только визуальную историю в текущем окне мониторинга. Она не влияет на запись текущего анализа и не удаляет данные из файла хроматограммы, если идет процесс регистрации. Используется для «обнуления» экрана перед выходом на новый режим.
{% endhint %}

### Управление масштабом и навигация

Для детального изучения хроматограммы в окне Видеосамописца предусмотрены инструменты интерактивного масштабирования. По умолчанию система автоматически устанавливает масштаб, при котором в окне виден весь график от начала записи.

#### Управление «Мышью»

* **Произвольное увеличение:** Чтобы увеличить интересующий участок, нажмите левую кнопку мыши в одном из углов нужной области и, не отпуская её, растяните прямоугольник выделения в противоположный угол. После отпускания кнопки выбранная область развернется на все окно.
* **Быстрый возврат (Сброс):** Для возврата к отображению всего графика выполните **двойной щелчок** левой кнопкой мыши в любом месте графического поля (вне границ пика).

#### Использование клавиатуры (Курсорные кнопки)

Инструменты мыши удобно сочетать с клавиатурой для быстрого кратного изменения масштаба. Ориентируйтесь на стрелки управления курсором:

* **↑ (Вверх):** Растянуть график в два раза по вертикали (увеличение амплитуды сигнала).
* **↓ (Вниз):** Сжать график в два раза по вертикали.
* **→ (Вправо):** Растянуть график в два раза по горизонтали (увеличение временного разрешения).
* **← (Влево):** Сжать график в два раза по горизонтали.

#### <mark style="color:$danger;">!!!Диагностика по видеосамописцу (Неисправности хроматограммы или как то так там)</mark>

<mark style="color:$danger;">Визуальный анализ «живого» сигнала позволяет оперативно выявить ряд проблем</mark>:

| Вид сигнала                            | Вероятная причина                                                                |
| -------------------------------------- | -------------------------------------------------------------------------------- |
| **Хаотичные всплески («пики-иголки»)** | Электрические наводки в сети или загрязнение детектора.                          |
| **Резкий уход линии вверх/вниз**       | Изменение расхода газа или резкий скачок температуры.                            |
| **Широкие волны (циклический дрейф)**  | Нестабильность давления на входе в прибор или работа кондиционера в лаборатории. |
| **Ступенька сигнала**                  | Момент розжига пламени (для ПИД/ПФД) или переключение кранов.                    |

{% hint style="info" %}
Перед запуском анализа убедитесь по Видеосамописцу, что нулевая линия стабильна (отсутствует видимый дрейф). Это гарантирует правильный расчет площадей пиков при последующей обработке.
{% endhint %}

#### Рекомендации по анализу пиков

* **Просмотр малых пиков:** Если целевой компонент имеет малую концентрацию, используйте клавишу **↑** несколько раз, пока шум нулевой линии не станет отчетливо виден. Это позволит визуально оценить соотношение «сигнал/шум».
* **Оценка разделения:** Если два пика выходят слишком близко друг к другу, воспользуйтесь выделением мышью или клавишей **→** для детального осмотра «долины» между ними.
* **Панорамирование:** При сильном увеличении используйте полосы прокрутки для перемещения вдоль хроматограммы без изменения масштаба.
